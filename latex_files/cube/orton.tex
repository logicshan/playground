\chapter{引言}
外延性是这样一个原则：如果两个数学对象具有相同的可观测属性，那么它们就被视为相等。通常，数学的形式系统会包含旨在体现这一原则的公理。在公理集合论的背景下，我们有外延公理，它表明如果两个集合包含相同的元素，那么这两个集合相等。其形式表述为：
\[\forall A . \forall B .((\forall x .(x \in A \Leftrightarrow x \in B)) \Rightarrow A=B)\] 
在（单值/同伦）类型论的语境中，我们有沃沃斯基（Voevodsky）的单值公理[62，第2.10节]，大致来说，该公理表明如果两个类型是同构的，那么它们相等。在类型论的语言中，它可以表述为：
\[\prod_{A, B: U} A \simeq B \to A=B\]
其中\(A \simeq B\)是\(A\)和\(B\)之间等价关系的类型。例如，我们可能有十进制自然数类型\(\mathbb{N}_{dec}\)，其项为\(0, 1, 2, 3, \cdots: \mathbb{N}_{dec}\)，还有二进制自然数类型\(\mathbb{N}_{bin}\)，其项为\(0, 1, 10, 11, \cdots: \mathbb{N}_{bin}\) 。很容易展示如何在这两种编码之间来回转换，也就是说，这两种类型是等价的。因此，根据单值公理，我们可以推断出\(\mathbb{N}_{dec} = \mathbb{N}_{bin}\)。 

乍一看，这些类型相等可能会让人觉得奇怪。例如，7是\(\mathbb{N}_{dec}\)（十进制自然数类型）的一个元素，但不是\(\mathbb{N}_{bin}\)（二进制自然数类型）的元素，那么这两种类型怎么会相等呢？这个公理之所以是自洽的，是因为类型论的规则不允许你构造诸如“7是\(\mathbb{N}_{bin}\)的元素”这样的陈述。这些规则确保所有陈述都以一种适当的通用方式表述，这样一来，陈述就只能利用一个类型的外部可观察到的属性，而不是所使用的特定编码。

我们来看一个类型论中有效陈述的例子：交换幺半群的性质。我们说，类型X构成一个交换幺半群，当且仅当存在一个运算\(\ul +\ul : X \to X \to X\)，该运算满足结合律、交换律，并且存在一个单位元\(e: X\)。这个定义在类型论中是合理的，因为它适用于任意类型；它对$X$的元素具体是什么样子没有任何假设。 

在类型论中，可以证明\(\mathbb{N}_{dec}\)（十进制自然数类型）构成一个交换幺半群，其中“+”为加法运算，\(e\)为\(0\)。由于\(\mathbb{N}_{dec}\)和\(\mathbb{N}_{bin}\)（二进制自然数类型）只是同一对象（自然数）的不同编码形式，那么\(\mathbb{N}_{bin}\)也应该构成一个交换幺半群。然而，如果没有单值公理，要证明这一点，就需要在两种编码之间来回转换，手动重新构建针对\(\mathbb{N}_{dec}\)的证明过程。单值公理的强大之处在于，由于\(\mathbb{N}_{dec} \simeq \mathbb{N}_{bin}\)，进而\(\mathbb{N}_{dec}=\mathbb{N}_{bin}\)，它能让我们直接将交换幺半群的结构从\(\mathbb{N}_{dec}\)迁移到\(\mathbb{N}_{bin}\) 。 

用这种新的单值性原则扩展类型论的最简单方法，就是将其作为一条公理添加进去。也就是说，假设存在一个项能够证明单值性原则。然而，这种方法存在一个问题。类型论是一个构造性系统，其基本元素具有计算意义。这意味着我们不仅知道如何构造项，还知道如何对这些项进行计算。这种特性的一个例子是，如果我们证明了一个形如 “存在一个自然数$n$使得\ldots” 的命题，那么我们应该能够从这个证明中实际计算出$n$的值。然而，仅仅把单值性作为一条公理提出来，我们无法先验地知道涉及单值性的表达式应该如何计算。 

为了理解这类表达式应如何计算，我们研究单值类型论的模型。一旦我们有了单值类型论（UTT）的模型，我们就有可能通过查看表达式在模型中的解释来计算其值。或者，更有可能的是，我们可以利用该模型来证明新的计算规则的合理性，然后将这些规则添加到类型论中，以恢复良好的计算属性。 

为了理解这类表达式应如何计算，我们研究单值类型论的模型。一旦我们有了单值类型论（UTT）的模型，我们就有可能通过查看表达式在模型中的解释来计算其值。或者，更有可能的是，我们可以利用该模型来证明新的计算规则的合理性，然后将这些规则添加到类型论中，以恢复良好的计算属性。 

单值类型论的首个模型——单纯集模型[39]，本质上利用经典逻辑来判定某些不可判定的命题。因此，虽然它表明单值公理是一致的，但实际上它并不能为恢复类型论的计算内容提供良好基础，因为在该模型中对表达式求值是不可计算的。 

为了解决这些问题，科恩等人引入了类型论的立方体集模型[18]。这个新模型是用构造性元理论构建的，在该模型中对表达式求值是可计算的，从而为恢复类型论的计算内容提供了一种方法。本论文中的大部分工作都基于这个模型展开。 

总的来说，本文使用立方体集拓扑斯的内部语言分析立方体集模型。这种方法具有诸多优势。具体而言，该方法能使立方体集模型的某些方面更加清晰、易于理解；通过对构建单值类型论模型所需的属性进行公理化，我们得以对现有模型进行推广；同时，它还能明确为模拟类型论的各个方面需要哪些属性。

对模型进行公理化的过程并非总是一帆风顺。例如，我们将在第7章中看到，构建单值全域的问题给这种方法带来了特殊挑战。然而，我们认为这样做是一项值得的努力。这种方法为许多应用提供了坚实的基础。例如，在第6章中，我们使用这种方法对单值公理进行了另一种表述。其他作者也将这项工作应用于其他方面，比如证明独立性结果[61]。 

- 在第5章中，我们展示了如何在任何具有适当结构的基本拓扑斯中，使用内部语言对类型论的立方体集模型进行公理化。然而，我们无法定义一个单值全域，尽管我们确实证明了一种与全域无关的单值形式。 

- 在第6章中，我们将单值公理分解为一些替代公理，并表明在具有第5章所规定额外结构的任何拓扑斯中，这些公理都能得到满足。这本质上是对前面提到的与全域无关的单值性证明的一种简化。 

- 在第7章中，我们将解释如何通过一种模态来扩展内部语言，以便能够对拓扑斯的某些全局性质进行公理化。利用这种模态，我们展示如何构建一个单值全域，从而解决前面章节中的不足。 

\chapter{类型论导论}

在本章中，我们将介绍阅读本论文其余部分所需的一些背景知识。特别地，我们将引入单值类型论（也称为同伦类型论），并解释沃沃斯基（Voevodsky）的单值公理。然后，我们将介绍立方体类型论系统，它可被视为标准单值类型论的扩展或细化。立方体类型论的关键特性在于，它为单值公理提供了一种计算解释。 

\section{单值类型论}
单值类型论（UTT），也称为同伦类型论（HoTT），是内涵马丁-洛夫类型论的一个变体。在这个理论中，我们通过引入一个新的公理，即沃沃斯基单值公理，将数学中把同构的结构认为是相等的这样一个常见做法形式化。这样做使我们不得不重新思考内涵类型论中关于等式本质的许多直觉。 

在本节中，我们将回顾马丁-洛夫类型论（Martin-Löf type theory）的基本内容，包括内涵恒等类型的概念，并探讨一种新的直观理解，即把这些恒等类型的元素看作是某个拓扑空间中的路径。我们会发现，这种直观理解使我们对 “任意两个相等性证明本身也相等” 这一原则产生质疑，该原则通常被称为恒等证明的唯一性（UIP）。然后，我们将讨论沃沃斯基（Voevodsky）的单值公理，它体现了这样一种概念：当且仅当两个空间可以相互连续变形时，这两个空间是等价的。最后，我们会看到单值公理实际上与UIP原则是不一致的。 

这里呈现的信息仅作为总结。为避免疑问，我们在此描述的类型论与《同伦类型论》书籍[62]附录A中定义的完全一致。 

\subsection{马丁-洛夫类型论}
马丁-洛夫类型论（MLTT）是一种数学基础体系，它基于类型而非更为常见的集合概念。它允许将命题编码为类型，而这些类型的项则是相关命题的证明。这种解释通常被称为柯里 - 霍华德对应（Curry - Howard correspondence），有时也被称作命题即类型。马丁-洛夫类型论包含许多组件，我们可以用它们来构建和操作类型及项。这些组件包括： 

恒等类型——给定一个类型A以及两个项\(x,y:A\)，我们可以构造出类型\(\mathtt{Id}_{A}(x,y)\)，该类型用于证明\(x\)等于\(y\) 。根据自反性，我们知道任何事物都必然与自身相等，所以对于任意的A以及\(x:A\)，我们都有一个证明\(\mathtt{refl}_{x}:\mathtt{Id}_{A}(x,x)\)，它表明\(x\)等于自身。我们也会将类型\(\mathtt{Id}_{A}(x,y)\)写作\(x =_{A} y\)，或者简写成\(x = y\)。当我们有一个证明\(p:\mathtt{Id}_{A}(x,y)\)时，我们就说\(x\)和\(y\)在命题上相等，这与通过一系列\(\beta\eta\) -归约可相互转换的情况不同，在后一种情况下，我们说它们在定义上相等，并写作\(x \equiv y\)。

\subsection{恒等类型与恒等证明的唯一性（UIP）}
如上文所述，对于任意类型\(A\)以及项\(x,y\in A\)，我们可以构造恒等类型\(\mathtt{Id}_{A}(x,y)\)。我们通过J -消除器来利用恒等证明。这个消除器表明，给定一个依赖于项\(y\in A\)和\(p:x = y\)的类型\(C(x,y,p)\)，我们只需给出一个类型为\(C(x,x,\mathtt{refl}_{x})\)的项，就可以得到\(C(x,y,p)\)的一个实例。或者说，为了证明关于某个与\(x\)命题上相等（由\(p\)见证）的\(y\)的性质\(C\)，只需假设\(y\equiv x\)且\(p\equiv \mathtt{refl}_{x}\) 。 

对类型论的一种标准直觉是，类型代表集合，其项代表该集合的元素。在这种解释下，如果\(x\)和\(y\)不同，我们可能会预期恒等类型对应空集；如果\(x\)和\(y\)实际上代表同一元素，恒等类型则对应单元素集。这让我们可以简单地把自反构造子\(\mathtt{refl}\)解释为单元素集中的唯一元素。理解J-消除器也很容易，因为证明\(p: \mathtt{Id}_{A}(x, y)\)告诉我们\(x\)和\(y\)代表同一元素，并且\(\mathtt{refl}\)和\(p\)都表示单元素集的唯一元素，因此\(C(x, y, p)\)实际上必然与\(C(x, x, \mathtt{refl}_{x})\)表示同一个集合。 

事实上，在这种解释下，由于恒等类型表示的集合最多包含一个元素，那么任何\(p,q:\mathtt{Id}_{A}(x,y)\)实际上必然表示同一个元素。所以我们会期望嵌套恒等类型\(\mathtt{Id}_{\mathtt{Id}_{A}(x,y)}(p,q)\)总是有元素的。这个原则通常被称为恒等证明的唯一性（UIP）。然而，霍夫曼（Hofmann）和施特赖歇尔（Streicher）[29, 32]通过在群胚范畴（每个态射都可逆的范畴）中展示一个模型，表明这个原则并不能从类型论的常规规则推出。在这个模型中：类型被解释为群胚，项\(x,y:A\)被解释为群胚中的元素\(x,y\in \obj A\)，恒等类型\(\mathtt{Id}_{A}(x,y)\)被解释为\(\hom_{A}(x,y)\)上的离散群胚 。UIP原则不成立，因为群胚中两个对象之间可以有多个不同的态射。 

单值类型论（UTT）/同伦类型论（HoTT）的许多工作都受到这样一种直觉的启发：我们不应把类型看作集合、把类型中的项看作集合的元素，而应将类型视为表现得像拓扑空间的对象，其项则是空间中的点。然后，我们把恒等类型的元素\(p: \mathtt{Id}_{A}(x, y)\)视为从点\(x\)到点\(y\)的路径。所以，我们认为\(p\)的行为类似于一个连续函数\(p: \mathbb{I} \to A\)，其中\(\mathbb{I}\)是单位区间\([0, 1]\)，\(p(0) = x\)且\(p(1) = y\)。接着，我们把嵌套恒等类型的元素\(\alpha: \mathtt{Id}_{\mathtt{Id}_{A}(x, y)}(p, q)\)视为同伦\(\alpha: \mathbb{I}×\mathbb{I} \to A\)，使得对于所有\(i \in \mathbb{I}\)，有\(\alpha(0, i) = p(i)\)、\(\alpha(1, i) = q(i)\)，以及\(\alpha(i, 0) = x\)和\(\alpha(i, 1) = y\)。 

在这种类型论的解释下，恒等证明唯一性（UIP）原则似乎不再显而易见。事实上，通过考虑图2.1所示的例子，我们就能明白为什么完全摒弃这一原则是合理的。点\(x\)和\(y\)代表类型\(A\)的两个元素。图中所示的两条路径都是从\(x\)到\(y\)的，因此在这种解释下，它们对应于恒等类型的元素\(p,q:\mathtt{Id}_{A}(x,y)\)。然而，由于空间中心存在空洞，显然在不移动任何一个端点的情况下，无法将其中一条路径连续变形为另一条路径。因此，它们之间不存在同伦，所以，嵌套恒等类型\(\mathtt{Id}_{\mathtt{Id}_{A}(x,y)}(p,q)\)中不存在元素。

\subsection{沃沃斯基单值公理}
在上一节中，我们讨论了单值类型论（UTT）/同伦类型论（HoTT）背后的新直觉，以及为什么这可能会让我们对恒等证明的唯一性（UIP）原则产生质疑。然而，我们实际上并没有以任何方式扩展类型论，将这种直觉形式化。这是通过沃沃斯基（Voevodsky）的单值公理实现的（见参考文献[62]的2.10节）。该公理将类型之间的相等类型\(\mathtt{Id}_{U}(A, B)\)（也写作\(A = B\)）与等价类型\(A \simeq B\)等同起来。

单值公理体现了一种同伦直观：具有相同同伦群的两个空间（即两个相互等价的空间）在某个分类空间中由一条路径相连。也就是说，我们将\(U\)视为（小）同伦型的分类空间。

从逻辑的角度来看，我们可以把单值公理视为将数学中同构对象视为同一这一常见做法进行形式化的工具。例如，一位数学家可能证明了某个数学对象，比如群\(A\)，与另一个对象\(B\)同构，而我们已经知道\(B\)具有某些性质。接着，这位数学家会在不验证所讨论的性质在同构关系下是否保持的情况下，很自然地得出\(A\)也具有该性质的结论。单值公理通过本质上声明类型论中的每一种构造在等价关系下都保持不变，从而将这种推理形式进行了形式化。 

单值公理的确切表述涉及到一些微妙之处，尤其是关于类型之间等价关系的定义。我们现在给出确切的定义，先从等价关系的定义以及可缩类型这一辅助概念开始。 

定义 2.1.1（可缩性）。如果类型
\[\mathtt{isContr} (A) \triangleq \sum_{a_{0}: A} \prod_{a: A}\left(a_{0}=a\right)\]
有元素，那么称类型\(A\)是可缩的。可缩性表达了这样一个事实，即一个类型有唯一的元素。 

定义 2.1.2（等价关系）。一个等价关系\(A \simeq B\)是一个序对\((f, e)\)，其中\(f: A \to B\)，并且\(e\)是一个证明，它表明对于每一个\(b: B\)，\(f\)在\(b\)处的纤维是可缩的。准确地说：
\[A \simeq B \triangleq \sum_{f: A \to B} \mathtt{isEquiv}(f)\]
其中
\[\fib_{f}(b) \triangleq \sum_{a: A}(f\ a = b)\] 且 \[\mathtt{isEquiv}(f) \triangleq \prod_{b: B} \mathtt{isContr}\left(\fib_{f}(b)\right)\]
对于\(A: U_{i}\)，\(B: U_{j}\)，其中\(i\)、\(j\)为任意值。 

等价关系的一个简单例子是对于任意类型\(A\)的恒等函数\(id_{A}: A \to A\)。为了证明\(id_{A}\)是一个等价关系，我们必须证明\(\prod_{a: A} \mathtt{isContr} (\sum_{x: A}(x = a))\)，利用J-消除器很容易证明这一事实。 

请注意，存在许多相互竞争的等价概念（见参考文献[62]的第4章）。我们选择了这里所使用的这个定义，是因为它简化了第6章中的一些证明。同样值得注意的是，在单值类型论（UTT）中，传统的同构定义并不是一个表现良好的等价概念。具体来说，将\(isEquiv(f)\)定义为存在一个映射\(g: B \to A\)，使得\(\prod_{a: A} g(f a)=a\)且\(\prod_{b: B} f(g b)=b\)，这种定义方式表现欠佳。这是因为这个被称为拟逆的概念并非一个简单的命题（见参考文献[62]的定理4.1.3），因此可能存在多个不同的证明来表明\(f\)具有一个拟逆。这会带来一些不良后果。例如，拟逆等价的类型并非类型\(A \to B\)的子类型，而且事实上，使用拟逆等价来陈述的单值公理是不一致的（见参考文献[62]的练习4.6）。 

接下来，需注意的是，从两个类型之间的相等关系类型到它们之间的等价关系类型，总是存在一个典范映射。 

定义 2.1.3（\textbf{idtoeqv}）。对于所有的\(i\)，以及类型\(A,B: U_{i}\)，存在一个典范映射\(\mathtt{idtoeqv}: (A = B) \to (A \simeq B)\)，它是通过对证明\(A = B\)进行路径归纳法\footnote{这只不过是2.1.2节中所描述的J消除器。我们使用这个名称是考虑到将相等性证明看作路径的这种直观概念。 }来定义的：
\[\mathtt{idtoeqv} \left(\mathtt{refl}_{A}\right) \triangleq id_{A}\]
其中\(id_{A}: A \simeq A\)是被视为等价关系的恒等映射。

最后，我们将单值公理形式化为这样一个陈述：上述映射（\(\mathtt{idtoeqv}\)）本身就是一个等价关系。 

定义 2.1.4（沃沃斯基单值公理）。对于全域\(U_{i}\) 的单值公理断言，对于所有\(A, B \in U_{i}\)，映射\(\mathtt{idtoeqv}: (A = B) \to (A \simeq B)\) 是一个等价关系。 

请注意，单值公理是关于特定全域\(U_{i}\)的一个陈述。一般来说，在单值类型论（UTT）中进行研究时，除非另有明确说明，我们假定所有的全域都是单值的。正是这一假设，将单值类型论/同伦类型论与普通的马丁-洛夫类型论区分开来。 

\subsection{函数外延性}
我们现在回顾一下类型论中的函数外延性原则： 

定义2.1.5（函数外延性）。函数外延性原理表明，对于两个函数\(f, g: \prod_{x: A} B(x)\)，只要它们逐点相等，那么这两个函数就相等。也就是说，我们有这样一个映射：
\[f \approx g \to f = g\qquad\text{其中}\qquad f \approx g \triangleq \prod_{x: A} f(x) = g(x)\]
当我们假定存在这样一个项时，我们就说我们“假定了函数外延性”：
\[\mathtt{funext}_{i, j}: \prod_{A: U_{i}} \prod_{B: A \to U_{j}} \prod_{f, g: \Pi_{x: A} B(x)} f \approx g \to f=g\]
对于所有的全域层级\(i\)和\(j\) 。 

请注意，函数外延性无法从标准的马丁-洛夫类型论的规则中推导得出。然而，它却可以从单值公理中推导出来（见参考文献[62]的4.9节）。 

\subsection{单值公理的另一种表述形式}

虽然定义2.1.4非常简洁，但有时使用单值公理的一种略有不同但等价的表述来进行研究工作会更加容易，我们在此给出这种表述。鉴于以下定义，我们有时会将单值公理称为“真正的单值公理”。 

定义 2.1.6（\textbf{Coerce}）。对于所有的\(i\)，以及类型\(A\)、\(B \in U_{i}\)，我们可以通过路径归纳法，或者按照如下方式来定义一个映射\(\mathtt{coerce} : (A = B) \to A \to B\)：
\[\mathtt{coerce}(p, a) \triangleq \mathtt{fst}(\mathtt{idtoeqv}(p))(a)\]
其中\(\mathtt{fst}\)表示取第一个分量的投影操作。 

定义 2.1.7（朴素单值公理）。对于全域\(U_{i}\) 的朴素单值公理表明，对于所有\(A\)、\(B \in U_{i}\)，存在一个从等价关系到相等关系的映射。换句话说，它断言存在一个属于以下类型的元素：
\[\mathtt{UA}_{i} \triangleq \prod_{A, B: U_{i}} A \simeq B \to A = B\] 

当使用项\(ua : \mathtt{UA}_{i}\)时，我们常常会省略前两个参数（\(A\)和\(B\)）。朴素单值公理的证明可能还会带有一个相关的计算规则。也就是说，存在一个属于类型\(\mathtt{UA}\beta_{i}(ua)\)的元素，其中：
\[\mathtt{UA}\beta_{i}(ua) \triangleq \prod_{A, B: U_{i}} \prod_{f: A \to B} \prod_{e: \mathtt{isEquiv}(f)} \mathtt{coerce}(ua(f, e)) = f\] 

接下来，我们给出一个在单值类型论（UTT）/同伦类型论（HoTT）圈子里广为人知的结果。这一结果将真正的单值公理分解为朴素版本和一条计算规则。需注意，该结果要求函数外延性原理成立。首先，我们给出一个引理，它对这一结果的核心构造进行了推广。 

引理2.1.8。给定\(X: U_{i}\)，\(Y: X \to X \to U_{j}\)以及一个映射\(f: \prod_{x, x': X} x = x' \to Y(x, x')\)，那么对于所有的\(x, x': X\)，\(f\ x\ x'\)是一个等价关系，当且仅当存在一个映射
\[g: \prod_{x, x': X} Y\left(x, x'\right) \to x = x'\]
使得对于所有的\(x, x': X\)以及\(y: Y(x, x')\)，我们有\(f(g(y)) = y\)（我们默认省略\(f\)和\(g\)的前两个参数）。 

定理2.1.9。假设函数外延性成立，那么朴素单值公理连同一条计算规则在逻辑上等价于真正的单值公理。也就是说，存在项
\[ua: \mathtt{UA}_{i}, \qquad ua\beta: \mathtt{UA}\beta_{i}(ua)\]
当且仅当对于所有类型\(A, B \in U_{i}\)，映射\(\mathtt{idtoeqv}: (A = B) \to (A \simeq B)\)是一个等价关系。

\section{范畴族（CwFs）}

对于如何呈现依赖类型理论的模型，存在许多不同的概念。在本论文中，我们将采用迪布耶（Dybjer）提出的范畴族（CwF）概念[23]。我们使用的符号与迪布耶的略有不同，并且将他定义中属于族范畴的函子拆分为两个不同的函子，但除此之外，定义保持不变。我们首先回顾一下预层\(P: \C^{op} \to \Set\)的元素范畴的定义。

定义3.1.1（元素范畴）。给定任意预层\(P: \C^{op} \to \Set\)，\(P\)的元素范畴记为\(\int P\)，其对象为形如\((X, p)\)的对，其中\(X \in \C\)且\(p \in P(X)\)。态射\(f: (X, p) \to (X', p')\)就是\(\C\)中从\(X\)到\(X'\)的态射，并且满足\(P(f)(p') = p\)。 

定义3.1.2（范畴族）。一个范畴族由以下数据构成：
\begin{itemize}
  \item 一个上下文范畴$\C$，其对象代表类型论中的上下文，态射代表上下文之间的替换。这个范畴必须有一个终对象$1$，它代表空上下文。 
  \item 一个函子$\Ty:\C^{op}\to \Set$，它给每个上下文$\Gamma$指定一个集合$\Ty(\Gamma)$，该集合表示上下文$\Gamma$中的类型。对于任意态射$\gamma:\Delta\to\Gamma$以及类型$A\in \Ty(\Gamma)$， $\Ty$ 作用于态射的操作给出了替换的解释，我们用 $A[\gamma]$ 来表示 $\Ty(\gamma)(A) \in \Ty(\Delta)$，这样的表示很直观。$\Ty$ 的函子性质确保了替换是严格可结合的。
  \item 一个函子$\Ter: (\int \Ty)^{op} \to \Set$，它为每个上下文$\Gamma$以及$\Gamma$中的类型$A \in \Ty(\Gamma)$分配一个集合$\Ter(\Gamma, A)$，该集合表示在上下文$\Gamma$中类型为$A$的项。我们将$\Ter(\Gamma, A)$这个集合写作$\Ter(\Gamma \vdash A)$ 。同样，$\Ter$作用于态射的方式严格结合地模拟了替换操作。给定$\gamma: \Delta \to \Gamma$、$A \in \Ty(\Gamma)$以及$a \in \Ter(\Gamma \vdash A)$，我们把$\Ter(\gamma)(a) \in \Ter(\Delta \vdash A[\gamma])$写作$a[\gamma]$。
  \item 一种上下文扩展操作$\ul .\ul $，它以一个属于$\C$的上下文$\Gamma$和一个属于$\Ty(\Gamma)$的类型$A$为输入，返回一个属于$\C$的对象$\Gamma.A$。同时存在一个态射$\pp: \Gamma.A \to \Gamma$，它代表上下文弱化；还有一个项$\qq \in \Ter(\Gamma . A \vdash A[\pp])$，代表新添加的类型为$A$的变量。这种操作必须满足以下泛性质：对于任何从$\Delta$到$\Gamma$的替换$\gamma$以及属于$\Ter(\Delta \vdash A[\gamma])$的项$a$，都存在唯一的态射$\langle\gamma, a\rangle: \Delta \to \Gamma . A$，使得$\pp \circ\langle\gamma, a\rangle=\gamma$且$\qq[\langle\gamma, a\rangle]=a$。 
\end{itemize}

\section{立方集模型}
在本节中，我们将概述科恩（Cohen）等人[18]提出的类型论的预层模型，该模型对2.2节中讨论的立方类型论的所有特征进行了建模。 

回想一下，德摩根代数是一种分配格，配备了一个函数\(d \mapsto 1 - d\)，该函数是对合的（即\(1 - (1 - d) = d\)），并且满足德摩根定律\(1 - (d_1 \lor d_2) = (1 - d_1) \land (1 - d_2)\)；详见[9，第十一章]。德摩根代数的同态是一种保持有限交、并以及对合函数的函数。令\(\DM\)表示德摩根代数和同态构成的范畴，\(\dM(I)\)表示集合\(I\)上的自由德摩根代数。

定义3.3.1（立方体范畴，\(\square\)）。固定一个可数无限集\(\mathbb{D}\)，其元素我们称为名称，记作\(i\)，\(j\)，\(k\)，… 。\(\square\)的对象是\(\mathbb{D}\)的有限子集，我们记作\(I\)，\(J\)，\(K\)，… 。态射\(\square(I, J)\)是所有从\(J\)到\(\dM(I)\)的函数。这样的函数与从\(\dM(J)\)到\(\dM(I)\)的德摩根代数同态一一对应，并且对于\(f\in\square(I, J)\)和\(g\in\square(J, K)\)，在\(\square\)中的复合是集合范畴中\(g: K \to \dM(J)\)与对应于\(f\)的从\(\dM(J)\)到\(\dM(I)\)的同态的复合 。

因此，\(\square\)等价于德摩根代数范畴（\(\DM\)）的全子范畴的对偶范畴，该全子范畴的对象是有限生成的自由德摩根代数。 因此，作为一种劳威尔理论（Lawvere theory）[41, 2]，它是德摩根代数的代数理论：它是一个具有有限积的范畴，配备了一个内部德摩根代数（对于某个选定的\(i \in \mathbb{D}\)，其基础对象是\(\{i\}\) ），并且在这类范畴中具有泛性质。 

定义3.3.2。一个立方集是立方体范畴上的一个预层。 

立方集范畴\(\Set^{\square^{op}}\)构成了一个范畴族（CwF）的上下文范畴，这将使我们能够解释立方类型论的所有特征。在后续章节中我们将对这个范畴的某些性质进行公理化处理，并以公理化的方式证明前面的陈述。在本节的剩余部分，我们将非常简要地概述科恩等人在文献[18]中所采用的原始方法。

我们首先引入一些在本节中会用到的符号表示。对于任意属于范畴\(\square\)的符号集合\(I\)以及不属于\(I\)的符号\(i\)，我们将\(I,i\)写作\(I \cup \{i\}\)的缩写形式。我们还将\((i0)\)记为从\(I\)到\(I,i\)的映射，该映射将\(i\)映射到\(0\)，而在其他地方是恒等映射。类似地，我们将\((i1)\)记为从\(I\)到\(I,i\)的映射，它将\(i\)映射到\(1\)。这里\(0\)和\(1\)是自由德摩根代数\(\dM(I)\)中的最小元和最大元。最后，我们将\(s_{i}: I,i \to I\)记为包含映射\(I \subseteq \dM(I, i)\) 。 

在接下来的章节中，我们使用\(\Ty\)、\(\Ter\)和\(\ul .\ul \)来表示立方集范畴上的标准范畴族（CwF），如定义3.2.1中所描述的那样。

\chapter{拓扑斯的内部类型论}

\chapter{在拓扑斯中给出立方类型论模型需要的公理}

在第4章中，我们看到了每个拓扑斯（topos）如何容纳一种基于外延型马丁-洛夫类型论（extensional Martin-Löf type theory）的丰富内部语言。在本论文的其余部分，我们将展示第3.3节中描述的立方类型论模型如何使用这种内部语言来呈现。在此过程中，我们隐式地将Cohen等人[18]给出的模型推广到任何具有适当结构的拓扑斯。为了具体说明这种附加结构，我们采取了一种公理化方法，给出了一组我们要求在内部语言中成立的公理。我们对该模型的表述随后利用了这些公理，而不是假设我们总是在第3.3节描述的拓扑斯中工作。

\subsection{区间$\ttI$}

我们首先公理化建模带有区间对象 $\bbI$ 的类型论所需的结构，如第2.2.1节所述。首先，我们假设存在一个对象 $\ttI:\calU$ ，它将是 $\bbI$ 在模型中的解释。我们假设 $\ttI$ 具有足够的结构来建模区间上的所需运算。具体而言，我们假设 $\ttI$ 配备了态射 $\ttzero,\ttone:1→\ttI$ 和 $\ul \sqcap\ul , \ul \sqcup\ul  : \mathtt{I} \to \mathtt{I} \to \mathtt{I}$ ，满足图5.1中的公理 $\ax{1}$--$\ax{4}$。

公理 $\ax{1}$ 表达了区间 $\ttI$ 在内部是连通的，在这个意义上，其元素的任何可判定子集要么是空的，要么是整个 $\ttI$ 。这意味着如果归纳数据类型中的一条路径在路径的某一点具有某种构造子形式，那么它在任何其他点也具有相同的形式。这在第5.3.3节末尾用于证明拓扑斯中的自然数对象是纤维化的（即表示一个类型），以及纤维化在二元余积下封闭。它还在第5.4节中用于证明粘合构造的性质。与公理 $\ax{2}$ 一起，$\ttI$ 的连通性意味着在 $1+1$  中没有从 $\mathtt{inl}\ \ast$ 到 $\mathtt{inr}\ \ast$ 的路径，因此由这些公理确定的马丁-洛夫类型论的路径模型在逻辑上是非退化的。

公理 $\ax{3}$ 和 $\ax{4}$ 赋予 $\ttI$ 一种连接代数（connection algebra）结构[15]。它们捕捉了实数单位区间 $[0,1]$ 上最小值和最大值运算的一些非常简单的性质，这些性质足以确保单例类型的可缩性（第5.2节），并且与后续公理结合，用于定义纤维化从复合的路径提升（见第5.3.1节）。在[18]的模型中，连接代数结构由区间的格结构给出，取 $\ul \sqcap\ul $ 为二元交（meet），$\ul \sqcup\ul $ 为二元并（join），并利用 $\ttzero$ 和 $\ttone$ 分别是最小元和最大元这一事实。

\subsection{上纤维化命题}

接下来，我们需要将立方类型论中面格 $\mathbb{F}$ 的性质公理化。一种方法是假设存在一个具有特定代数性质的对象来建模 $\mathbb{F}$，就像我们在前一节对 $\mathbb{I}$ 所做的那样。然而，我们采取了另一种方法，基于面格的原始直观，即指定一个类 Kan 填充问题的输入，如第 2.2.4 节所述。

Kan 填充以及子空间集合上的其他上纤维化条件，涉及将定义在子空间上的映射扩展到定义在整个空间上的映射。在这里，我们将``空间的子空间''理解为拓扑斯中对象的子对象。由于子对象由到 $\Omega$ 的态射来分类，因此可以考虑由某些命题一般性地指定的子对象集合。更具体地说，给定命题的一个性质 $\mathtt{cof} : \Omega \to \Omega$，我们得到一个相应的命题集合：
\begin{equation}
\Cof \triangleq \{\varphi : \Omega \mid \mathtt{cof}\ \varphi\}
\end{equation}

利用这种直观，我们将面格 $\mathbb{F}$ 内部化为这样一个命题集合。然后我们可以将这个集合的性质公理化，以便构造立方类型论的模型。这些性质展示在图 5.2 中。公理 $\ax{5}$、$\ax{6}$ 和 $\ax{7}$ 与第 2.2.3 节中给出的语法最后四种情况密切对应，该语法生成面格的元素。在这种表述中，$\ax{7}$ 明确表明合取可能是依赖的，这一点在句法表述中并不清晰。公理 $\ax{8}$ 表明上纤维化命题在 $\mathtt{I}$-索引量词下封闭。与 $\ax{7}$ 中的依赖性一样，这不是显式添加到句法表述的面格中的，而是对于定义 $\mathbb{F}$ 的自由生成项集合恰好成立的事实。公理 $\ax{8}$ 用于证明重新对齐引理（引理 5.3.10），该引理用于弱形式粘合的定义中，以确保诱导的纤维化结构扩展到我们正在``粘合''的族上的纤维化结构。

考虑单态射类 $m : A \rightarrowtail B$，其分类态射
\[
\lambda(y : B) \to \exists(x : A).\ m\,x = y : B \to \Omega
\]
通过 $\Cof \rightarrowtail \Omega$ 分解。我们称这样的单态射为上纤维化。类 Kan 填充性质涉及态射 $A \to X$ 何时可以沿着上纤维化 $m : A \rightarrowtail B$ 扩展。相反，在 $\mathcal{E}$ 的内部语言中工作，我们将考虑定义域在 $\Cof$ 中的部分元何时可以扩展到全定义的元。回想一下，在直觉主义逻辑中，类型 $A$ 的部分元通常由子单例表示，即由满足以下条件的函数 $s : A \to \Omega$ 表示：
\[
\forall(x\,x' : A).\ s\,x \land s\,x' \Rightarrow x = x'
\]
然而，使用外延等价的依赖对 $\varphi : \Omega$ 和 $f : [\varphi] \to A$ 将更加方便，如下一定义所示。命题 $\varphi$ 是部分元的范围；用子单例的术语来说，它等于 $\exists(x : A).\ s\,x$。

\begin{definition}[上纤维化部分元，$\square A$]
我们称类型 $\Cof$ 的元素为上纤维化命题。给定类型 $A : \mathcal{U}$，我们定义 $A$ 的上纤维化部分元类型为：
\begin{equation}
\square A \triangleq (\varphi : \Cof) \times ([\varphi] \to A)
\end{equation}

这样一个部分元 $(\varphi, f) : \square A$ 的扩展是一个元素 $a : A$ 连同以下关系的证明：
\begin{equation}
(\varphi, f) \nearrow a \triangleq \forall(u : [\varphi]).\ f\,u = a
\end{equation}
\end{definition}

注意，通过在公理 $\ax{5}$ 中取 $i = 0$，我们有 $\mathtt{cof}(0 = 0)$（即 $\mathtt{cof}\,\top$）和 $\mathtt{cof}(0 = 1)$；将后者与公理 $\ax{2}$ 结合，我们还推导出 $\mathtt{cof}\,\bot$ 成立。因此 $\top$ 和 $\bot$ 都是上纤维化命题，对应于生成面格元素的语法中的前两种情况（第 2.2.3 节）。所以 $A \rightarrowtail A$ 和 $\emptyset \rightarrowtail A$ 总是上纤维化，其中 $\emptyset$ 是始对象。由于 $\mathtt{cof}\,\top$ 成立，对于每个 $a : A$ 存在一个全上纤维化部分元 $(\top, \lambda\ul  \to a) : \square A$，其中 $a$ 是扩展 $(\top, \lambda\ul  \to a)$ 的唯一元素。由于 $\mathtt{cof}\,\bot$ 成立，每个对象 $A$ 都有一个由 $(\bot, \mathtt{elim}_\emptyset) : \square A$ 给出的空上纤维化部分元，使得每个 $a : A$ 都是 $(\bot, \mathtt{elim}_\emptyset)$ 的扩展。（对于任何 $B : \mathcal{U}$，$\mathtt{elim}_\emptyset : [\bot] \to B$ 表示由 $[\bot]$ 的始性给出的唯一函数。）

\begin{definition}[相容部分元的并]
称两个部分元 $f : [\varphi] \to A$ 和 $g : [\psi] \to A$ 是相容的，如果它们在都有定义的地方一致：
\begin{equation}
(\varphi, f) \smallsmile (\psi, g) \triangleq \forall(u : [\varphi])(v : [\psi]).\ f\,u = g\,v
\end{equation}

在这种情况下，我们可以形成它们的并 $f \cup g : [\varphi \lor \psi] \to A$，使得：
\[
\forall(u : [\varphi]).\ (f \cup g)\,u = f\,u \qquad \forall(v : [\psi]).\ (f \cup g)\,v = g\,v
\]

为了理解为什么，考虑拓扑斯中的如下推出方：
\[
\begin{tikzcd}
{[\varphi \land \psi]} \arrow[r, tail] \arrow[d,tail] \arrow[dr, phantom, "\ul corner" very near end] & {[\psi]} \arrow[d,tail] \arrow[ddr, bend left, "g"] \\
{[\varphi]} \arrow[r, tail] \arrow[drr, bend right, "f"'] & {[\varphi \lor \psi]} \arrow[dr, dashed, "f \cup g" description] \\
& & A
\end{tikzcd}
\]

外方交换是因为 $(\varphi, f) \smallsmile (\psi, g)$ 成立，然后 $f \cup g$ 是从推出诱导的唯一态射。注意，图 5.2 中的公理 $\ax{6}$ 意味着上纤维化部分元的集合在取相容部分元的二元并下封闭。
\end{definition}

以下引理给出了公理 $\ax{7}$ 和 $\ax{8}$ 的另一种刻画。由于我们在上面注意到 $\mathtt{cof}\,\top$ 成立，引理的第 (i) 部分告诉我们上纤维化在综合域论的意义上形成一个优势（dominance）[56]；我们仅在从命题相等类型构造定义性相等类型时使用 $\Cof$ 的这一性质（见第 5.3.4 节）。

\begin{lemma}
\begin{enumerate}
\item[(i)] 公理 $\ax{7}$ 等价于要求上纤维化类在复合下封闭。
\item[(ii)] 公理 $\ax{8}$ 等价于要求上纤维化类在由 $\mathtt{I}$ 的指数化下封闭。
\end{enumerate}
\end{lemma}

\begin{proof}
对于第 (i) 部分，首先假设 $\ax{7}$ 成立，且 $f : A \rightarrowtail B$ 和 $g : B \rightarrowtail C$ 是上纤维化。因此 $\forall(b : B).\ \mathtt{cof}(\exists(a : A).\ f\,a = b)$ 和 $\forall(c : C).\ \mathtt{cof}(\exists(b : B).\ g\,b = c)$ 都成立，我们希望证明 $\forall(c : C).\ \mathtt{cof}(\exists(a : A).\ g(f\,a) = c)$。注意对于 $b : B$ 和 $c : C$：
\begin{align*}
g\,b = c &\Rightarrow (\exists(a : A).\ g(f\,a) = c) = (\exists(a : A).\ f\,a = b) \quad \text{因为 } g \text{ 是单态射} \\
&\Rightarrow \mathtt{cof}(\exists(a : A).\ g(f\,a) = c) = \mathtt{cof}(\exists(a : A).\ f\,a = b) = \top
\end{align*}

因此令 $\varphi \triangleq \exists(b : B).\ g\,b = c$ 和 $\psi \triangleq \exists(a : A).\ g(f\,a) = c$，我们有 $\mathtt{cof}\,\varphi$ 和 $\varphi \Rightarrow \mathtt{cof}\,\psi$。因此由 $\ax{7}$ 我们得到 $\mathtt{cof}(\varphi \land \psi)$，由于 $\psi \Rightarrow \varphi$，这等于 $\mathtt{cof}(\psi)$。所以我们确实有 $\forall(c : C).\ \mathtt{cof}(\exists(a : A).\ g(f\,a) = c)$。

反之，假设上纤维化在复合下封闭，且 $\varphi, \psi : \Omega$ 满足 $\mathtt{cof}\,\varphi$ 和 $\varphi \Rightarrow \mathtt{cof}\,\psi$。$\mathtt{cof}\,\varphi$ 成立等价于单态射 $[\varphi] \rightarrowtail 1$ 是上纤维化；并且由于
\[
\varphi \Rightarrow (\psi = \varphi \land \psi) \Rightarrow (\mathtt{cof}\,\psi = \mathtt{cof}(\varphi \land \psi))
\]
由 $\varphi \Rightarrow \mathtt{cof}\,\psi$ 我们得到 $\varphi \Rightarrow \mathtt{cof}(\varphi \land \psi)$，因此单态射 $[\varphi \land \psi] \rightarrowtail [\varphi]$ 是上纤维化。复合这些单态射，我们有 $[\varphi \land \psi] \rightarrowtail 1$ 是上纤维化，即 $\mathtt{cof}(\varphi \land \psi)$ 成立。

对于第 (ii) 部分，首先假设 $\ax{8}$ 成立，且 $f : A \rightarrowtail B$ 是上纤维化。我们必须证明 $\mathtt{I} \to f : (\mathtt{I} \to A) \rightarrowtail (\mathtt{I} \to B)$ 也是上纤维化。给定 $\beta : \mathtt{I} \to B$ 我们有
\begin{align*}
&(\forall i : \mathtt{I})(\exists a : A).\ f\,a = \beta\,i \\
&\Rightarrow (\forall i : \mathtt{I})(\exists! a : A).\ f\,a = \beta\,i \quad \text{（因为 } f \text{ 是单态射）} \\
&\Rightarrow (\exists \alpha : \mathtt{I} \to A)(\forall i : \mathtt{I}).\ f(\alpha\,i) = \beta\,i \quad \text{（由拓扑斯中的唯一选择）} \\
&\Rightarrow (\forall i : \mathtt{I})(\exists a : A).\ f\,a = \beta\,i
\end{align*}

因此 $(\forall i : \mathtt{I})(\exists a : A).\ f\,a = \beta\,i$ 等于 $(\exists \alpha : \mathtt{I} \to A)(\forall i : \mathtt{I}).\ f(\alpha\,i) = \beta\,i$；并且后者由拓扑斯中的函数外延性等于 $(\exists \alpha : \mathtt{I} \to A).\ (\mathtt{I} \to f)\,\alpha = \beta$。由于 $f$ 是上纤维化，对于每个 $i : \mathtt{I}$ 我们有 $\mathtt{cof}(\exists(a : A).\ f\,a = \beta\,i)$。因此由公理 $\ax{8}$ 我们也有 $\mathtt{cof}((\forall i : \mathtt{I})(\exists a : A).\ f\,a = \beta\,i)$，即 $\mathtt{cof}(\exists \alpha : \mathtt{I} \to A).\ (\mathtt{I} \to f)\,\alpha = \beta)$，正如 $\mathtt{I} \to f$ 为上纤维化所要求的。

反之，假设上纤维化在 $\mathtt{I} \to (\ul )$ 下封闭，且 $\varphi : \mathtt{I} \to \Omega$ 满足 $(\forall i : \mathtt{I}).\ \mathtt{cof}(\varphi\,i)$。后者意味着 $\{i : \mathtt{I} \mid \varphi\,i\} \rightarrowtail \mathtt{I}$ 是上纤维化。因此单态射 $(\mathtt{I} \to \{i : \mathtt{I} \mid \varphi\,i\}) \rightarrowtail (\mathtt{I} \to \mathtt{I})$ 也是。由于 $id : \mathtt{I} \to \mathtt{I}$ 在这个单态射的像中当且仅当 $(\forall i : \mathtt{I}).\ \varphi\,i$ 成立，我们有 $\mathtt{cof}((\forall i : \mathtt{I}).\ \varphi\,i)$，正如公理 $\ax{8}$ 所要求的。
\end{proof}

\subsection{严格性公理}

我们的最后一个公理与其他公理不同，它与其说是关于公理化立方类型论的某些本质方面，不如说是关于解决内部语言方法的一个弱点。这最后一个公理 $\ax{9}$ 指出，任何上纤维化部分类型 $A$，如果在 $A$ 有定义的任何地方都与全类型 $B$ 同构，则可以扩展为一个与 $B$ 同构的全类型 $B'$。该公理的精确定义在图 5.3 中给出，同构的定义如下：

\begin{definition}[同构]
给定对象 $A, B : \mathcal{U}$，$A$ 和 $B$ 之间的同构是一个具有双边逆的函数 $f : A \to B$。令 $A \cong B$ 为 $A$ 和 $B$ 之间同构的类型，定义为：
\[
A \cong B \triangleq \{f : A \to B \mid (\exists g : B \to A)\, (g \circ f = id) \land (f \circ g = id)\}
\]

我们说 $A$ 和 $B$ 同构，如果存在同构 $f : A \cong B$。我们说两个族 $A, B : \Gamma \to \mathcal{U}$ 同构，如果它们的每个纤维都同构，并且我们滥用记号写作：
\[
A \cong B \triangleq (x : \Gamma) \to A\,x \cong B\,x
\]
\end{definition}

同构在内部类型论的外延相等意义下具有逆，这与我们后面引入的等价概念形成对比，后者仅在路径相等意义下具有逆。此外，这些逆是唯一的，因此，使用唯一选择，我们总能构造同构的逆。因此，给定 $f : A \cong B$，我们将 $f^{-1} : B \cong A$ 写作 $f$ 的这个逆。

为了对这个公理的作用获得一些直观理解，考虑以下例子。我们有一个在区间上的类型族 $B : \mathtt{I} \to \mathcal{U}$ 和一个单一类型 $A : \mathcal{U}$ 使得 $A \cong B\,0$。我们希望能够在某种意义上``覆盖'' $B$ 在 $0$ 处的值，用 $A$ 替换它。这应该给我们一个族 $B' : \mathtt{I} \to \mathcal{U}$，它与原始族 $B$ 同构，但满足 $A = B'\,0$。给定公理 $\ax{9}$，我们可以将 $B'\,i : \mathcal{U}$ 定义为 $\ax{9}\,\{i=0\}\,(\lambda\ul  \to A)\,(B\,i)\,(\lambda\ul  \to s)$ 的第一个投影，其中 $s : A \cong B\,0$ 是见证 $A$ 在 $0$ 处与 $B$ 同构的同构。第二个投影给出新的同构，见证 $B'$ 与 $B$ 同构，特别是当 $i = 0$ 时这个新的同构就是原来的那个。

先验地，没有公理 $\ax{9}$，在内部类型论中没有办法执行这样的构造。事实上，从范畴论的角度来看，这样的构造似乎相当奇怪，因为我们通常只在``同构意义下''工作。然而，这个公理后面将需要用于验证类型论中必须成立的某些定义性相等。在这个意义上，对公理 $\ax{9}$ 的需求让人想起在范畴模型中将类型论中的替换解释为拉回时出现的一些问题。

公理 $\ax{9}$ 将用于恢复 Cohen 等人 [18] 使用的严格形式的粘合。它在预层模型中的有效性依赖于外部元理论中的一个构造，该构造无法在内部复制；详见定理 5.6.3。

\section{路径类型}

给定 $A : \mathcal{U}$，我们称类型 $\mathtt{I} \to A$ 的元素为 $A$ 中的路径。与 $A$ 关联的路径类型是 $\ul \sim\ul  : A \to A \to \mathcal{U}$，其中
\begin{equation}
a_0 \sim a_1 \triangleq \{p : \mathtt{I} \to A \mid p\,0 = a_0 \land p\,1 = a_1\}
\end{equation}

这些类型能否用于建模马丁-洛夫相等类型的规则？我们当然可以解释相等引入规则（自反性），因为由常数函数给出的退化路径
\begin{equation}
\mathtt{k}\,a\,i \triangleq a
\end{equation}
满足 $\mathtt{k} : \{A : \mathcal{U}\}(a : A) \to a \sim a$。然而，我们需要进一步的假设来解释相等消去规则，也称为路径归纳，在第 2.1.2 节中描述。Coquand 给出了相等消去的另一种（命题等价的）表述，用替换函数 $a_0 \sim a_1 \to P\,a_0 \to P\,a_1$ 和单例类型 $(a_1 : A) \times (a_0 \sim a_1)$ 的可缩性来表述；见 [11,图2]。连接代数结构给出后者，因为使用 $\ax{3}$ 和 $\ax{4}$ 我们有

&\mathtt{ctr} : \{A : \mathcal{U}\}\{a_0\,a_1 : A\}(p : a_0 \sim a_1) \to (a_0, \mathtt{k}\,a_0) \sim (a_1, p) \\

\section{Cohen-Coquand-Huber-Mörttberg（CCHM）纤维化}

在本节中，我们将展示如何把第 3.3.4 节中针对特定预层模型引入的纤维化概念推广到任何具有区间对象的拓扑斯，如前几节所述。为此，我们使用图 5.4 中的\textbf{上纤维化命题}概念来内部化文献 [18] 中描述的复合和填充运算。我们证明这个概念在形成 $\Sigma$-、$\Pi$- 和路径类型以及基本数据类型下封闭；最后我们展示如何定义具有定义性计算规则的等同类型，即 $\mathtt{Id}$-类型，并证明纤维化在形成 $\mathtt{Id}$-类型下也封闭。

\subsection{复合与填充结构}

给定区间索引的类型族 $A : \mathtt{I} \to \mathcal{U}$，我们将依赖函数类型 $\Pi_\mathtt{I} A \triangleq (i : \mathtt{I}) \to A\,i$ 的元素视为\textbf{依赖类型路径}。我们称类型 $\square(\Pi_\mathtt{I} A)$ 的元素为\textbf{上纤维化部分路径}。给定 $(\varphi, f) : \square(\Pi_\mathtt{I} A)$，我们可以在区间的点 $i : \mathtt{I}$ 处求值，得到一个上纤维化部分元 $(\varphi, f) \ @\  i : \square(A\,i)$：
\begin{equation}
(\varphi, f) \ @\  i \triangleq (\varphi, \lambda(u : [\varphi]) \to f\,u\,i)
\end{equation}

在 $A : \mathtt{I} \to \mathcal{U}$ 中\textbf{从 $\ttzero$ 填充}的运算接受任何 $(\varphi, f) : \square(\Pi_\mathtt{I} A)$ 和任何 $a_0 : A\,\ttzero$ 使得 $(\varphi, f) \ @\  \ttzero \nearrow a_0$，并将 $(\varphi, f)$ 扩展为依赖类型路径 $g : \Pi_\mathtt{I} A$ 且满足 $g\,\ttzero = a_0$。这是一种\textbf{均匀同伦扩展与提升性质（HELP）}[49, 第 10 章, 第 3 节] 的形式，用\textbf{上纤维化命题}内部陈述，而非用\textbf{上纤维化}外部陈述。与 Cohen 等人相比，我们内部方法的一个特点是，他们对复合/填充运算的\textbf{均匀性}条件（定义 3.3.4）---这使得可以避免经典 Kan 填充概念 [10] 的非构造性方面---当运算用上纤维化命题的内部集合 $\Cof$ 表述时自动成立。

由于我们不假设区间具有反转路径的任何结构（见备注 5.1.1），我们还需要考虑从 $\ttone$ 填充的对称概念。令
\begin{equation}
\{\ttzero, \ttone\} \triangleq \{i : \mathtt{I} \mid i = \ttzero \lor i = \ttone\}
\end{equation}
注意，由于公理 $\ax{2}$，这与布尔值对象 $1 + 1$ 同构，因此存在一个函数
\begin{equation}
\overline{\phantom{x}} : \{\ttzero, \ttone\} \to \{\ttzero, \ttone\}
\end{equation}
满足 $\overline{\ttzero} = \ttone$ 和 $\overline{\ttone} = \ttzero$。在下文中，我们不使用路径反转，而是用 $e : \{\ttzero, 1\}$ 参数化定义，并使用 (5.10) 来交换 $\ttzero$ 和 $\ttone$。

\begin{definition}[填充结构]
给定 $e : \{0, 1\}$，类型 $\mathtt{Fill}\,e\,A : \mathcal{U}$ 表示 $\mathtt{I}$-索引类型族 $A : \mathtt{I} \to \mathcal{U}$ 的\textbf{填充结构}，定义为：
\begin{align}
\mathtt{Fill}\,e\,A \triangleq &(\varphi : \Cof)(f : [\varphi] \to \Pi_\mathtt{I} A)(a : \{a' : A\,e \mid (\varphi, f) \ @\  e \nearrow a'\}) \to \\
&\{g : \Pi_\mathtt{I} A \mid (\varphi, f) \nearrow g \land g\,e = a\}\notag
\end{align}
\end{definition}

\begin{example}
为了直观理解为什么这样的运算被称为填充，考虑以下例子。为简单起见，假设 $A : \mathtt{I} \to \mathcal{U}$ 是一个常值族 $A \triangleq \lambda(\ul  : \mathtt{I}) \to A'$。回想一下，我们将类型 $\mathtt{I}$ 的变量视为空间中的维度；因此，在给定环境上下文 $j, k : \mathtt{I}$ 中的元素 $a : A$，我们将 $a$ 视为空间 $A$ 中的一个正方形。我们希望将这个二维正方形扩展为三维立方体，如下所示。

\begin{definition}[CCHM 纤维化]
类型 $\Gamma : \mathcal{U}$ 上的 \textbf{CCHM 纤维化} $(A, \alpha)$ 是一个配备纤维化结构 $\alpha : \mathtt{isFib}\,A$ 的族 $A : \Gamma \to \mathcal{U}$，其中 $\mathtt{isFib} : \{\Gamma : \mathcal{U}\}(A : \Gamma \to \mathcal{U}) \to \mathcal{U}$ 定义为：
\begin{equation}
\mathtt{isFib}\,\{\Gamma\}\,A \triangleq (e : \{0, 1\})(p : \mathtt{I} \to \Gamma) \to \mathtt{Comp}\,e\,(A \circ p)
\end{equation}

这里 $\mathtt{Comp} : (e : \{0, 1\})(A : \mathtt{I} \to \mathcal{U}) \to \mathcal{U}$ 是 $\mathtt{I}$-索引族的\textbf{复合结构}类型：
\begin{align}
\mathtt{Comp}\,e\,A \triangleq &(\varphi : \Cof)(f : [\varphi] \to \Pi_\mathtt{I} A) \to \\
&\{a_0 : A\,e \mid (\varphi, f) \ @\  e \nearrow a_0\} \to \{a_1 : A\,\overline{e} \mid (\varphi, f) \ @\  \overline{e} \nearrow a_1\}\notag
\end{align}
\end{definition}

\begin{example}

\begin{definition}[CCHM 纤维化的 CwF]
令 $\mathtt{Fib}\,\Gamma$ 为对象 $\Gamma$ 上 CCHM 纤维化的类型，定义为：
\begin{equation}
\mathtt{Fib}\,\Gamma \triangleq (A : \Gamma \to \mathcal{U}) \times \mathtt{isFib}\,A
\end{equation}

CCHM 纤维化在重索引下封闭：给定 $\gamma : \Delta \to \Gamma$ 和 $A : \Gamma \to \mathcal{U}$，我们得到一个函数 $\ul [\gamma] : \mathtt{isFib}\,A \to \mathtt{isFib}(A \circ \gamma)$，定义为 $\alpha[\gamma]\,e\,p \triangleq \alpha\,e\,(\gamma \circ p)$。因此我们得到一个函数 $\ul [\ul ] : \mathtt{Fib}\,\Gamma \to (\Delta \to \Gamma) \to \mathtt{Fib}\,\Delta$，定义为：
\begin{equation}
(A, \alpha)[\gamma] \triangleq (A \circ \gamma, \alpha[\gamma])
\end{equation}

这是函子性的：$(A, \alpha)[id] = (A, \alpha)$ 且 $(A, \alpha)[g \circ f] = (A, \alpha)[g][f]$。由此可得，$\mathtt{Fib}$ 具有\textbf{带族的范畴（Category with Families）}的结构，通过将族取为每个 $\Gamma : \mathcal{U}$ 上的 CCHM 纤维化 $(A, \alpha) : \mathtt{Fib}\,\Gamma$，并将这种族的元素取为 $(x : \Gamma) \to A\,x$ 中的依赖函数。
\end{definition}

\begin{remark}[纤维化对象]
我们说 $A : \mathcal{U}$ 是一个\textbf{纤维化对象}，如果我们在终对象 $1$ 上的常值族 $\lambda(\ul  : 1) \to A$ 有一个纤维化结构。注意，如果 $(A, \alpha) : \mathtt{Fib}\ \Gamma$ 是一个纤维化，那么对于每个 $x : \Gamma$，类型 $A\ x : \mathcal{U}$ 是纤维化的，其纤维化结构由通过映射 $\lambda(\ul  : 1) \to x : 1 \to \Gamma$ 重索引 $\alpha$ 给出。然而，其逆不成立：拥有一族纤维化结构，即 $(x : \Gamma) \to \mathtt{isFib}(\lambda(\ul  : 1) \to A\ x)$ 的元素，比拥有 $A : \Gamma \to \mathcal{U}$ 的纤维化结构要弱。为了理解为什么，考虑由以下定义的族 $P : \mathtt{I} \to \mathcal{U}$：
\begin{equation}
P\ i \triangleq [0 = i]
\end{equation}

对于每个 $i : \mathtt{I}$，纤维 $P\ i : \mathcal{U}$ 是一个纤维化对象，具有纤维化结构 $\rho_i : \mathtt{isFib}(\lambda(\ul  : 1) \to P\ i)$，由 $\rho_i\ e\ p\ \varphi\ f\ x \triangleq x$ 给出。然而，不可能构造 $\rho : \mathtt{isFib}\ P$。因为如果可能，我们可以定义 $\rho\ 0\ id\ \bot\ \mathtt{elim}_\emptyset\ \ast : [0 = 1]$；结合 $\ax{2}$，这将导致矛盾。这个例子将在后面第 7.1 节中更详细地探讨。
\end{remark}

如果 $\alpha : \mathtt{Fill}\ e\ A$，那么 $\lambda\varphi\ f\ a \to \alpha\ \varphi\ f\ a\ \overline{e} : \mathtt{Comp}\ e\ A$，因此每个填充结构都产生一个复合结构。反之，CCHM 纤维化的复合结构产生填充结构：

\begin{lemma}[从复合结构构造填充结构]
给定 $\Gamma : \mathcal{U}$，$A : \Gamma \to \mathcal{U}$，$e : \{0, 1\}$，$\alpha : \mathtt{isFib}\ A$ 和 $p : \mathtt{I} \to \Gamma$，存在一个填充结构 $\mathtt{fill}\ e\ \alpha\ p : \mathtt{Fill}\ e\ (A \circ p)$，它在 $\overline{e}$ 处与 $\alpha$ 一致，即：
\begin{align}
\forall(\varphi : \Cof)(f : [\varphi] \to \Pi_\mathtt{I} A)(a : A&(p\ e)).\notag\\
&(\varphi, f) \ @\  e \nearrow a \Rightarrow \mathtt{fill}\ e\ \alpha\ p\ \varphi\ f\ a\ \overline{e} = \alpha\ e\ p\ \varphi\ f\ a
\end{align}

此外，$\mathtt{fill}$ 在重索引下是\textbf{稳定的}，在这个意义上，对于所有 $\gamma : \Delta \to \Gamma$ 和 $p : \mathtt{I} \to \Delta$：
\begin{equation}
\mathtt{fill}\ e\ \alpha\ (\gamma \circ p) = \mathtt{fill}\ e\ (\alpha[\gamma])\ p
\end{equation}
\end{lemma}

\begin{proof}
从复合构造填充遵循 [18, 第 4.4 节]，但仅使用 $\mathtt{I}$ 上的连接代数结构（公理 $\ax{3}$ 和 $\ax{4}$），而非德摩根代数结构。假设 $\Gamma : \mathcal{U}$，$A : \Gamma \to \mathcal{U}$，$e : \{0, 1\}$，$\alpha : \mathtt{isFib}\ A$，$p : \mathtt{I} \to \Gamma$，$\varphi : \Cof$，$f : [\varphi] \to \Pi_\mathtt{I}(A \circ p)$，$a : A(p\ e)$ 满足 $(\varphi, f) \ @\  e \nearrow a$，以及 $i : \mathtt{I}$。然后使用定义 5.1.4 我们可以定义：
\begin{equation}
\mathtt{fill}\ e\ \alpha\ p\ \varphi\ f\ a\ i \triangleq \alpha\ e\ (p'\ i)\ (\varphi \lor i = e)\ (f'\ i \cup g\ i)\ a
\end{equation}

其中
\begin{align*}
&p' : \mathtt{I} \to \mathtt{I} \to \Gamma \\
&p'\ i\ j \triangleq p(i \sqcap_e j) \\
&f' : (i : \mathtt{I}) \to [\varphi] \to \Pi_\mathtt{I}(A \circ (p'\ i)) \\
&f'\ i\ u\ j \triangleq f\ u\ (i \sqcap_e j) \\
&g : (i : \mathtt{I}) \to \{g' : [i = e] \to \Pi_\mathtt{I}(A \circ (p'\ i)) \mid (\varphi, f'\ i) \smallsmile  (i = e, g')\} \\
&g\ i\ v\ j \triangleq a
\end{align*}

其中 $\sqcap_e$ 由 $\sqcap_0 \triangleq \sqcap$ 和 $\sqcap_1 \triangleq \sqcup$ 给出。最后，性质 (5.18) 从定义 (5.15) 和 (5.19) 立即可得。
\end{proof}

与 [11] 相比，填充可以从复合定义这一事实大大简化了通过通常类型形成构造提升纤维化结构的过程，如下两个定理所展示的。它们的证明是 [18, 第 4.5 节] 中证明的内部化，只是我们避免了 Cohen 等人使用的德摩根对合。


\subsection{纤维化的性质}

我们现在证明一些关于纤维化的性质，这些性质在后续会很有用。首先，我们证明纤维化类在同构下是封闭的。我们说两个纤维化 $(A, \alpha)$ 和 $(B, \beta) : \mathtt{Fib}\ \Gamma$ 是同构的，如果其底层族 $A$ 和 $B$ 是同构的，我们滥用记号写作
\[
(A, \alpha) \cong (B, \beta) \triangleq A \cong B
\]
其中 $A \cong B$ 如定义 5.1.6 中所述。

\begin{lemma}
给定一个族 $A : \Gamma \to \mathcal{U}$ 和一个纤维化 $(B, \beta) : \mathtt{Fib}\ \Gamma$，使得 $A \cong B$，那么我们可以构造 $\alpha$ 使得 $(A, \alpha) : \mathtt{Fib}\ \Gamma$。
\end{lemma}

\begin{proof}
假设我们给定上述的 $A$ 和 $(B, \beta)$，以及一个同构 $f : A \simeq B$。我们可以如下定义 $A$ 的复合结构：
\[
\alpha\ e\ p\ \varphi\ q\ a_0 \triangleq f^{-1}\ (p\ \overline{e})\ (\beta\ e\ p\ \varphi\ (\lambda u\ i \to f\ (p\ i)\ (q\ u\ i))\ (f\ (p\ e)\ a_0))
\]
这个构造具有如下所需性质：给定 $u : [\varphi]$：
\begin{align*}
\alpha\ e\ p\ \varphi\ q\ a_0 &= f^{-1}\ (p\ \overline{e})\ (\beta\ e\ p\ \varphi\ (\lambda u\ i \to f\ (p\ i)\ (q\ u\ i))\ (f\ (p\ e)\ a_0)) \\
&= f^{-1}\ (p\ \overline{e})\ (f\ (p\ \overline{e})\ (q\ u\ \overline{e})) \\
&= q\ u\ \overline{e}
\end{align*}
因此 $(\varphi, q)\ @\ \overline{e} \nearrow \alpha\ e\ p\ \varphi\ q\ a_0$，从而 $(A, \alpha) : \mathtt{Fib}\ \Gamma$。
\end{proof}

注意，这个证明只使用了 $f^{-1}\ x \circ f\ x = id$ 这一事实，因此事实上该引理在更一般的情况下也成立，即当 $\langle f, f^{-1}\rangle$ 只是一个收缩-回缩对而非完整的同构时。不过在本文中，我们只在同构的语境下使用它。

接下来，我们证明可以对一个族上的复合结构进行\textbf{调整}或\textbf{重新对齐}。首先，我们引入\textbf{上纤维-部分类型族}的概念：

\begin{definition}[上纤维-部分族]
给定一个对象 $\Gamma : \mathcal{U}$ 和一个上纤维性质 $\Phi : \Gamma \to \Cof$，定义 $\Gamma$ 被 $\Phi$ \textbf{限制}为
\begin{equation}
\Gamma|\Phi \triangleq (x : \Gamma) \times [\Phi\ x]
\end{equation}
因此 $\Gamma|\Phi : \mathcal{U}$，并且存在一个单射 $\iota : \Gamma|\Phi \rightarrowtail \Gamma$，由第一投影给出。（注意 $\Gamma|\Phi$ 同构于理解子类型 $\{x : \Gamma \mid \Phi\ x\}$，但我们使用上述表示以使各种构造中 $\Phi\ x$ 的证明更加明确。）然后，给定一个对象 $\Gamma : \mathcal{U}$ 和一个上纤维性质 $\Phi : \Gamma \to \Cof$，$\Gamma$ 上的\textbf{上纤维-部分类型族}是一个在限制 $\Gamma|\Phi$ 上的类型族 $A$，即 $A : (\Gamma|\Phi) \to \mathcal{U}$。
\end{definition}

\begin{lemma}[Realignment 引理]
给定 $\Gamma : \mathcal{U}$ 和 $\Phi : \Gamma \to \Cof$，令 $\iota : \Gamma|\Phi \rightarrowtail \Gamma$ 为第一投影。对于任意 $A : \Gamma \to \mathcal{U}$，$\beta : \mathtt{isFib}(A \circ \iota)$ 和 $\alpha : \mathtt{isFib}\ A$，存在一个复合结构 $\mathtt{realign}(\Phi, \beta, \alpha) : \mathtt{isFib}\ A$ 使得 $\beta = \mathtt{realign}(\Phi, \beta, \alpha)[\iota]$。
\end{lemma}

\begin{proof}
给定上述的 $\Gamma, \Phi, A, \beta, \alpha$，使用公理 $\ax{8}$（和 $\ax{6}$）我们可以定义 $\mathtt{realign}(\Phi, \beta, \alpha)$ 为
\begin{equation}
\mathtt{realign}(\Phi, \beta, \alpha)\ e\ p\ \psi\ f\ g \triangleq \alpha\ e\ p\ (\psi \lor (\forall(i : \mathtt{I}).\ \Phi\ (p\ i)))\ (f \cup f')\ g
\end{equation}
其中 $f' : [\forall(i : \mathtt{I}).\ \Phi\ (p\ i)] \to \Pi_{\mathtt{I}}(A \circ p)$ 由 $f'\ u \triangleq \mathtt{fill}\ e\ \beta\ (\lambda i \to (p, u\ i))\ \psi\ f\ g$ 给出。$f$ 和 $f'$ 相容的事实来自于我们在 $f'$ 的定义中使用了 $\psi$ 和 $f$，因此根据填充的性质，$f'$ 必须在它们共同定义的地方与 $f$ 一致。
\end{proof}

用话来说，给定一个纤维化类型 $A$ 和一个只在 $A$ 的某个限制上定义的替代复合结构，我们可以\textbf{重新对齐}原始复合结构，使其在该限制上与替代结构一致。

注意，这个构造在重索引下是稳定的，在这个意义上，给定以下数据：
\[
\gamma : \Delta \to \Gamma, \quad \Phi : \Gamma \to \Cof, \quad A : \Gamma \to \mathcal{U}, \quad \beta : \mathtt{isFib}(A \circ \iota), \quad \alpha : \mathtt{isFib}\ A
\]
那么，
\begin{align*}
&\mathtt{realign}(\Phi, \beta, \alpha)[\gamma]\ e\ p\ \psi\ f\ a \\
&= \mathtt{realign}(\Phi, \beta, \alpha)\ e\ (\gamma \circ p)\ \psi\ f\ a \\
&= \alpha\ e\ (\gamma \circ p)\ (\psi \lor (\forall(i : \mathtt{I}).\ \Phi\ ((\gamma \circ p)\ i)))\ (f \cup \mathtt{fill}\ e\ \beta\ (\lambda i \to (\gamma \circ p, u\ i))\ \psi\ f\ a)\ a \\
&= \alpha[\gamma]\ e\ p\ (\psi \lor (\forall(i : \mathtt{I}).\ (\Phi \circ \gamma)(p\ i)))\ (f \cup \mathtt{fill}\ e\ \beta[\gamma \times  id ]\ (\lambda i \to (p, u\ i))\ \psi\ f\ a)\ a \\
&= \mathtt{realign}(\Phi \circ \gamma, \beta[\gamma \times  id ], \alpha[\gamma])\ e\ p\ \psi\ f\ a
\end{align*}
因此我们有
\[
\mathtt{realign}(\Phi, \beta, \alpha)[\gamma] = \mathtt{realign}(\Phi \circ \gamma, \beta[\gamma \times  id ], \alpha[\gamma])
\]

\subsection{类型构造子和简单数据类型}

\begin{theorem}[纤维化 $\Sigma$-类型]
存在一个函数
\begin{align}
\mathtt{isFib}_\Sigma : &\{\Gamma : \mathcal{U}\}\{A_1 : \Gamma \to \mathcal{U}\}\{A_2 : (x : \Gamma) \times A_1\ x \to \mathcal{U}\} \to \\
&\mathtt{isFib}\ A_1 \to \mathtt{isFib}\ A_2 \to \mathtt{isFib}(\Sigma\,A_1\,A_2)\notag
\end{align}
其中 $\Sigma\,A_1\,A_2\ x \triangleq (a_1 : A_1\ x) \times A_2(x, a_1)$。该函数在重索引下是稳定的，在这个意义上，对于所有 $\gamma : \Delta \to \Gamma$
\begin{equation}
(\mathtt{isFib}_\Sigma\ \alpha_1\ \alpha_2)[\gamma] = \mathtt{isFib}_\Sigma(\alpha_1[\gamma])(\alpha_2[\gamma \times  id ])
\end{equation}
因此，由 CCHM 纤维化给出的带族范畴支持 $\Sigma$-类型的解释 [30, 定义 3.18]。
\end{theorem}

\begin{proof}
$\mathtt{isFib}_\Sigma$ 的构造使用了引理 5.3.7 中的填充操作。给定 $\Gamma : \mathcal{U}$，$A_1 : \Gamma \to \mathcal{U}$，$A_2 : (x : \Gamma) \times A_1\ x \to \mathcal{U}$，$\alpha_1 : \mathtt{isFib}\ A_1$，$\alpha_2 : \mathtt{isFib}\ A_2$，$e : \{0,1\}$，$p : \mathtt{I} \to \Gamma$，$\varphi : \mathtt{Cof}$，$f : [\varphi] \to \Pi_{\mathtt{I}}((\Sigma\,A_1\,A_2) \circ p)$ 以及 $(a_1, a_2) : (\Sigma\,A_1\,A_2)(p\ e)$ 且满足 $(\varphi, f)\ @\ e \nearrow (a_1, a_2)$，定义
\begin{equation}
\mathtt{isFib}_\Sigma\ \alpha_1\ \alpha_2\ e\ p\ \varphi\ f\ (a_1, a_2) \triangleq (\alpha_1\ e\ p\ \varphi\ f_1\ a_1\ ,\ \alpha_2\ e\ q\ \varphi\ f_2\ a_2)
\end{equation}
其中
\begin{align*}
&f_1 : [\varphi] \to \Pi_{\mathtt{I}}(A_1 \circ p) \\
&f_1\ u\ i \triangleq \mathtt{fst}(f\ u\ i) \\
&q : \mathtt{I} \to (x : \Gamma) \times A_1\ x \\
&q \triangleq \langle p\ ,\ \mathtt{fill}\ e\ \alpha_1\ p\ \varphi\ f_1\ a_1\rangle \\
&f_2 : [\varphi] \to \Pi_{\mathtt{I}}(A_2 \circ q) \\
&f_2\ u\ i \triangleq \mathtt{snd}(f\ u\ i)
\end{align*}

因此 $\mathtt{isFib}_\Sigma\ \alpha_1\ \alpha_2\ e\ p\ \varphi\ f\ (a_1, a_2) : (\Sigma\,A_1\,A_2)(p\ \overline{e})$；并且由于
\begin{align*}
&\forall(u : [\varphi]).\ f_1\ u\ \overline{e} = \alpha_1\ e\ p\ \varphi\ f_1\ a_1 = \mathtt{fill}\ e\ \alpha_1\ p\ \varphi\ f_1\ a_1\ \overline{e} \\
&\forall(u : [\varphi]).\ f_2\ u\ \overline{e} = \alpha_2\ e\ q\ \varphi\ f_2\ a_2
\end{align*}
成立，可得
\[
(\varphi, f)\ @\ \overline{e} \nearrow \mathtt{isFib}_\Sigma\ \alpha_1\ \alpha_2\ e\ p\ \varphi\ f\ (a_1, a_2).
\]
因此 $\mathtt{isFib}_\Sigma\ \alpha_1\ \alpha_2 : \mathtt{isFib}(\Sigma\,A_1\,A_2)$。最后，性质 (5.23) 由 (5.18) 和 (5.24) 得出。
\end{proof}

\begin{theorem}[纤维化 $\Pi$-类型]
存在一个函数
\begin{align}
\mathtt{isFib}_\Pi : &\{\Gamma : \mathcal{U}\}\{A_1 : \Gamma \to \mathcal{U}\}\{A_2 : (x : \Gamma) \times A_1\ x \to \mathcal{U}\} \to \\
&\mathtt{isFib}\ A_1 \to \mathtt{isFib}\ A_2 \to \mathtt{isFib}(\Pi\,A_1\,A_2)\notag
\end{align}
其中 $\Pi\,A_1\,A_2\ x \triangleq (a_1 : A_1\ x) \to A_2(x, a_1)$。该函数在重索引下是稳定的（参见 (5.23)），因此由 CCHM 纤维化给出的带族范畴支持 $\Pi$-类型的解释 [30, 定义 3.15]。
\end{theorem}

\begin{proof}
给定 $\Gamma : \mathcal{U}$，$A_1 : \Gamma \to \mathcal{U}$，$A_2 : (x : \Gamma) \times A_1\ x \to \mathcal{U}$，$\alpha_1 : \mathtt{isFib}\ A_1$，$\alpha_2 : \mathtt{isFib}\ A_2$，$e : \{0,1\}$，$p : \mathtt{I} \to \Gamma$，$\varphi : \mathtt{Cof}$，$f : [\varphi] \to \Pi_{\mathtt{I}}((\Pi\,A_1\,A_2) \circ p)$，$g : (\Pi\,A_1\,A_2)(p\ e)$ 且满足 $(\varphi, f)\ @\ e \nearrow g$，以及 $a_1 : A_1(p\ \overline{e})$，使用引理 5.3.7 我们定义
\begin{equation}
\mathtt{isFib}_\Pi\ \alpha_1\ \alpha_2\ e\ p\ \varphi\ f\ g\ a_1 \triangleq \alpha_2\ e\ q\ \varphi\ f_2\ a_2
\end{equation}
其中
\begin{align*}
&f_1 : \Pi_{\mathtt{I}}(A_1 \circ p) \\
&f_1 \triangleq \mathtt{fill}\ \overline{e}\ \alpha_1\ p\ \bot\ \mathtt{elim}_{\emptyset}\ a_1 \\
&q : \mathtt{I} \to (x : \Gamma) \times A_1\ x \\
&q \triangleq \langle p\ ,\ f_1\rangle \\
&f_2 : [\varphi] \to \Pi_{\mathtt{I}}(A_2 \circ q) \\
&f_2\ u\ i \triangleq f\ u\ i\ (f_1\ i) \\
&a_2 : \{a_2' : A_2(q\ e) \mid (\varphi, f_2)\ @\ e \nearrow a_2'\} \\
&a_2 \triangleq g(f_1\ e)
\end{align*}

由于我们知道 $f_1\ \overline{e} = \mathtt{fill}\ \overline{e}\ \alpha_1\ p\ \bot\ \mathtt{elim}_{\emptyset}\ a_1\ \overline{e} = a_1$，因此我们有
\begin{equation}
\mathtt{isFib}_\Pi\ \alpha_1\ \alpha_2\ e\ p\ \varphi\ f\ g\ a_1 : A_2(q\ \overline{e}) = A_2(p\ \overline{e}, f_1\ \overline{e}) = A_2(p\ \overline{e}, a_1)
\end{equation}
此外，由于 $(\varphi, f_2)\ @\ \overline{e} \nearrow \alpha_2\ e\ q\ \varphi\ f_2\ a_2$，对于任意 $u : [\varphi]$ 我们有
\begin{equation}
f\ u\ \overline{e}\ a_1 = f\ u\ \overline{e}\ (f_1\ \overline{e}) = f_2\ u\ \overline{e} = \alpha_2\ e\ q\ \varphi\ f_2\ a_2 = \mathtt{isFib}_\Pi\ \alpha_1\ \alpha_2\ e\ p\ \varphi\ f\ g\ a_1
\end{equation}
由于 (5.27) 和 (5.28) 对所有 $a_1 : A_1(p\ \overline{e})$ 成立，由前者可得
\[
\mathtt{isFib}_\Pi\ \alpha_1\ \alpha_2\ e\ p\ \varphi\ f\ g : (\Pi\,A_1\,A_2)(p\ \overline{e})
\]
由后者可得 $(\varphi, f)\ @\ \overline{e} \nearrow \mathtt{isFib}_\Pi\ \alpha_1\ \alpha_2\ e\ p\ \varphi\ f\ g$。因此我们有 (5.26) 确实给出了 $\mathtt{isFib}(\Pi\,A_1\,A_2)$ 的一个元素。最后，$\mathtt{isFib}_\Pi\ \alpha_1\ \alpha_2$ 在重索引下的稳定性由 (5.18) 得出。
\end{proof}

这些定理允许我们为 $\Sigma$-类型和 $\Pi$-类型构造纤维化结构，前提是它们的组成类型具有纤维化结构。但是，一开始是否存在任何纤维化结构呢？我们通过证明自然数对象 $\mathtt{N}$ 在拓扑斯中总是纤维化的来回答这个问题。这在立方体集合的拓扑斯 $\hatcube$ 中通过原始递归定义复合结构来证明 [11, 第 4.5 节]。我们给出一个更初等的证明，利用 $\hatcube$ 中的区间对象满足公理 $\ax{1}$ 这一事实（见定理 5.6.1）。

\begin{theorem}[$\mathtt{N}$ 是纤维化的]
如果 $\mathtt{N}$ 是一个具有可判定相等的对象，那么存在一个函数 $\mathtt{isFib}_\mathtt{N} : \{\Gamma : \mathcal{U}\} \to \mathtt{isFib}(\lambda(\ul  : \Gamma) \to \mathtt{N})$。特别地，如果拓扑斯 $\mathcal{E}$ 有一个自然数对象 $1 \xrightarrow{\mathtt{z}} \mathtt{N} \xrightarrow{\mathtt{s}} \mathtt{N}$，那么由 CCHM 纤维化给出的带族范畴具有一个自然数对象。
\end{theorem}

\begin{proof}
假设 $\Gamma : \mathcal{U}$，$e : \{0,1\}$，$p : \mathtt{I} \to \Gamma$，$\varphi : \mathtt{Cof}$，$f : [\varphi] \to \Pi_{\mathtt{I}}(\lambda\ul  \to \mathtt{N})$ 以及 $n : \mathtt{N}$ 且满足 $(\varphi, f)@e \nearrow n$。根据对 $\mathtt{N}$ 的假设，对于每个 $u : [\varphi]$，性质 $\lambda(i : \mathtt{I}) \to (f\ u\ i = n) : \mathtt{I} \to \Omega$ 是可判定的；因此根据公理 $\ax{1}$ 以及 $f\ u\ e = n$ 的事实，我们也有 $f\ u\ \bar{e} = n$。因此我们可以得到 $\mathtt{isFib}_\mathtt{N}\ e\ p\ \varphi\ f\ n : \{n' : \mathtt{N} \mid (\varphi, f)\ @\ \bar{e} \nearrow n'\}$，只需定义：$\mathtt{isFib}_\mathtt{N}\ e\ p\ \varphi\ f\ n \triangleq n$。对于定理的最后部分，我们使用在具有自然数对象的拓扑斯中，数的等式是可判定的这一事实。
\end{proof}

类似地使用公理 $\ax{1}$ 足以证明：

\begin{theorem}[纤维化余积]
记 $A_1 \xrightarrow{\mathtt{inl}} A_1 + A_2 \xleftarrow{\mathtt{inr}} A_2$ 为 $\mathcal{E}$ 中 $A_1$ 和 $A_2$ 的余积，我们将其提升到类型族，$\ul  \uplus \ul  : \{\Gamma : \mathcal{U}\}(A_1\ A_2 : \Gamma \to \mathcal{U}) \to \Gamma \to \mathcal{U}$，通过定义 $(A_1 \uplus A_2)\ x \triangleq A_1\ x + A_2\ x$。那么存在一个函数
\begin{equation}
\mathtt{isFib}_\uplus : \{\Gamma : \mathcal{U}\}\{A_1\ A_2 : \Gamma \to \mathcal{U}\} \to \mathtt{isFib}\ A_1 \to \mathtt{isFib}\ A_2 \to \mathtt{isFib}(A_1 \uplus A_2)
\end{equation}
并且余积上的这个纤维化结构在重索引下是稳定的。因此由 CCHM 纤维化给出的带族范畴具有二元余积。
\end{theorem}

\begin{proof}
该证明使用了唯一选择原理，它在拓扑的内部类型论中成立：
\begin{equation}
\mathtt{uc} : (A : \mathcal{U})(\varphi : A \to \Omega) \to [\exists!(a : A).\ \varphi\ a] \to \{a : A \mid \varphi\ a\}
\end{equation}
其中 $\exists!(a : A).\ \varphi\ a \triangleq \exists(a : A).\ \varphi\ a \land \forall(a' : A).\ \varphi\ a' \Rightarrow a = a'$。

假设我们有 $\Gamma : \mathcal{U}$，$A_1\ A_2 : \Gamma \to \mathcal{U}$，$\alpha_1 : \mathtt{isFib}\ A_1$，$\alpha_2 : \mathtt{isFib}\ A_2$，$e : \{0,1\}$，$p : \mathtt{I} \to \Gamma$，$\varphi : \mathtt{Cof}$，$g : [\varphi] \to \Pi_{\mathtt{I}}((A_1 \uplus A_2) \circ p)$ 以及 $c : A_1(p\ e) + A_2(p\ e)$ 且满足 $(\varphi, g)\ @\ e \nearrow c$。注意对于所有 $u : [\varphi]$ 和 $i : \mathtt{I}$
\begin{align*}
&P_1, P_2 : [\varphi] \to \mathtt{I} \to \Omega \\
&P_1\ u\ i \triangleq \exists!(a_1 : A_1(p\ i)).\ g\ u\ i = \mathtt{inl}\ a_1 \\
&P_2\ u\ i \triangleq \exists!(a_2 : A_2(p\ i)).\ g\ u\ i = \mathtt{inr}\ a_2
\end{align*}
是互补命题（$P_1\ u\ i \land P_2\ u\ i = \bot$ 且 $P_1\ u\ i \lor P_2\ u\ i = \top$）；因此根据 $\ax{1}$ 我们有 $(\forall(i : \mathtt{I}).\ P_1\ u\ i) \lor (\forall(i : \mathtt{I}).\ P_2\ u\ i)$。要么 $c = \mathtt{inl}\ a_1$ 对于某个 $a_1 : A_1(p\ e)$，要么 $c = \mathtt{inr}\ a_2$ 对于某个 $a_2 : A_2(p\ e)$。在第一种情况下，由于 $\forall(u : [\varphi]).\ g\ u\ e = c$，可得 $\forall(u : [\varphi])(i : \mathtt{I}).\ P_1\ u\ i$；然后使用 $\mathtt{uc}$ 我们得到某个 $f_1 : [\varphi] \to \Pi_{\mathtt{I}}(A_1 \circ p)$ 满足 $\forall(u : [\varphi])(i : \mathtt{I}).\ g\ u\ i = \mathtt{inl}(f_1\ u\ i)$ 并且我们可以定义
\[
\mathtt{isFib}_\uplus\ \alpha_1\ \alpha_2\ e\ p\ \varphi\ g\ (\mathtt{inl}\ a_1) \triangleq \mathtt{inl}(\alpha_1\ e\ p\ \varphi\ f_1\ a_1)
\]
类似地，如果 $c = \mathtt{inr}\ a_2$，那么存在某个 $f_2 : [\varphi] \to \Pi_{\mathtt{I}}(A_2 \circ p)$ 满足 $\forall(u : [\varphi])(i : \mathtt{I}).\ g\ u\ i = \mathtt{inr}(f_2\ u\ i)$ 并且我们可以定义
\[
\mathtt{isFib}_\uplus\ \alpha_1\ \alpha_2\ e\ p\ \varphi\ g\ (\mathtt{inr}\ a_2) \triangleq \mathtt{inr}(\alpha_2\ e\ p\ \varphi\ f_2\ a_2). \qedhere
\]
\end{proof}

\subsection{路径和恒等类型}

\begin{theorem}[纤维化路径类型]
存在一个函数
\begin{equation}
\mathtt{isFib}_{\mathtt{Path}} : \{\Gamma : \mathcal{U}\}\{A : \Gamma \to \mathcal{U}\} \to \mathtt{isFib}\ A \to \mathtt{isFib}(\mathtt{Path}\ A)
\end{equation}
其中 $\mathtt{Path}\ A : (x : \Gamma) \times (A\ x \times A\ x) \to \mathcal{U}$ 由下式给出
\begin{equation}
\mathtt{Path}\ A\ (x, (a_0, a_1)) \triangleq a_0 \sim a_1
\end{equation}
其中 $\sim$ 如 (5.5) 中所述。路径类型上的这个纤维化结构在重索引下是稳定的，在这个意义上，对于所有 $\gamma : \Delta \to \Gamma$
\begin{equation}
(\mathtt{isFib}_{\mathtt{Path}}\ \alpha)[\gamma \times ( id  \times  id )] = \mathtt{isFib}_{\mathtt{Path}}(\alpha[\gamma])
\end{equation}
\end{theorem}

\begin{proof}
给定 $\Gamma : \mathcal{U}$，$A : \Gamma \to \mathcal{U}$，$\alpha : \mathtt{isFib}\ A$，$e : \{0,1\}$，$p : \mathtt{I} \to (x : \Gamma) \times (A\ x \times A\ x)$，$\varphi : \mathtt{Cof}$，$f : [\varphi] \to \Pi_{\mathtt{I}}((\mathtt{Path}\ A) \circ p)$，$q : \mathtt{Path}\ A\ (p\ e)$ 且满足 $(\varphi, f)\ @\ e \nearrow q$ 以及 $i : \mathtt{I}$，假设 $p = \langle p', \langle q_0, q_1\rangle\rangle$ 其中 $p' : \mathtt{I} \to \Gamma$ 且 $q_0, q_1 : \Pi_{\mathtt{I}}(A \circ p)$，并定义
\begin{equation}
\mathtt{isFib}_{\mathtt{Path}}\ \alpha\ e\ p\ \varphi\ f\ q\ i \triangleq \alpha\ e\ p'\ (\varphi \lor i = 0 \lor i = 1)\ (f' \cup f_0 \cup f_1)\ (q\ i)
\end{equation}
其中
\begin{align*}
&f' : [\varphi] \to \Pi_{\mathtt{I}}(A \circ p') \\
&f'\ u\ j \triangleq f\ u\ j\ i \\
&f_0 : \{g : [i = 0] \to \Pi_{\mathtt{I}}(A \circ p') \mid (\varphi, f') \sim (i = 0, g)\} \\
&f_0\ \ul  \triangleq q_0 \\
&f_1 : \{g : [i = 1] \to \Pi_{\mathtt{I}}(A \circ p') \mid (\varphi \lor i = 0, f' \cup f_0) \sim (i = 1, g)\} \\
&f_1\ \ul  \triangleq q_1
\end{align*}

因此对于每个 $i : \mathtt{I}$ 我们有 $\mathtt{isFib}_{\mathtt{Path}}\ \alpha\ e\ p\ \varphi\ f\ q\ i : A(p'\bar{e})$，从而 $\mathtt{isFib}_{\mathtt{Path}}\ \alpha\ e\ p\ \varphi\ f\ q : \mathtt{I} \to A(p'\bar{e})$。由于 $\alpha\ e\ p' : \mathtt{Comp}\ e\ (A \circ p')$，我们有
\begin{align*}
\forall(u : [\varphi \lor i = 0 \lor i = 1]).\ (f' \cup f_0 \cup f_1)\ u\ \bar{e} = \\
\alpha\ e\ p'\ (\varphi \lor i = 0 \lor i = 1)\ (f' \cup f_0 \cup f_1)\ (q\ i) = \mathtt{isFib}_{\mathtt{Path}}\ \alpha\ e\ p\ \varphi\ f\ q\ i
\end{align*}
因此 $\mathtt{isFib}_{\mathtt{Path}}\ \alpha\ e\ p\ \varphi\ f\ q\ 0 = q_0\ \bar{e}$ 且 $\mathtt{isFib}_{\mathtt{Path}}\ \alpha\ e\ p\ \varphi\ f\ q\ 1 = q_1\ \bar{e}$，给出了从 $q_0\ \bar{e}$ 到 $q_1\ \bar{e}$ 的一条路径，从而
\[
\mathtt{isFib}_{\mathtt{Path}}\ \alpha\ e\ p\ \varphi\ f\ q : \mathtt{Path}\ A\ (p'\bar{e}, (q_0\ \bar{e}, q_1\ \bar{e})) = \mathtt{Path}\ A\ (p\ \bar{e})
\]
并且 furthermore，$(\varphi, f)\ @\ \bar{e} \nearrow \mathtt{isFib}_{\mathtt{Path}}\ \alpha\ e\ p\ \varphi\ f\ q$。因此 $\mathtt{isFib}_{\mathtt{Path}}\ \alpha : \mathtt{isFib}(\mathtt{Path}\ A)$。最后，性质 (5.33) 直接由 (5.34) 和 $\ul [\ul ]$ (5.15) 的定义得出。
\end{proof}

CCHM 纤维化的 CwF 中的这些路径类型（定义 5.3.5）满足 Coquand 提出的具有命题计算性质的恒等类型表述 [11, 图 2]。因此，除了单例类型的可收缩性 (5.7) 之外，我们还得到了沿路径传输纤维化元素的\textbf{代换函数}
\begin{align}
&\mathtt{subst} : \{\Gamma : \mathcal{U}\}\{A : \Gamma \to \mathcal{U}\}\{\alpha : \mathtt{isFib}\ A\}\{x_0\ x_1 : \Gamma\} \to (x_0 \sim x_1) \to A\ x_0 \to A\ x_1 \\
&\mathtt{subst}\ p\ a \triangleq \alpha\ 0\ p\ \bot\ \mathtt{elim}_{\emptyset}\ a \notag
\end{align}
使用了定义 5.1.2 之后提到的余纤维部分元素 $(\bot, \mathtt{elim}_{\emptyset})$。根据引理 5.3.7，这些代换函数满足常数路径 (5.6) 的命题计算规则：
\begin{align}
&\mathtt{H} : \{\Gamma : \mathcal{U}\}\{A : \Gamma \to \mathcal{U}\}\{\alpha : \mathtt{isFib}\ A\}\{x : \Gamma\}(a : A\ x) \to (a \sim \mathtt{subst}\ (\mathtt{k}\ x)\ a) \\
&\mathtt{H}\ a \triangleq \mathtt{fill}\ 0\ \alpha\ (\mathtt{k}\ x)\ \bot\ \mathtt{elim}_{\emptyset}\ a \notag
\end{align}

\begin{remark}[函数外延性]
如 [62, 引理 6.3.2] 中所预期，定理 5.3.15 的路径类型满足函数外延性。给定 $A : \mathcal{U}$，$B : A \to \mathcal{U}$，$f, g : (x : A) \to B\ x$ 以及 $p : (x : A) \to (f\ x \sim g\ x)$，我们得到一条路径 $\mathtt{funext}\ p : f \sim g$ 在 $(x : A) \to B\ x$ 中，由下式给出
\[
\mathtt{funext}\ p\ i \triangleq \lambda(x : A) \to p\ x\ i
\]
对于所有 $i : \mathtt{I}$。
\end{remark}

为了从这些路径类型得到具有标准定义性（而非命题性）计算性质的 Martin-Löf 恒等类型，我们使用 Swan 构造 [60] 的一个版本，类似于 [18] 第 9.1 节中的版本，但只使用 $\mathtt{I}$ 上的连接代数结构，而非 De Morgan 代数结构。这是唯一使用公理 $\ax{7}$ 的地方；我们需要由 $\mathtt{Cof}$ 和 $[\ul ] : \mathtt{Cof} \to \mathcal{U}$ 给出的宇宙在依赖积下封闭这一事实：

\begin{lemma}
以下类型 $\Omega$ 的元素是可证的：$\forall(\varphi : \Omega)(f : [\varphi] \to \Omega).\ \mathtt{cof}\ \varphi \Rightarrow (\forall(u : [\varphi]).\ \mathtt{cof}(f\ u)) \Rightarrow \mathtt{cof}(\exists(u : [\varphi]).\ f\ u)$。
\end{lemma}

\begin{proof}
注意如果 $u : [\varphi]$ 那么 $(\exists(v : [\varphi]).\ f\ v) = f\ u$，从而 $\mathtt{cof}(\exists(v : [\varphi]).\ f\ v) = \mathtt{cof}(f\ u)$。因此 $\forall(u : [\varphi]).\ \mathtt{cof}(f\ u)$ 等于 $\varphi \Rightarrow \mathtt{cof}(\exists(v : [\varphi]).\ f\ v)$。因此由 $\mathtt{cof}\ \varphi$ 和 $\forall(u : [\varphi]).\ \mathtt{cof}(f\ u)$ 根据公理 $\ax{7}$ 我们得到 $\mathtt{cof}(\varphi \land \exists(v : [\varphi]).\ f\ v)$，从而 $\mathtt{cof}(\exists(v : [\varphi]).\ f\ v)$，因为 $(\exists(v : [\varphi]).\ f\ v) \Rightarrow \varphi$。
\end{proof}

\begin{theorem}[纤维化恒等类型]
如下定义恒等类型：
\begin{align}
&\mathtt{Id} : \{\Gamma : \mathcal{U}\}(A : \Gamma \to \mathcal{U}) \to (x : \Gamma) \times (A\ x \times A\ x) \to \mathcal{U} \\
&\mathtt{Id}\ A\ (x, (a_0, a_1)) \triangleq (p : \mathtt{Path}\ A\ (x, (a_0, a_1))) \times \{\varphi : \mathtt{Cof} \mid \varphi \Rightarrow \forall(i : \mathtt{I}).\ p\ i = a_0\} \notag
\end{align}
那么存在一个函数 $\mathtt{isFib}_{\mathtt{Id}} : \{\Gamma : \mathcal{U}\}\{A : \Gamma \to \mathcal{U}\} \to \mathtt{isFib}\ A \to \mathtt{isFib}(\mathtt{Id}\ A)$，并且纤维化 $(\mathtt{Id}\ A, \mathtt{isFib}_{\mathtt{Id}}\ A)$ 可以被赋予 CCHM 纤维化的 CwF 中 Martin-Löf 恒等类型的结构 [30, 定义 3.19]。
\end{theorem}

\begin{proof}
给定 $\Gamma : \mathcal{U}$，$A : \Gamma \to \mathcal{U}$ 和 $\alpha : \mathtt{isFib}\ A$，使用定理 5.3.11 和 5.3.15 我们定义 $\mathtt{isFib}_{\mathtt{Id}}\ \alpha \triangleq \mathtt{isFib}_{\Sigma}(\mathtt{isFib}_{\mathtt{Path}}\ \alpha)\ \beta$，其中 $\beta : \mathtt{isFib}\ \Phi$ 且
\begin{align*}
&\Phi : (y : (x : \Gamma) \times (A\ x \times A\ x)) \times \mathtt{Path}\ A\ y \to \mathcal{U} \\
&\Phi((x, (a_0, a_1)), p) \triangleq \{\varphi : \mathtt{Cof} \mid \varphi \Rightarrow \forall(i : \mathtt{I}).\ p\ i = a_0\}
\end{align*}
并且纤维化结构 $\beta$ 将 $e : \{0,1\}$，$p : \mathtt{I} \to (y : (x : \Gamma) \times (A\ x \times A\ x)) \times \mathtt{Path}\ A\ y$，$\varphi : \mathtt{Cof}$，$f : [\varphi] \to \Pi_{\mathtt{I}}(\Phi \circ p)$ 和 $\varphi' : \Phi(p\ e)$ 且满足 $(\varphi, f)\ @\ e \nearrow \varphi'$ 映射到元素
\[
\beta\ e\ p\ \varphi\ f\ \varphi' \triangleq \exists(u : [\varphi]).\ f\ u\ \bar{e}
\]
（使用引理 5.3.17 来验证这是良定义的）。我们如下得到这些恒等类型的通常引入、消去和计算规则。由于 $\top : \mathtt{Cof}$ 根据公理 $\ax{5}$ 成立，恒等引入
\begin{equation}
\mathtt{refl} : \{\Gamma : \mathcal{U}\}\{A : \Gamma \to \mathcal{U}\}\{x : \Gamma\}(a : A\ x) \to \mathtt{Id}\ A\ (x, (a, a))
\end{equation}
可以定义为 $\mathtt{refl}\ a \triangleq (\lambda a\ i \to a, \top)$。恒等消去
\begin{align}
&\mathtt{J} : \{\Gamma : \mathcal{U}\}(A : \Gamma \to \mathcal{U})(x : \Gamma)(a_0 : A\ x)(B : (a : A\ x) \times \mathtt{Id}\ A\ (x, (a_0, a)) \to \mathcal{U}) \\
&\quad(\beta : \mathtt{isFib}\ B)(a_1 : A\ x)(e : \mathtt{Id}\ A\ (x, (a_0, a_1))) \to B(a_0, \mathtt{refl}\ a_0) \to B(a_1, e) \notag
\end{align}
由下式给出
\[
\mathtt{J}\ A\ x\ a_0\ B\ \beta\ a_1(p, \varphi)\ b \triangleq \beta\ 0\ \langle p, q\rangle\ \varphi\ f\ b
\]
其中 $q : (i : \mathtt{I}) \to \mathtt{Id}\ A\ x\ (a_0, p\ i)$ 是 $q\ i\ j \triangleq (p(i \sqcap j), \varphi \lor i = 0)$ 且 $f : [\varphi] \to \Pi_{\mathtt{I}}(B \circ \langle p, q\rangle)$ 是 $f\ u\ i \triangleq b$。（在上述元素中，由于 $(p, \varphi) : \mathtt{Id}\ A\ (x, (a_0, a_1))$ 我们有 $p\ 0 = a_0$，$p\ 1 = a_1$ 且 $\varphi \Rightarrow \forall(i : \mathtt{I}).\ p\ i = a_0$；因此 $\varphi \Rightarrow \forall(i : \mathtt{I}).\ q\ i = \mathtt{refl}\ a_0$，所以 $f$ 是良定义的。）注意根据公理 $\ax{3}$ 和 $\ax{4}$ 我们有 $q\ 0 = \mathtt{refl}\ a_0$ 且 $q\ 1 = (p, \varphi)$，从而 $\mathtt{J}\ A\ x\ a_0\ B\ \beta\ a_1(p, \varphi)\ b = \beta\ 0\ \langle p, q\rangle\ \varphi\ f\ b : B(p\ 1, q\ 1) = B(a_1, (p, \varphi))$，如所要求。

此外，由于 $(\varphi, f)\ @\ 1 \nearrow \beta\ 0\ \langle p, q\rangle\ \varphi\ f\ b$，我们有
\[
\forall(u : [\varphi]).\ b = f\ u\ 1 = \mathtt{J}\ A\ x\ a_0\ B\ \beta\ a_1(p, \varphi)\ b
\]
因此当 $(p, \varphi) = \mathtt{refl}\ a_0 = (\mathtt{k}\ a_0, \top)$ 从而 $a_1 = a_0$ 时，我们有
\begin{equation}
\mathtt{J}\ A\ x\ a_0\ B\ \beta\ a_0(\mathtt{refl}\ a_0)\ b = b
\end{equation}
换言之，恒等消去的计算性质作为判断等式成立，而不仅仅是命题等式。最后，为了正确支持内涵恒等类型的解释，需要 $(\mathtt{Id}\ A, \mathtt{isFib}_{\mathtt{Id}}\ A)$、$\mathtt{refl}$ 和 $\mathtt{J}\ A$ 在重索引下的稳定性；但这由 $\mathtt{isFib}_{\Sigma}$ 和 $\mathtt{isFib}_{\mathtt{Path}}$ 的稳定性性质得出。
\end{proof}

\section{粘合}

在本节中，我们给出 Cohen 等人 [18] 提出的\textbf{粘合}构造的内部表述。粘合类似于类型族的复合结构（定义 5.3.3），只不过它涉及的不是类型的部分路径，而是类型之间的部分等价。粘合对于 5.5 节中给出的与泛等性 [62] 相关的构造至关重要。

我们首先为余纤维-部分类型定义粘合构造，即对于函数 $A : [\varphi] \to \mathcal{U}$ 其中 $\varphi : \mathtt{Cof}$：

\begin{definition}[粘合]
给定 $\varphi : \mathtt{Cof}$，$A : [\varphi] \to \mathcal{U}$，$B : \mathcal{U}$ 和 $f : (u : [\varphi]) \to A\ u \to B$，类型 $\mathtt{Glue}\ \varphi\ A\ B\ f : \mathcal{U}$ 定义为
\begin{equation}
\mathtt{Glue}\ \varphi\ A\ B\ f \triangleq (a : (u : [\varphi]) \to A\ u) \times \{b : B \mid \forall(u : [\varphi]).\ f\ u\ (a\ u) = b\}
\end{equation}
\end{definition}

该类型的元素由对 $(a, b)$ 组成，其中 $a$ 是部分类型 $A$ 的部分元素，$b$ 是类型 $B$ 的元素，使得将 $f$ 应用于 $a$ 得到 $B$ 的一个部分元素，该元素可以扩展到 $b$。当 $\varphi = \top$ 时，$A$ 和 $f$ 都是全的，因此 $\mathtt{Glue}\ \varphi\ A\ B\ f$ 本质上由对 $(a, f\ a)$ 组成，对于每个 $a : A$，因此显然同构于 $A$。当 $\varphi = \bot$ 时，$A$ 和 $f$ 都是唯一确定的，$\mathtt{Glue}\ \varphi\ A\ B\ f$ 将由对 $(\mathtt{elim}_{\emptyset}, b)$ 组成，对于每个 $b : B$，因此显然同构于 $B$。

我们现在将这个粘合操作从余纤维-部分类型扩展到余纤维-部分类型族：

\begin{definition}[族的粘合]
我们将粘合操作从类型提升到类型族如下。给定 $\Gamma : \mathcal{U}$，$\Phi : \Gamma \to \mathtt{Cof}$，$A : \Gamma|\Phi \to \mathcal{U}$，$B : \Gamma \to \mathcal{U}$ 和
\[
f : (x : \Gamma)(v : [\Phi\ x]) \to A(x, v) \to B\ x,
\]
定义族 $\mathtt{Glue}\ \Phi\,A\,B\,f : \Gamma \to \mathcal{U}$ 为
\begin{equation}
\mathtt{Glue}\ \Phi\,A\,B\,f\ x \triangleq \mathtt{Glue}\ (\Phi\ x)\ (A(x, \ul )))\ (B\ x)\ (f(x, \ul ))
\end{equation}
\end{definition}

粘合构造适用于任何映射 $f : (x : \Gamma)(v : [\Phi\ x]) \to A(x, v) \to B\ x$。然而，我们希望看到这个构造可以提升到 CCHM 纤维化的 CwF。这意味着每当 $A$ 和 $B$ 有纤维化结构时，$\mathtt{Glue}\ \Phi\,A\,B\,f$ 也应该有纤维化结构，这对 $f$ 提出了一些要求。我们首先引入\textbf{扩展结构}的概念：

\begin{definition}[扩展结构]
扩展结构的类型，$\mathtt{Ext} : \mathcal{U} \to \mathcal{U}$，由下式给出
\[
\mathtt{Ext}\ A \triangleq (\tilde{a} : \square A) \to \{a : A \mid \tilde{a} \nearrow a\}
\]
\end{definition}

对于类型 $A : \mathcal{U}$ 具有扩展结构允许我们将 $A$ 的任何部分元素扩展为全元素。我们说一个族 $A : \Gamma \to \mathcal{U}$ 具有扩展结构，如果它的每个纤维都有，我们滥用记号写作
\[
\mathtt{Ext}\ A \triangleq (x : \Gamma) \to \mathtt{Ext}(A\ x)
\]

对于 $A : \Gamma \to \mathcal{U}$ 的扩展结构类似于 $A$ 的复合结构，在这个意义上两者都允许我们扩展部分元素；事实上，每个具有扩展结构的族都是一个纤维化。然而，扩展结构不需要一个全元素作为我们扩展/复合的起点，因此实际上是一个比复合结构更强的概念。首先注意，对于 $A : \mathcal{U}$ 具有扩展结构意味着 $A$ 是非空的，因为我们总是可以扩展空的部分元素。此外，给定任何元素 $a : A$，我们可以使用扩展结构来证明它与空部分元素的扩展是路径相等的。这些事实一起告诉我们 $A$ 是\textbf{可收缩的}：

\begin{definition}[可收缩性，参见定义 2.1.1]
如果类型 $A$ 有一个收缩中心 $a_0 : A$ 并且每个元素 $a : A$ 都命题相等于 $a_0$，即存在一条路径 $a_0 \sim a$，则称类型 $A$ 是\textbf{可收缩的}。因此，如果 $\mathtt{Contr}\ A$ 是非空的，则类型是可收缩的，其中 $\mathtt{Contr} : \mathcal{U} \to \mathcal{U}$ 定义为
\[
\mathtt{Contr}\ A \triangleq (a_0 : A) \times ((a : A) \to a_0 \sim a)
\]
\end{definition}

与扩展结构一样，我们说一个族 $A : \Gamma \to \mathcal{U}$ 是可收缩的，如果它的每个纤维都是，并写作
\[
\mathtt{Contr}\ A \triangleq (x : \Gamma) \to \mathtt{Contr}(A\ x)
\]

如上所述，对于族 $A : \Gamma \to \mathcal{U}$ 具有扩展结构意味着 $A$ 既是纤维化的又是可收缩的。事实上，逆命题也成立（参见 [18, 引理 5]）：

\begin{lemma}
存在函数
\begin{align}
&\mathtt{fromExt} : \{\Gamma : \mathcal{U}\}\{A : \Gamma \to \mathcal{U}\} \to \mathtt{Ext}\ A \to \mathtt{isFib}\ A \times \mathtt{Contr}\ A \\
&\mathtt{toExt} : \{\Gamma : \mathcal{U}\}\{A : \Gamma \to \mathcal{U}\} \to \mathtt{isFib}\ A \to \mathtt{Contr}\ A \to \mathtt{Ext}\ A
\end{align}
\end{lemma}

\begin{proof}
给定 $\Gamma : \mathcal{U}$，$A : \Gamma \to \mathcal{U}$ 和 $\varepsilon : \mathtt{Ext}\ A$，我们定义 $\alpha : \mathtt{isFib}\ A$
\[
\alpha\ e\ p\ \varphi\ f\ a_0 \triangleq \varepsilon\ (p\ \bar{e})\ ((\varphi, f)\ @\ \bar{e})
\]
对于每个 $x : \Gamma$，我们使用完全未定义余纤维-部分元素 $(\bot, \mathtt{elim}_{\emptyset}) : \square A$ 来定义 $a_0 : A$
\[
a_0 \triangleq \varepsilon\ x\ (\bot, \mathtt{elim}_{\emptyset})
\]
对于每个 $a : A\ x$ 和 $i : \ttI$，我们有 $\lambda\ul  \to a : [i = 1] \to A\ x$；因此我们得到一条路径 $p_a : \ttI \to A\ x$
\[
p_a\ i \triangleq \varepsilon\ x\ (i = 1, \lambda\ul  \to a)
\]
根据 $\mathtt{Ext}\ A$ 的定义，我们有 $p_a\ 1 = \varepsilon\ x\ (\top, \lambda\ul  \to a) = a$，并且根据 $\ax{2}$ 我们有 $p_a\ 0 = \varepsilon\ x\ (\bot, \mathtt{elim}_{\emptyset}) = a_0$。因此 $p_a : a_0 \sim a$。因此 $A\ x$ 是可收缩的。这些一起表明，给定 $\varepsilon : \mathtt{Ext}\ A$，我们可以定义 $\mathtt{isFib}\ A$ 和 $\mathtt{Contr}\ A$ 类型的元素。因此存在一个函数 $\mathtt{fromExt} : \{\Gamma : \mathcal{U}\}\{A : \Gamma \to \mathcal{U}\} \to \mathtt{Ext}\ A \to \mathtt{isFib}\ A \times \mathtt{Contr}\ A$。

反之，给定 $\Gamma : \mathcal{U}$，$A : \Gamma \to \mathcal{U}$，$\alpha : \mathtt{isFib}\ A$，$\langle a_0, p\rangle : \mathtt{Contr}\ A$，注意对于任何 $x : \Gamma$，$\varphi : \mathtt{Cof}$ 和 $f : [\varphi] \to A\ x$ 我们有 $(p\ x) \circ f : [\varphi] \to (\ttI \to A\ x)$ 使得 $\forall(u : [\varphi]).\ ((p\ x) \circ f)\ u : a_0\ x \sim f\ u$；因此 $(\varphi, (p\ x) \circ f)\ @\ 0 \nearrow (a_0\ x)$ 从而定义
\[
\varepsilon\ x\ (\varphi, f) \triangleq \alpha\ 0\ (\lambda\ul  \to x)\ \varphi\ ((p\ x) \circ f)\ (a_0\ x)
\]
我们得到 $\varepsilon\ x\ (\varphi, f) : A\ x$。此外，由于 $(\varphi, (p\ x) \circ f)\ @\ 1 = (\varphi, f)$ 根据 $\alpha$ 的类型，我们得到 $(\varphi, f) \nearrow \varepsilon\ x\ (\varphi, f)$。因此 $\varepsilon\ x\ (\varphi, f) : \mathtt{Ext}(A\ x)$，如所要求。
\end{proof}

我们现在来到本节的主要结果：证明纤维化在粘合下封闭。证明这一点要求函数 $f$ 是一个\textbf{等价}：

\begin{definition}[等价 [62, 第 4.4 节]]
给定类型 $A, B : \mathcal{U}$，函数 $f : A \to B$ 是一个\textbf{等价}，如果类型 $\mathtt{Equiv}\ f$ 是非空的，其中
\[
\mathtt{Equiv}\ f \triangleq (b : B) \to \mathtt{Contr}((a : A) \times f\ a \sim b)
\]
\end{definition}

同样，这显然可以提升到族：给定 $\Gamma : \mathcal{U}$，$A, B : \Gamma \to \mathcal{U}$ 和 $f : (x : \Gamma) \to A\ x \to B\ x$，定义
\[
\mathtt{Equiv}\ f \triangleq (x : \Gamma) \to \mathtt{Equiv}(f\ x)
\]

\begin{theorem}[粘合的复合]
令 $\Phi, A, B$ 和 $f$ 如定义 5.4.2 中所述。那么如果 $A$ 和 $B$ 都有纤维化结构且 $f$ 具有等价的结构，则 $\mathtt{Glue}\ \Phi\,A\,B\,f$ 具有纤维化结构。换言之，存在一个函数
\begin{align}
\mathtt{isFib}_{\mathtt{Glue}} : &\{\Gamma : \mathcal{U}\}\{\Phi : \Gamma \to \mathtt{Cof}\}\{A : \Gamma|\Phi \to \mathcal{U}\}\{B : \Gamma \to \mathcal{U}\} \\
&(f : (x : \Gamma)(u : [\Phi\ x]) \to A(x, u) \to B\ x) \to\notag\\
&((x : \Gamma)(v : [\Phi\ x]) \to \mathtt{Equiv}(f\ x\ v)) \to\notag\\
&\mathtt{isFib}\ A \to \mathtt{isFib}\ B \to \mathtt{isFib}(\mathtt{Glue}\ \Phi\,A\,B\,f)\notag
\end{align}
\end{theorem}

\begin{proof}
给定 $\Gamma : \mathcal{U}$，$\Phi : \Gamma \to \mathtt{Cof}$，$A : \Gamma|\Phi \to \mathcal{U}$，$B : \Gamma \to \mathcal{U}$，$f : (x : \Gamma)(u : [\Phi\ x]) \to A(x, u) \to B\ x$，$eq : (x : \Gamma)(u : [\Phi\ x]) \to \mathtt{Equiv}(f\ x\ v)$，$\alpha : \mathtt{isFib}\ A$，$\beta : \mathtt{isFib}\ B$，我们希望定义一个 $\mathtt{isFib}(\mathtt{Glue}\ \Phi\,A\,B\,f)$ 类型的元素。因此，取
\begin{align*}
&e : \{0,1\},\ p : \ttI \to \Gamma,\ \psi : \mathtt{Cof},\ q : [\psi] \to \Pi_{\ttI}(\mathtt{Glue}\ \Phi\,A\,B\,f)\\
&(a_0, b_0) : \{(a_0, b_0) : (\mathtt{Glue}\ \Phi\,A\,B\,f)(p\ e) \mid (\psi, q)\ @\ e \nearrow (a_0, b_0)\}
\end{align*}
我们的目标是定义 $(a_1, b_1) : (\mathtt{Glue}\ \Phi\,A\,B\,f)\ x$ 使得 $(\psi, \tilde{a_1}) \nearrow a_1$ 且 $(\psi, \tilde{b_1}) \nearrow b_1$，其中 $x : \Gamma$，$\tilde{a_1} : [\psi] \to ((u : [\Phi\ x]) \to A(x, u))$ 和 $\tilde{b_1} : [\psi] \to B\ x$ 由下式定义
\begin{align*}
&x \triangleq p\ \bar{e}\\
&\tilde{a_1}\ v \triangleq \mathtt{fst}(q\ v\ \bar{e})\\
&\tilde{b_1}\ v \triangleq \mathtt{snd}(q\ v\ \bar{e})
\end{align*}
并且满足 $\forall(v : [\psi])(u : [\Phi\ x]).\ f\ x\ u\ (\tilde{a_1}\ v\ u) = \tilde{b_1}\ v$，由 $\mathtt{Glue}\ \Phi\,A\,B\,f$ 的定义。

我们首先通过在 $B$ 中沿 $p$ 复合来得到 $b_1' : B\ x$，这可以看作是 $b_1$ 的一个初步近似：
\[
b_1' \triangleq \beta\ e\ p\ \psi\ (\lambda(v : [\psi])(i : \ttI) \to \mathtt{snd}(q\ v\ i))\ b_0
\]
回忆 $a_1$ 将具有类型 $(u : [\Phi\ x]) \to A(x, u)$，因此我们假设 $u : [\Phi\ x]$ 以定义一个 $A(x, u)$ 类型的元素。注意我们不能简单地在 $A$ 中沿 $p$ 复合，因为我们不知道 $\Phi(p\ i)$ 对所有 $i : \ttI$ 成立。相反，我们将使用等价结构来定义 $a_1$。

令 $C \triangleq (a : A(x, u)) \times f\ x\ u\ a \sim b_1'$ 为 $f\ x\ u$ 在 $b_1'$ 处的纤维。使用定理 5.3.11 和 5.3.15 以及 $A$ 和 $B$ 都是纤维化（分别由 $\alpha$ 和 $\beta$ 见证）的事实，我们可以推导出 $C$ 是一个纤维化对象。结合 $eq\ x\ u\ b_1' : \mathtt{Contr}\ C$ 的事实，我们可以使用引理 5.4.5 来定义 $\varepsilon : \mathtt{Ext}\ C$。然后我们可以定义
\[
(\psi, (\lambda(v : [\psi]) \to \tilde{a_1}\ v\ u,\ \mathtt{refl} \circ \tilde{b_1})) : \square C
\]
这是良定义的，因为如上所述，我们有 $\forall(v : [\psi]).\ f\ x\ u\ (\tilde{a_1}\ v\ u) = \tilde{b_1}\ v$，并且由复合结构的类型，我们有 $(\psi, \tilde{b_1}) \nearrow b_1'$，因此给定 $v : [\psi]$ 我们有 $\mathtt{refl}(\tilde{b_1}\ v) : f\ x\ u\ (\tilde{a_1}\ v\ u) \sim b_1'$。现在我们可以定义
\[
\varepsilon\ (\psi, (\lambda(v : [\psi]) \to \tilde{a_1}\ v\ u,\ \mathtt{refl} \circ \tilde{b_1})) : C
\]
解除假设 $u : [\Phi\ x]$ 并取上述定义的对的第一和第二投影，我们得到：
\[
a_1 : (u : [\Phi\ x]) \to A(x, u) \qquad p_b : (u : [\Phi\ x]) \to f\ x\ u\ (a_1\ u) \sim b_1'
\]

我们现在有 $a_1$ 和 $b_1'$ 使得 $\tilde{a_1} \nearrow a_1$ 和 $\tilde{b_1} \nearrow b_1'$。然而，我们不能简单地取 $b_1$ 为 $b_1'$，因为我们不知道 $\forall(u : \Phi\ x).\ f\ x\ u\ (a_1\ u) = b_1'$，因此不能得出 $(a_1, b_1') : (\mathtt{Glue}\ \Phi\,A\,B\,f)\ x$。为了解决这个问题，我们在 $B\ x$ 中执行最后一次复合以``修正''$b_1'$ 以实现这一性质。考虑以下连接
\[
p_b \cup (\mathtt{refl} \circ \tilde{b_1}) : [\Phi\ x \lor \psi] \to \Pi_{\ttI}(B\ x)
\]
这是良定义的，因为 $p_b$ 是通过扩展 $\mathtt{refl} \circ \tilde{b_1}$ 定义的，因此它们在共同定义的地方必须相等。我们用这个在 $B\ x$ 中执行最后一次复合：
\[
b_1 \triangleq \beta\ 1\ (\lambda(\ul  : \ttI) \to x)\ (\Phi\ x \lor \psi)\ (p_b \cup (\mathtt{refl} \circ \tilde{b_1}))\ b_1'
\]
我们现在有 $(a_1, b_1) : (\mathtt{Glue}\ \Phi\,A\,B\,f)\ x$ 使得 $(\psi, \tilde{a_1}) \nearrow a_1$ 和 $(\psi, \tilde{b_1}) \nearrow b_1$，如所要求。
\end{proof}

我们现在有一种方法来解释 [18] 中的粘合操作，它满足一些必要的要求；参见 [18, 图 4]。然而，当前的构造未能满足沿包含 $\iota : \Gamma|\Phi \rightarrowtail \Gamma$ 重索引时 $\mathtt{Glue}\ \Phi\,A\,B\,f$ 应该等于 $A$ 的要求。事实上，这个等式应该在 CCHM 纤维化的 CwF 中成立。这意味着不仅 $A = (\mathtt{Glue}\ \Phi\,A\,B\,f) \circ \iota : \Gamma|\Phi \to \mathcal{U}$，而且重索引定理 5.4.7 中导出的纤维化结构应该得到与我们开始时相同的纤维化结构。精确地说，我们所要求的是：
\begin{equation}
(A, \alpha) = (\mathtt{Glue}\ \Phi\,A\,B\,f, \mathtt{isFib}_{\mathtt{Glue}}\ f\ eq\ \alpha\ \beta)[\iota]
\end{equation}

我们目前有的是族 $A$ 和 $(\mathtt{Glue}\ \Phi\,A\,B\,f) \circ \iota$ 在定义 5.1.6 的意义上是同构的。

我们分两步达到 (5.46)。首先我们使用公理 $\ax{9}$ 来\textbf{严格化}粘合构造，得到一个新的、严格形式的粘合，$\mathtt{SGlue}$，使得 $\mathtt{Glue}\ \Phi\,A\,B\,f \cong \mathtt{SGlue}\ \Phi\,A\,B\,f$ 但 $A = (\mathtt{SGlue}\ \Phi\,A\,B\,f) \circ \iota$。然后我们使用公理 $\ax{8}$ 来调整 $\mathtt{SGlue}$ 上的纤维化结构，使得沿 $\iota$ 重索引时它等于 $A$ 上的纤维化结构。这些步骤的顺序似乎并不重要；我们也可以先调整 $\mathtt{Glue}$ 上的纤维化结构，然后用这个新的纤维化结构严格化 $\mathtt{Glue}$。

\begin{remark}
除了使用拓扑斯的内部语言之外，我们获得具有良好性质的粘合操作的方法与 Cohen 等人 [18] 的方法不同，在那里粘合是直接以所有所需性质定义的。然而，可以看到我们最后两个步骤在原始工作中的位置。严格化可以在 [18, 定义 15] 中对 $\varphi\rho = 1_\mathbb{F}$ 的情况分析中看到；更多细节参见第 5.6.2 节。Cohen 等人不是为粘合定义一个初始复合结构然后修改它以获得所需的重索引性质，而是直接定义复合结构。从 [18, 第 6.2 节] 中移除所有 $\forall$ 算子的使用将得到我们的初始复合结构，然后我们使用 $\mathtt{Cof}$ 在 $\forall(i : \ttI)$（公理 $\ax{8}$）下的封闭性在单独的步骤中修改这个复合。我们更喜欢这种方法，因为它简化了粘合的核心复合结构，并更明确地展示了 $\ax{8}$ 在立方类型论模型构造中的作用。
\end{remark}

我们现在回忆图 5.4 中的公理 $\ax{9}$：
\begin{align*}
&\ax{9} : \{\varphi : \mathtt{Cof}\}(A : [\varphi] \to \mathcal{U})(B : \mathcal{U})(s : (u : [\varphi]) \to (A\ u \cong B)) \to\\
&\quad(B' : \mathcal{U}) \times \{s' : B' \cong B \mid \forall(u : [\varphi]).\ A\ u = B' \land s\ u = s'\}
\end{align*}

这表明任何部分类型 $A$，如果在其定义的每个地方都同构于一个全类型 $B$，则可以扩展为一个全类型 $B'$，它同构于 $B$。我们在第 5.6.2 节中研究为什么立方体预层拓扑斯 [18] 满足这个公理。给定 $\ax{9}$，定义一个严格形式的粘合是直接的。

\begin{definition}[严格粘合]
给定 $\Gamma : \mathcal{U}$，$\Phi : \Gamma \to \mathtt{Cof}$，$A : \Gamma|\Phi \to \mathcal{U}$，$B : \Gamma \to \mathcal{U}$ 和 $f : (x : \Gamma)(u : [\Phi\ x]) \to A(x, u) \to B\ x$，定义 $\mathtt{SGlue}\ \Phi\,A\,B\,f : \Gamma \to \mathcal{U}$ 为
\begin{align}
&\mathtt{SGlue}\ \Phi\,A\,B\,f\ x \triangleq\notag\\
&\quad\mathtt{fst}(\ax{9}\ (\lambda u : [\Phi\ x] \to A(x, u))\ (\mathtt{Glue}\ \Phi\,A\,B\,f\ x)\ (\lambda u : [\Phi\ x] \to \mathtt{glue}\ (x, u)))
\end{align}
其中 $\mathtt{glue}\ (x, u) : A(x, u) \cong \mathtt{Glue}\ \Phi\,A\,B\,f\ x$ 是定义 5.4.1 中提到的同构，由下式给出
\[
\mathtt{glue}\ (x, u)\ a \triangleq (\lambda\ul  \to a, f\ x\ u\ a)
\]

注意 $\mathtt{SGlue}$ 具有所需的严格性：给定任何 $(x, u) : \Gamma|\Phi$，由 $\ax{9}$ 我们有 $A(x, u) = \mathtt{fst}(\ax{9}\ (\lambda u : [\Phi\ x] \to A(x, u))\ (\mathtt{Glue}\ \Phi\,A\,B\,f\ x)\ (\lambda u : [\Phi\ x] \to \mathtt{glue}\ (x, u)))$，因此
\begin{equation}
\forall(x : \Gamma)(u : [\Phi\ x]).\ \mathtt{SGlue}\ \Phi\,A\,B\,f\ x = A(x, u)
\end{equation}
\end{definition}

\begin{theorem}[严格粘合的复合结构]
给定 $\Gamma, \Phi, A, B$ 和 $f$ 如定义 5.4.9 中所述，如果 $A$ 和 $B$ 有纤维化结构且 $f$ 具有等价的结构，则 $\mathtt{SGlue}\ \Phi\,A\,B\,f : \Gamma \to \mathcal{U}$ 具有纤维化结构。
\end{theorem}

\begin{proof}
由引理 5.3.8 我们有（逐纤维）同构保持纤维化结构。因此我们通过将 $\mathtt{isFib}_{\mathtt{Glue}}$（定理 5.4.7）得到的结构沿 $\ax{9}$ 的同构传输，得到 $\mathtt{SGlue}$ 上的纤维化结构。
\end{proof}

本节的最后一步是使用公理 $\ax{8}$ 来调整 $\mathtt{SGlue}$ 的复合结构，以便我们恢复 $A$ 上的原始复合结构。

\begin{corollary}
给定 $\Gamma : \mathcal{U}$，$\Phi : \Gamma \to \mathtt{Cof}$，$(A, \alpha) : \mathtt{Fib}(\Gamma|\Phi)$，$(B, \beta) : \mathtt{Fib}\ \Gamma$ 和 $f : (x : \Gamma)(v : [\Phi\ x]) \to A(x, v) \to B\ x$，存在 $(G, \gamma) : \mathtt{Fib}\ \Gamma$ 使得 $(A, \alpha) = (G, \gamma)[\iota]$。
\end{corollary}

\begin{proof}
简单地取 $G = \mathtt{SGlue}\ \Phi\,A\,B\,f$；由定理 5.4.10 我们得到 $G$ 的复合结构，然后我们使用引理 5.3.10 来调整它以得到一个新的复合结构 $\gamma$ 满足所需的等式。
\end{proof}

\section{泛等性}

Voevodsky 的\textbf{泛等性公理}（第 2.1.3 节）对于 CwF 中的宇宙 $\mathcal{V}$（至少具有 $\Sigma$-、$\Pi$- 和 $\mathtt{Id}$-类型）指出，对于每个 $A, B : \mathcal{V}$，从 $\mathtt{Id}\ \mathcal{V}\ A\ B$ 到 $(f : A \to B) \times \mathtt{Equiv}\ f$ 的典范函数是一个等价。Cohen 等人构造了立方体集合预层拓扑斯中的（与 CwF 相关联的）宇宙，其类型族对于具有小纤维的 CCHM 纤维化是通用的（对于元理论中``小''的适当概念），并证明它满足泛等性公理。他们通过调整 Hofmann-Streicher 预层范畴的宇宙构造 [31] 来实现这一点，细节参见第 3.3.5 节。仅使用一般拓扑斯的内部类型论来表达他们的宇宙构造是不可能的，原因我们将在定理 7.1.1 中讨论。在第 7 章中，我们解释了为什么宇宙构造不能简单地被公理化，然后展示了如何通过工作于内部类型论的模态扩展来解决这个问题。目前，我们只证明一个不涉及纤维化宇宙的泛等性版本。

为了理解这可能意味着什么，考虑以下情况：如果存在一个宇宙 $\mathcal{V}$，其元素是 CCHM 纤维化的编码，那么给定由函数 $a, b : \Gamma \to \mathcal{V}$ 命名的纤维化 $(A, \alpha), (B, \beta) : \mathtt{Fib}\ \Gamma$，$a$ 和 $b$ 之间的路径相等性给出（通过柯里化）一个函数 $p : \Gamma \times \ttI \to \mathcal{V}$ 使得对于所有 $x : \Gamma$ 有 $p(x, 0) = a\ x$ 和 $p(x, 1) = b\ x$。然后 $p$ 命名一个纤维化 $(P, \rho) : \mathtt{Fib}(\Gamma \times \ttI)$ 使得 $(P, \rho)[\langle  id , 0\rangle] = (A, \alpha)$ 和 $(P, \rho)[\langle  id , 1\rangle] = (B, \beta)$。后者给出了类型族之间路径相等性的概念，无论是否存在这样的 $\mathcal{V}$，我们在本节中研究与纤维化之间等价性的关系。

为了更正式地给出定义，我们扩展我们的持续假设，即环境拓扑斯 $\mathcal{E}$ 带有一个宇宙（内部全子拓扑斯）$\mathcal{U}$，到有第二个宇宙 $\mathcal{U}_1$ 且 $\mathcal{U} : \mathcal{U}_1$ 的情况。我们有时将 $\mathcal{U}$ 类型的对象称为\textbf{小类型}，将 $\mathcal{U}_1$ 类型的对象称为\textbf{大类型}。

\begin{definition}[纤维化之间的路径相等性]
定义 CCHM 纤维化之间的路径类型 $\ul \sim_{\mathcal{U}}\ul  : \{\Gamma : \mathcal{U}\} \to \mathtt{Fib}\ \Gamma \to \mathtt{Fib}\ \Gamma \to \mathcal{U}_1$ 为
\[
A \sim_{\mathcal{U}} B \triangleq \{P : \mathtt{Fib}(\Gamma \times \ttI) \mid P[\langle  id , \ttzero\rangle] = A \land P[\langle  id , \ttone\rangle] = B\}
\]
\end{definition}

先前，我们指出了为什么这样的路径概念应该对应于存在的 CCHM 纤维化宇宙。在这里，我们解释为什么这个概念在 [18] 给出的具体模型中确实是等价的。回忆那里的宇宙构造是由预层范畴的通常 Hofmann-Streicher 宇宙构造的变体给出的，如第 3.3.5 节所述。这意味着，在外部元理论中，对于所有 $\Gamma$ 存在一个类型 $\mathcal{V} \in \mathtt{Ty}(\Gamma)$ 由 $\mathcal{V}(I, \rho) \triangleq \mathtt{FTy}_0(\mathtt{y}I)$ 给出，其中 $\mathtt{y}I$ 表示 $I$ 的 Yoneda 嵌入。每个（小）纤维化类型 $A \in \mathtt{FTy}_0(\Gamma)$ 有一个编码 $\ulcorner A \urcorner \in \mathtt{Ter}(\Gamma \vdash \mathcal{V})$，每个 $a \in \mathtt{Ter}(\Gamma \vdash \mathcal{V})$ 编码一个类型 $\mathtt{El}\ a \in \mathtt{FTy}_0(\Gamma)$ 使得 $\mathtt{El}\ (\ulcorner A \urcorner) = A$ 且 $\ulcorner \mathtt{El}\ a \urcorner = a$ 对所有 $a$ 和 $A$ 成立。

现在考虑以下情况：外部地，路径 $P : A \sim_{\mathcal{U}} B$ 对应于纤维化 $P \in \mathtt{FTy}_0(\Gamma.\bbI)$ 使得对于某个 $\Gamma \in \hatsquare$ 和 $A, B \in \mathtt{FTy}_0(\Gamma)$ 有 $P[\langle  id , \ttzero\rangle] = A$ 和 $P[\langle  id , \ttone\rangle] = B$。从这些数据我们可以构造 $p \in \mathtt{Ter}(\Gamma \vdash \mathtt{Path}_{\mathcal{V}}\ \ulcorner A \urcorner\ \ulcorner B \urcorner)$ 如下：
\[
p(\rho) \triangleq \langle i \rangle\ \ulcorner P \urcorner(\rho s_i, i)
\]

对于 $I \in \square, \rho \in \Gamma(I)$，其中 $s_i : I, i \to I$ 是 $I$ 在 $I, i$ 中的明显包含。注意这确实定义了一条具有正确端点的路径，因为将 $0$ 代入 $i$ 我们得到：
\[
\ulcorner P \urcorner(\rho, 0) = \ulcorner P \urcorner[\langle  id , 0\rangle](\rho) = \ulcorner P[\langle  id , 0\rangle] \urcorner(\rho) = \ulcorner A \urcorner(\rho)
\]
因此 $(\ulcorner P \urcorner(\rho s_i, i))(i0) = \ulcorner A \urcorner(\rho)$。类似地，当 $i = 1$ 时我们有 $(\ulcorner P \urcorner(\rho s_i, i))(i1) = \ulcorner B \urcorner(\rho)$，因此 $\langle i \rangle\ \ulcorner P \urcorner(\rho s_i, i) \in \mathtt{Path}_{\mathcal{V}}\ \ulcorner A \urcorner\ \ulcorner B \urcorner(\rho)$ 如所要求。

反之，给定 $p \in \mathtt{Ter}(\Gamma \vdash \mathtt{Path}_{\mathcal{V}}\ \ulcorner A \urcorner\ \ulcorner B \urcorner)$ 我们可以定义 $P \in \mathtt{FTy}_0(\Gamma.\bbI)$ 具有所需性质如下：
\[
P \triangleq \mathtt{El}(p') \qquad \text{其中} \qquad p'(\rho, i) \triangleq p\ \rho\ i
\]
对于 $I \in \square, \rho \in \Gamma(I), i \in \ttI$。再次注意，这具有正确的性质，例如在 $0$ 处：
\[
P[\langle  id , 0\rangle] = \mathtt{El}(p')[\langle  id , 0\rangle] = \mathtt{El}(p'[\langle  id , 0\rangle]) = \mathtt{El}(\ulcorner A \urcorner) = A
\]

容易验证这两个构造是互逆的。因此由 $\ul \sim_{\mathcal{U}}\ul $ 描述的数据恰好对应于描述宇宙中路径所需的数据。

我们现在证明，给定这样的路径 $(P, \rho) : (A, \alpha) \sim_{\mathcal{U}} (B, \beta)$，总是可以构造一个等价 $f : (x : \Gamma) \to A\ x \to B\ x$。反之，给定纤维化 $(A, \alpha)$ 和 $(B, \beta)$ 之间的等价 $f : (x : \Gamma) \to A\ x \to B\ x$，总是可以构造这样的 $(P, \rho)$。

\begin{theorem}[路径转换为等价]
存在一个函数
\begin{align}
\mathtt{pathToEquiv} : &\{\Gamma : \mathcal{U}\}\{A\ B : \mathtt{Fib}\ \Gamma\} \\
&(P : A \sim_{\mathcal{U}} B) \to (f : (x : \Gamma) \to \mathtt{fst}\ A\ x \to \mathtt{fst}\ B\ x) \times \mathtt{Equiv}\ f \notag
\end{align}
\end{theorem}

\begin{proof}
给定 $\Gamma : \mathcal{U}$，$(A, \alpha), (B, \beta) : \mathtt{Fib}\ \Gamma$ 和 $(P, \rho) : A \sim_{\mathcal{U}} B$，我们定义映射 $f : (x : \Gamma) \to A\ x \to B\ x$ 和 $g : (x : \Gamma) \to B\ x \to A\ x$。首先，给定 $x : \Gamma$，记 $\langle x,  id \rangle : (x, 0) \sim (x, 1)$ 为由 $\langle x,  id \rangle\ i \triangleq (x, i)$ 给出的路径。现在定义 $f$ 和 $g$ 如下：
\[
f\ x\ a \triangleq \rho\ 0\ \langle x,  id \rangle\ \bot\ \mathtt{elim}_{\emptyset}\ a \qquad g\ x\ b \triangleq \rho\ 1\ \langle x,  id \rangle\ \bot\ \mathtt{elim}_{\emptyset}\ b
\]

这个定义是良类型的，因为 $P(x, 0) = A\ x$ 和 $P(x, 1) = B\ x$。由于两个函数都使用合成结构定义，我们可以使用填充（引理 5.3.7）来找到依赖类型的路径：
\begin{align*}
&p : (x : \Gamma)(a : A\ x) \to \Pi_{\ttI}P \text{ 定义为 } p\ x\ a \triangleq \mathtt{fill}\ 0\ \rho\ \langle x,  id \rangle\ \bot\ \mathtt{elim}_{\emptyset}\ a\\
&q : (x : \Gamma)(b : B\ x) \to \Pi_{\ttI}P \text{ 定义为 } q\ x\ b \triangleq \mathtt{fill}\ 1\ \rho\ \langle x,  id \rangle\ \bot\ \mathtt{elim}_{\emptyset}\ b
\end{align*}

注意对于所有 $x : \Gamma$ 和 $a : A\ x$ 我们有 $p\ x\ a\ 0 = a$ 和 $p\ x\ a\ 1 = f\ x\ a$。类似地，对于所有 $b : B\ x$ 我们有 $q\ x\ b\ 0 = g\ x\ b$ 和 $q\ x\ b\ 1 = b$。现在我们定义：
\begin{align*}
&r : (x : \Gamma)(a : A\ x) \to a \sim g\ x\ (f\ x\ a)\\
&r\ x\ a\ i \triangleq \rho\ 1\ \langle x,  id \rangle\ (i = 0 \lor i = 1)\ ((\lambda\ul  \to p\ x\ a) \cup (\lambda\ul  \to q\ x\ (f\ x\ a)))\ (f\ x\ a)\\
&s : (x : \Gamma)(b : B\ x) \to b \sim f\ x\ (g\ x\ b)\\
&s\ x\ b\ i \triangleq \rho\ 0\ \langle x,  id \rangle\ (i = 0 \lor i = 1)\ ((\lambda\ul  \to q\ x\ b) \cup (\lambda\ul  \to p\ x\ (g\ x\ b)))\ (g\ x\ b)
\end{align*}

因此 $f$ 和 $g$ 是准逆；由此我们可以构造一个等价结构 [62, 第 4 章]。
\end{proof}

我们现在希望证明，反之，可以将等价转换为纤维化之间的路径。为此我们使用第 5.4 节中给出的粘合构造。

\begin{theorem}[等价转换为路径]
存在一个函数
\begin{align}
\mathtt{equivToPath} : &\{\Gamma : \mathcal{U}\}\{A\ B : \mathtt{Fib}\ \Gamma\} \\
&(f : (x : \Gamma) \to \mathtt{fst}\ A\ x \to \mathtt{fst}\ B\ x) \to (\mathtt{Equiv}\ f) \to A \sim_{\mathcal{U}} B \notag
\end{align}
\end{theorem}

\begin{proof}
给定 $\Gamma : \mathcal{U}$，$(A, \alpha), (B, \beta) : \mathtt{Fib}\ \Gamma$，$f : (x : \Gamma) \to \mathtt{fst}\ A\ x \to \mathtt{fst}\ B\ x$ 和 $eq : \mathtt{Equiv}\ f$，定义如下：
\begin{align*}
&\Phi : \Gamma \times \ttI \to \mathtt{Cof}\\
&\Phi\ (x, i) \triangleq (i = 0) \lor (i = 1)\\
&C : (\Gamma \times \ttI)|\Phi \to \mathcal{U}\\
&C\ ((x, i), u) \triangleq ((\lambda\ul  : [i = 0] \to A\ x) \cup (\lambda\ul  : [i = 1] \to B\ x))\ u\\
&f' : ((x, i) : \Gamma \times \ttI)(u : [\Phi(x, i)]) \to C((x, i), u) \to B\ x\\
&f'\ (x, i) \triangleq (\lambda\ul  : [i = 0] \to f\ x) \cup (\lambda\ul  : [i = 1] \to  id )
\end{align*}

现在令 $P \triangleq \mathtt{SGlue}\ \Phi\ C\ (\lambda(x, \ul ) \to B\ x)\ f'$ 并观察到由 $\mathtt{SGlue}$ 的严格性性质，$P(x, 0) = A\ x$ 和 $P(x, 1) = B\ x$。

现在我们证明 $P$ 具有纤维化结构。首先，我们观察到 $C$ 具有纤维化结构，使用公理 $\ax{1}$。为了定义 $\gamma : \mathtt{isFib}\ C$ 我们取：
\[
e : \{0, 1\}, \quad p : \ttI \to (\Gamma \times \ttI)|\Phi, \quad \varphi : \mathtt{Cof}, \quad f : [\varphi] \to \Pi(C \circ p), \quad c : C(p\ e)
\]
并旨在定义 $\gamma\ e\ p\ \varphi\ f\ c : C(p\ \bar{e})$。首先，定义谓词 $\mathtt{pZero} : \ttI \to \Omega$ 为：
\[
\mathtt{pZero}\ i \triangleq (\mathtt{snd}(\mathtt{fst}(p\ i)) = 0)
\]
观察到由于 $p : \ttI \to (\Gamma \times \ttI)|\Phi$ 我们知道，由 $\mathtt{snd} \circ p$，有：
\begin{equation}
(\forall i : \ttI)\ \mathtt{snd}(\mathtt{fst}(p\ i)) = 0 \lor \mathtt{snd}(\mathtt{fst}(p\ i)) = 1
\end{equation}
因此，使用 $\ax{2}$，我们有 $(\forall i : \ttI)\ (\mathtt{pZero}\ i \lor \neg(\mathtt{pZero}\ i))$，因此使用 $\ax{1}$ 我们得到
\[
((\forall i : \ttI)\ \mathtt{pZero}\ i) \lor ((\forall i : \ttI)\ \neg(\mathtt{pZero}\ i))
\]

在情况 $(\forall i : \ttI)\ \mathtt{pZero}\ i$ 中，我们推导出 $C \circ p = A \circ \mathtt{fst} \circ \mathtt{fst} \circ p$ 并定义：
\[
\gamma\ e\ p\ \varphi\ f\ c \triangleq \alpha\ e\ (\mathtt{fst} \circ \mathtt{fst} \circ p)\ \varphi\ f\ c
\]
否则，在情况 $(\forall i : \ttI)\ \neg(\mathtt{pZero}\ i)$ 中，我们使用 $\ax{2}$ 推导出 $(\forall i : \ttI)\ \mathtt{snd}(\mathtt{fst}(p\ i)) = 1$，

因此 $C \circ p = B \circ \mathtt{fst} \circ \mathtt{fst} \circ p$，于是定义：
\[
\gamma\ e\ p\ \varphi\ f\ c \triangleq \beta\ e\ (\mathtt{fst} \circ \mathtt{fst} \circ p)\ \varphi\ f\ c
\]

因此 $C$ 具有纤维化结构。接下来我们证明对于每个 $x : \Gamma$，$i : \ttI$ 和 $u : [\Phi\ i]$，$f'$ 是一个等价。首先注意恒等函数 $ id  : B\ x \to B\ x$ 总是一个等价；令 $\mathtt{idEq} : \mathtt{Equiv}\  id $ 是这一事实的证明。定义 $eq' : (x : \Gamma)(i : \ttI)(u : [\Phi\ i]) \to \mathtt{Equiv}\ (f'\ x\ i\ u)$ 为：
\[
eq'\ x\ i \triangleq (\lambda u : [i = 0] \to eq) \cup (\lambda u : [i = 1] \to \mathtt{idEq})
\]

因此，由推论 5.4.11，我们得到一个纤维化结构 $\rho : \mathtt{isFib}\ P$，使得
\[
(P, \rho)[\iota : (\Gamma \times \ttI)|\Phi \rightarrowtail \Gamma \times \ttI] = (C, \gamma)
\]
因此 $(P, \rho)[\langle  id , 0\rangle] = (P, \rho)[\iota \circ \langle  id , 0, *\rangle] = (C, \gamma)[\langle  id , 0, *\rangle] = (A, \alpha)$，类似地 $(P, \rho)[\langle  id , 1\rangle] = (B, \beta)$。因此我们定义 $\mathtt{equivToPath}\ f\ e \triangleq (P, \rho)$。
\end{proof}

泛等性公理（定义 2.1.4）在这里成为映射 $\mathtt{pathToEquiv}$ 本身是一个等价的性质。定理 2.1.9 告诉我们，这实际上等同于拥有一个从等价到路径的映射，我们在 $\mathtt{equivToPath}$ 中有这样的映射，使得沿 $\mathtt{equivToPath}\ f\ e$ 的强制与 $f$ 路径相等。这个性质确实成立：

\begin{theorem}[$\sim_{\mathcal{U}}$ 的泛等性]
定义
\begin{align*}
&\mathtt{coerce} : \{\Gamma : \mathcal{U}\}\{A\ B : \mathtt{Fib}\ \Gamma\}(P : A \sim_{\mathcal{U}} B)(x : \Gamma) \to \mathtt{fst}\ A\ x \to \mathtt{fst}\ B\ x\\
&\mathtt{coerce}(P, \rho) \triangleq \mathtt{fst}(\mathtt{pathToEquiv}(P, \rho))
\end{align*}

给定 $\Gamma : \mathcal{U}$，$(A, \alpha), (B, \beta) : \mathtt{Fib}\ \Gamma$，$f : (x : \Gamma) \to A\ x \to B\ x$ 和 $eq : \mathtt{Equiv}\ f$，存在一条路径 $f \sim \mathtt{coerce}(\mathtt{equivToPath}\ f\ e)$。
\end{theorem}

\begin{proof}
令 $(P, \rho) \triangleq \mathtt{equivToPath}\ f\ e$。展开 $\mathtt{coerce}$ 的定义，我们有
\[
\mathtt{coerce}\ (P, \rho)\ x\ a = \rho\ 0\ \langle x,  id \rangle\ \bot\ \mathtt{elim}_{\emptyset}\ a
\]

回忆 $\rho$ 是 $\mathtt{SGlue}$ 的合成结构，我们可以计算 $\rho\ 0\ \langle x,  id \rangle\ \bot\ \mathtt{elim}_{\emptyset}\ a$。我们首先跨越与 $\mathtt{Glue}$ 的同构，使得 $a$ 变为 $(\lambda\ul  \to a, f\ x\ a)$。我们现在追踪 $\mathtt{Glue}$ 中的合成算法：我们开始在 $B\ x$ 中合成以得到 $b_1' \triangleq \beta\ 0\ (\lambda\ul  \to x)\ \bot\ \mathtt{elim}_{\emptyset}\ (f\ x\ a)$。接下来，我们使用等价结构来推导出 $a_1 : (u : [1 = 0 \lor 1 = 1]) \to B\ x$ 和 $p_b : (u : [1 = 0 \lor 1 = 1]) \to  id (a_1\ u) \sim b_1'$，其中特别地，$a_1$ 由 $a_1\ u \triangleq \beta\ 1\ (\lambda\ul  \to x)\ \bot\ \mathtt{elim}_{\emptyset}\ b_1'$ 给出。然后我们执行合成的最后一步，即从 $b_1'$ 在 $B\ x$ 中合成以得到 $b_1$。这给我们留下了在 $\mathtt{Glue}$ 中合成的结果 $(a_1, b_1)$。当跨越同构传回时，我们简单地取对的第一分量，即 $a_1$，但现在被视为一个全元素，以得到在 $\mathtt{SGlue}$ 中合成的最终结果 $\beta\ 1\ (\lambda\ul  \to x)\ \bot\ \mathtt{elim}_{\emptyset}\ b_1'$。因此，总结起来我们有：
\begin{align*}
\mathtt{coerce}\ (P, \rho)\ x\ a &= \rho\ 0\ \langle x,  id \rangle\ \bot\ \mathtt{elim}_{\emptyset}\ a\\
&= \beta\ 1\ (\lambda\ul  \to x)\ \bot\ \mathtt{elim}_{\emptyset}\ b_1'\\
&= \beta\ 1\ (\lambda\ul  \to x)\ \bot\ \mathtt{elim}_{\emptyset}\ (\beta\ 0\ (\lambda\ul  \to x)\ \bot\ \mathtt{elim}_{\emptyset}\ (f\ x\ a))
\end{align*}

由于这只是应用于 $f\ x\ a$ 的两个平凡合成，我们可以使用填充来推导出一条路径 $f\ x\ a \sim \mathtt{coerce}\ (P, \rho)\ x\ a$。现在，两次应用函数外延性（注记 5.3.16）得到一条路径 $f \sim \mathtt{coerce}\ (P, \rho)$，如所要求。
\end{proof}

\section{满足公理}

在构造性集合论中非形式化地工作，[18] 的作者使用特定小范畴 $\square$（他们称之为\textbf{立方体范畴}，如第 3.3 节所述）上的反变集值函子构成的拓扑斯 $\hatcube = \mathbf{Set}^{\square^{\mathrm{op}}}$ 给出了他们类型论的一个模型。在本节中，我们给出任意小范畴 $\mathbf{C}$ 的充分条件，使得集值预层拓扑斯 $\widehat{\mathbf{C}} = \mathbf{Set}^{\mathbf{C}^{\mathrm{op}}}$（在直觉主义 ZF 集合论 [1, 第 3.2 节] 中）具有满足图 5.4 中公理的区间对象和余纤维命题的子对象。我们证明立方体范畴是这种 $\mathbf{C}$ 的一个实例（以不止一种方式）；并且我们还证明在排中律成立的情况下，单纯形范畴 $\Delta$ 也是如此。

\subsection{区间对象 ($\ax{1}$--$\ax{4}$)}

回到具有其相关预层拓扑斯 $\widehat{\mathbf{C}}$ 的小范畴 $\mathbf{C}$ 的一般情况，如果我们取区间对象 $\ttI \in \widehat{\mathbf{C}}$ 为某个对象 $\mathtt{i} \in \mathbf{C}$ 的可表函子 $\mathtt{y}_{\tti} \triangleq \mathbf{C}(\ul , \tti)$，那么以下定理给出了这种区间对象满足公理 $\ax{1}$ 的一个有用判据。

\begin{theorem}
在预层拓扑斯 $\widehat{\mathbf{C}}$ 中，可表函子 $\ttI = \mathtt{y}_{\tti}$ 满足公理 $\ax{1}$，如果 $\mathbf{C}$ 是一个\textbf{余筛选范畴}，即如果 $\mathbf{Set}$ 中的有限积与 $\mathbf{C}^{\mathrm{op}}$ 上的余极限交换 [26]。
\end{theorem}

\begin{proof}
$\mathbf{C}$ 是余筛选的，如果余极限函子 $\operatorname{colim}_{\mathbf{C}^{\mathrm{op}}} : \widehat{\mathbf{C}} \to \mathbf{Set}$ 保持有限积。

回忆 $\operatorname{colim}_{\mathbf{C}^{\mathrm{op}}} : \widehat{\mathbf{C}} \to \mathbf{Set}$ 是常值预层函子 $\Delta : \mathbf{Set} \to \widehat{\mathbf{C}}$ 的左伴随，并且（因此）对于任何 $c \in \mathbf{C}$ 有 $\operatorname{colim}_{\mathbf{C}^{\mathrm{op}}} \mathtt{y}_c \cong 1$。因此当 $\mathbf{C}$ 是余筛选时，
\begin{align*}
\widehat{\mathbf{C}}(\mathtt{y}_c \times \mathtt{y}_{\tti}, \Delta\{0,1\}) &\cong \mathbf{Set}(\operatorname{colim}_{\mathbf{C}^{\mathrm{op}}}(\mathtt{y}_c \times \mathtt{y}_{\tti}), \{0,1\}) \\
&\cong \mathbf{Set}(\operatorname{colim}_{\mathbf{C}^{\mathrm{op}}} \mathtt{y}_c \times \mathtt{y}_{\tti}, \{0,1\}) \\
&\cong \mathbf{Set}(1 \times 1, \{0,1\}) \cong \{0,1\}
\end{align*}
由于 $\widehat{\mathbf{C}}$ 中的可判定子对象由 $1 + 1 = \Delta\{0,1\}$ 分类，这意味着 $\mathtt{y}_c \times \mathtt{y}_{\tti}$ 只有两个可判定子对象：最小和最大子对象。由于这对所有 $c \in \mathbf{C}$ 成立，因此 $\ttI = \mathtt{y}_{\tti}$ 满足 $\ax{1}$。
\end{proof}

余筛选性的一个更初等的刻画是：$\mathbf{C}$ 是非空的，并且对于每对对象 $c, c' \in \mathbf{C}$，跨度范畴 $c \leftarrow \cdot \rightarrow c'$ 是连通范畴 [2, 定理 2.15]。任何具有有限积的范畴显然具有这一性质。这对于上面定义的立方体范畴 $\square$ 成立，因此 [18] 中的模型（其中 $\mathbf{C} = \square$ 且 $i$ 是泛 De Morgan 代数）的区间满足 $\ax{1}$。不具有有限积但仍然是余筛选的一个相关例子是 $\Delta$，居住有限线性有序集 $[0 < 1 < \cdots < n]$ 的范畴，其 $\widehat{\Delta}$ 是\textbf{单纯形集}范畴，在同伦论中被广泛使用 [28]。因此 $\widehat{\Delta}$ 中区间的自然候选，即当 $\tti$ 是 1-单纯形 $[0 < 1]$ 时的 $\mathtt{y}_{\tti}$，满足 $\ax{1}$。

除了 $\ax{1}$ 之外，图 5.4 中关于区间的其他公理说 $\ttI$ 是由 $\ax{3}$ 和 $\ax{4}$ 给出的代数理论的一个非平凡 ($\ax{2}$) 模型，我们称之为\textbf{连接代数}。（另见 [27, 定义 1.7]，它在更抽象的设定中考虑类似概念。）对于立方体集，Yoneda 嵌入 $\mathtt{y} : \square \to \hatcube$ 将 $\square$ 中的泛 De Morgan 代数发送到 $\hatcube$ 中的一个 De Morgan 代数。这是一个非平凡连接代数：常数是最小和最大元素，二元运算是交和并。[18] 主题的一个明显变体是用连接代数的 Lawvere 理论替换 $\square$。还注意到 $\widehat{\Delta}$ 中的 1-单纯形是一个非平凡连接代数，常数是它的两个端点，二元运算是由 $[0 < 1]$ 上的保序二元运算最小值和最大值诱导的。

\subsection{余纤维命题 ($\ax{5}$--$\ax{8}$) 和严格性公理 ($\ax{9}$)}

在具有区间对象的拓扑斯中，有许多候选的子对象 $\mathtt{Cof} \rightarrowtail \Omega$ 满足图 5.4 中的公理 $\ax{5}$--$\ax{8}$。在一个极端，我们可以取 $\mathtt{Cof}$ 为整个 $\Omega$。在相反的极端，我们可以取（内部地）由以下要求归纳定义的子对象：它包含命题 $i = \ttzero$ 和 $i = \ttone$（对于所有 $i : \ttI$）并且对二元析取、依赖和 $\ttI$-索引合取封闭，从而获得满足 $\ax{5}$--$\ax{8}$ 的最小 $\mathtt{Cof}$。然而，余纤维命题还必须满足严格性公理 $\ax{9}$，我们接下来考虑这一点。

给定预层拓扑斯 $\widehat{\mathbf{C}}$，我们在与 $\widehat{\mathbf{C}}$ 相关联的 CwF 中工作，如定义 3.2.1 中所述。特别地，预层 $\Gamma \in \widehat{\mathbf{C}}$ 上的族由函子 $(\int \Gamma)^{\mathrm{op}} \to \mathbf{Set}$ 给出，其中 $\int \Gamma$ 是 $\Gamma$ 的通常元素范畴，$\mathrm{obj}(\int \Gamma) = (c \in \mathrm{obj}\ \mathbf{C}) \times \Gamma\,c$ 且 $(\int \Gamma)((c, x), (d, y)) = \{f \in \mathbf{C}(c, d) \mid \Gamma\,f\ y = x\}$。

如果 $\mathcal{S}$ 是环境集合论中的一个 Grothendieck 宇宙，那么它在该 CwF 中的 Hofmann-Streicher 提升 [31] 到一个宇宙 $\mathcal{U}$ 满足：$\widehat{\mathbf{C}}$ 中的态射 $\Gamma \to \mathcal{U}$ 命名了那些取值于 $\mathcal{S}\subseteq\mathbf{Set}$ 的族 $(\int \Gamma)^{\mathrm{op}} \to \mathcal{S}$。

\begin{definition}[$\Omega_{\mathrm{dec}}$]
预层拓扑斯 $\widehat{\mathbf{C}}$ 中的子对象分类器 $\Omega$ 将每个 $c \in \mathbf{C}$ 映射到 $c$ 上的筛集合 $\Omega(c)$，即前复合封闭的子集 $S \subseteq \mathrm{obj}(\mathbf{C}/c)$。令 $\Omega_{\mathrm{dec}} \rightarrowtail \Omega$ 为子预层，其在每个 $c \in \mathrm{obj}\ \mathbf{C}$ 处的值是 $\Omega(c)$ 中由 $\mathrm{obj}(\mathbf{C}/c)$ 的可判定子集构成的那些筛组成的子集。
\end{definition}

当然，如果环境集合论满足排中律，那么 $\Omega_{\mathrm{dec}} = \Omega$。一般来说，$\Omega_{\mathrm{dec}}$ 分类 $\widehat{\mathbf{C}}$ 中的单态射 $\alpha : F \rightarrowtail G$，使得对于所有 $c \in \mathrm{obj}\ \mathbf{C}$，（单射）函数 $\alpha_c : F\ c \to G\ c$ 具有可判定像。

\begin{theorem}
将宇宙 $\mathcal{U}$ 解释为 $\mathbf{Set}$ 中 Grothendieck 宇宙的 Hofmann-Streicher 提升 [31]，预层拓扑斯 $\widehat{\mathbf{C}}$ 中的子对象 $\mathtt{cof} \rightarrowtail \Omega$ 满足严格性公理 $\ax{9}$，如果它包含在 $\Omega_{\mathrm{dec}} \rightarrowtail \Omega$ 中。
\end{theorem}

\begin{proof}
对于每个 $c \in \mathrm{obj}\ \mathbf{C}$，假设给定 $S \in \Omega_{\mathrm{dec}}(c)$。因此 $S$ 是 $c$ 上的一个筛，并且对于每个 $c' \in \mathrm{obj}\ \mathbf{C}$ 和 $\mathbf{C}$-态射 $f : c' \to c$，$f \in S$ 是否成立是可判定的。我们也可以将 $S$ 视为子预层 $\iota : S \hookrightarrow \mathtt{y}_c$。

假设我们有族 $A : (\int S)^{\mathrm{op}} \to \mathcal{S}$，$B : (\int \mathtt{y}_c)^{\mathrm{op}} \to \mathcal{S}$ 以及 $A$ 与 $B$ 沿 $S \hookrightarrow \mathtt{y}_c$ 的限制之间的自然同构 $s$。对于每个 $\mathbf{C}$-态射 $f : c' \to c$，利用 $S$ 的可判定性我们可以定义双射 $s'(f) : B'(f) \cong B(f)$ 如下：
\[
B'(f) \triangleq \begin{cases} A(f) & \text{如果 } f \in S \\ B(f) & \text{如果 } \neg(f \in S) \end{cases} \qquad \text{并且} \qquad s'(f) \triangleq \begin{cases} s(f) & \text{如果 } f \in S \\ f & \text{如果 } \neg(f \in S) \end{cases}
\]
（与 [18, 定义 15] 比较）。我们通过在这些双射上转移 $B$ 的函子作用使 $B'$ 成为函子 $(\int \mathtt{y}_c)^{\mathrm{op}} \to \mathcal{S}$。这样做之后，$s'$ 成为自然同构 $B' \cong B$；并且根据定义，它沿 $S \hookrightarrow \mathtt{y}_c$ 的限制是 $s$。
\end{proof}

\begin{corollary}
令 $\mathbf{C}$ 为具有有限积的小范畴，包含一个具有非平凡连接代数结构 $\ttzero, \ttone : 1 \to \tti, \sqcap, \sqcup : \tti \times \tti \to \tti$（参见图 5.4）的对象 $\tti$。假设对于每个对象 $c \in \mathbf{C}$，集合 $\mathbf{C}(c, \tti)$ 具有可判定相等。那么如果我们取区间 $\ttI$ 为 $\mathtt{y}_{\tti}$，$\mathtt{Cof}$ 为 $\Omega_{\mathrm{dec}}$，预层拓扑斯 $\widehat{\mathbf{C}}$ 满足图 5.4 中的所有公理。
\end{corollary}

\begin{proof}
我们已经在第 5.6.1 节中注意到，这种 $\ttI$ 的选择满足公理 $\ax{1}$--$\ax{4}$。每个集合 $\mathbf{C}(c, \tti)$ 的可判定性意味着 $\widehat{\mathbf{C}}$ 中的子对象 $\{0\} \rightarrowtail \ttI$ 和 $\{1\} \rightarrowtail \ttI$ 通过 $\Omega_{\mathrm{dec}} \rightarrowtail \Omega$ 分解，因此当 $\mathtt{Cof} = \Omega_{\mathrm{dec}}$ 时公理 $\ax{5}$ 得到满足。注意这种 $\mathtt{Cof}$ 的选择自动满足 $\ax{6}$ 和 $\ax{7}$；并且由定理 5.6.3 它满足公理 $\ax{9}$。因此只需检查公理 $\ax{8}$ 是否满足。我们在引理 5.1.5(ii) 中看到，这个公理等价于要求余纤维在由 $\ttI$ 取指数下封闭。在这种情况下，余纤维是由 $\Omega_{\mathrm{dec}}$ 分类的单态射 $\alpha : F \rightarrowtail G$，我们在定义 5.6.2 之后注意到它们由每个函数 $\alpha_c \in \mathbf{Set}(F\ c, G\ c)$ 具有可判定像这一事实刻画。这些单态射在由 $\ttI$ 取指数下的封闭性来自于 $\mathbf{C}$ 具有有限积且 $\ttI = \mathtt{y}_{\tti}$ 是可表的事实；因为那时 $\ttI \to (\ul )$ 同构于由 $(\ul) \times \tti : \mathbf{C} \to \mathbf{C}$ 预复合诱导的函子 $\widehat{\mathbf{C}} \to \widehat{\mathbf{C}}$，这显然保持逐分量可判定像性质。
\end{proof}

上述关于公理 $\ax{8}$ 的论证不适用于 $\Delta$，因为它不具有有限积；我们不知道构造性单纯形集是否满足该公理。然而，在排中律 (LEM) 成立的情况下，$\Omega_{\mathrm{dec}} = \Omega$ 并且我们有：

\begin{corollary}[经典单纯形集]
假设 LEM 在集合论语元理论中成立，那么如果我们取 $\ttI$ 为 1-单纯形上的可表预层，$\mathtt{Cof}$ 为整个 $\Omega$，单纯形集的预层拓扑斯 $\widehat{\Delta}$ 满足图 5.4 中的公理。
\end{corollary}

\begin{proof}
我们已经在第 5.6.1 节中注意到，这种 $\ttI$ 的选择满足公理 $\ax{1}$--$\ax{4}$。如果 $\mathtt{Cof} = \Omega$（即 $\mathtt{cof} = \lambda\ul  \to \top$），那么公理 $\ax{5}$--$\ax{8}$ 平凡成立。此外，如果 LEM 成立，那么 $\Omega_{\mathrm{dec}} = \Omega$，因此由定理 5.6.3 公理 $\ax{9}$ 成立。
\end{proof}

\begin{remark}
作为定理 5.6.3 的部分逆，我们有如果 $\ax{9}$ 被与 $\widehat{\mathbf{C}}$ 相关联的 CwF 中的 Hofmann-Streicher 宇宙满足，那么每个余纤维单态射 $\alpha : F \rightarrowtail G$ 的分量函数 $\alpha_c \in \mathbf{Set}(F\ c, G\ c)$（$c \in \mathrm{obj}\ \mathbf{C}$）的像是 $\neg\neg$-闭子集。为看到这一点，我们可以应用 Andrew Swan 的一个论证 [私人通信]，它依赖于环境集合论中有
\begin{equation}
(X = \emptyset) = \forall x \in X.\ \bot = \neg\neg(\forall x \in X.\ \bot) = \neg\neg(X = \emptyset)
\end{equation}
这一事实。

假设给定 $c \in \mathrm{obj}\ \mathbf{C}$ 和 $S \in \mathtt{Cof}(c)$。我们必须使用公理 $\ax{9}$ 来证明 $S$ 是 $\mathrm{obj}(\mathbf{C}/c)$ 的 $\neg\neg$-闭子集。令 $A : (\int S)^{\mathrm{op}} \to \mathcal{S}$ 为将每个 $(c', f)$ 映射到 $\{\emptyset\}$ 的常值函子；令 $B : (\int \mathtt{y}_c)^{\mathrm{op}} \to \mathcal{S}$ 将每个 $(c', f)$ 映射到 $\{\{\emptyset\}, \{\emptyset \mid f \in S\}\}$（这确实扩展为一个函子，因为 $S$ 是一个筛）。$B$ 沿 $S \hookrightarrow \mathtt{y}_c$ 的限制同构于 $A$，因此由 $\ax{9}$ 存在某个 $B' : (\int \mathtt{y}_c)^{\mathrm{op}} \to \mathcal{S}$ 其沿 $S \hookrightarrow \mathtt{y}_c$ 的限制等于 $A$，以及某个同构 $s' : B' \cong B$。对于任何 $(c', f) \in \mathrm{obj}(\int \mathtt{y}_c)$，假设 $X \in B'(c', f)$；那么 $f \in S \Rightarrow X = \emptyset$ 因此 $\neg\neg(f \in S) \Rightarrow \neg\neg(X = \emptyset)$，从而由 (5.52)，
\[
\neg\neg(f \in S) \Rightarrow B'(c', f) = \{\emptyset\} \Rightarrow B(c', f) \cong \{\emptyset\} \Rightarrow f \in S
\]
因此 $S$ 确实是 $\mathrm{obj}(\mathbf{C}/c)$ 的 $\neg\neg$-闭子集。

注意这个结果意味着，除非环境集合论满足 LEM，否则不可能取 $\mathtt{Cof}$ 为整个 $\Omega$ 并满足 $\ax{9}$。
\end{remark}

由于在构造性设定中自由 De Morgan 代数的相等是可判定的，因此当 $\mathbf{C} = \square$（立方体范畴，定义 3.3.1）时推论 5.6.4 给出了我们公理的一个模型。然而，这使用了与 [18, 第 4.1 节] 中不同的余纤维性选择。在本节剩余部分，我们检查 CCHM 的余纤维性概念是否满足我们的公理。

\begin{definition}[{[18]} 中的余纤维命题]
对于立方体范畴的每个对象 $I \in \mathrm{obj}\ \square$，Cohen 等人定义\textbf{面格} $\mathbb{F}(I)$ 为由符号 $(i = 0)$ 和 $(i = 1)$（对于每个 $i \in I$）生成的分配格，服从方程 $(i = 0) \land (i = 1) = \bot$。

由于自由 De Morgan 代数 $\mathsf{dM}(I)$ 作为分配格由符号 $i$ 和 $1\mi i$（当 $i$ 遍历有限集 $I$）自由生成，我们可以将 $\mathbb{F}(I)$ 视为 $\mathsf{dM}(I)$ 的商格，通过将 $i$ 映射到 $(i = 1)$、$1\mi i$ 映射到 $(i = 0)$ 的函数；我们记 $q_I : \mathsf{dM}(I) \to \mathbb{F}(I)$ 为商映射\footnote{如果 $r\in\mathsf{dM}$, [18] 中使用记号 $(r=1)$ 代表 $q_{I}(r)$。}。不难看到，对于每个 $f \in \mathbf{C}(J, I)$，相应的 De Morgan 代数同态 $\mathsf{dM}(I) \to \mathsf{dM}(J)$（我们也记为 $f$）诱导面格之间的格同态：
\[
\begin{tikzcd}
\mathsf{dM}(I) \arrow[r,"f"] \arrow[d, "q_I"'] & \mathsf{dM}(J) \arrow[d, "q_J"] \\
\mathbb{F}(I) \arrow[r, "\mathbb{F}f"'] & \mathbb{F}(J)
\end{tikzcd}
\]
这使得 $\mathbb{F}$ 成为 $\hatcube$ 中的一个对象，并且存在单态射 $m : \mathbb{F} \rightarrowtail \Omega$，其在 $I \in \mathrm{obj}\ \square$ 处的分量将每个 $\varphi \in \mathbb{F}(I)$ 发送到筛
\begin{equation}
m_I(\varphi) = \{\cdot \xrightarrow{f} I \mid \mathbb{F}f\ \varphi = \top\}
\end{equation}

我们现在在 $\hatcube$ 中取由这个单态射 $m : \mathbb{F} \rightarrowtail \Omega$ 给出的子对象 $\mathtt{Cof} \rightarrowtail \Omega$。公理 $\ax{1}$--$\ax{4}$ 无需改变即可由推论 5.6.4 得出；公理 $\ax{5}$--$\ax{7}$ 由面格 $\mathbb{F}(I)$ 的定义得出。因此只需检查公理 $\ax{8}$ 和 $\ax{9}$。

回忆 $\hatcube$ 中的区间对象是单元素子集 $\{i\} \in \mathrm{obj}\ \square$ 上的可表预层 $\ttI = \mathtt{y}_{\{i\}}$。对于任意对象 $I \in \mathrm{obj}\ \square$，设其中有 $n$ 个不同元素 $i_1, \ldots, i_n \in \mathbb{D}$，可表 $\mathtt{y}_I$ 同构于 $\hatcube$ 中的 $n$ 重积 $\ttI^n$。因此筛 (5.53) 对应于 $n$-立方体 $\ttI^n$ 的子对象。确实，面格 $\mathbb{F}(I)$ 中的每个元素是有限个不可约元素的有限并；每个不可约元素是形如 $i = 0$ 或 $j = 1$ 的原子条件的有限合取。$\ttI^n$ 的相应子对象是面的有限并，即通过将某些坐标设为 0 或 1 得到的 $\ttI^n$ 的子对象。$\mathbb{F}(I)$ 元素的这种析取范式意味着其等式是可判定的，因此这个 $\mathtt{Cof}$ 包含在 $\Omega_{\mathrm{dec}}$ 中且公理 $\ax{9}$ 得到满足（定理 5.6.3）。最后，公理 $\ax{8}$ 来自于余纤维沿投影 $\pi_1 : \ttI^n \times \ttI \to \ttI^n$ 的拉回具有（稳定的）右伴随，或者等价地，格态射 $\bbF\,\pi_1 : \mathbb{F}(I) \to \mathbb{F}(I \cup \{i\})$（对于任意 $I \in \mathrm{obj}\ \square$ 和 $i \in \mathbb{D} - I$）具有（稳定的）右伴随这一事实。这是面格的\textbf{量词消去}结果；参见 [18, 引理 2]。
\end{definition}

\section{相关工作}

本章呈现的工作受到 Coquand [19] 的建议启发，即 [18] 中发展的某些构造可能使用拓扑斯的内部逻辑来更好地理解。我们展示了如何在拓扑斯的内部类型论中表达 Cohen、Coquand、Huber 和 Mörtberg 的纤维化概念。与例如 Gambino 和 Sattler [27, 第 3 节] 为同样目的使用的更抽象的弱分解系统和模型范畴的范畴论方法相比，内部语言的使用允许一个令人愉悦的简单描述（定义 5.3.3）。Birkedal 等人 [12] 发展了守卫立方类型论，其语义基于 [18] 的公理化版本，并在预层拓扑斯的内部逻辑中给出。

在配备有类似区间对象的拓扑斯的框架内，我们发现相当简单的公理集合（图 5.4）足以使其建模具有满足弱形式泛等性的内涵恒等类型的 Martin-Löf 类型论。特别地，区间上只需要简单的连接代数结构，而非 De Morgan 代数结构。此外，适合一致 Kan 填充的命题集合不是紧密约束的，可以以多种方式选择。在第 5.6 节中，我们只考虑了预层范畴如何满足我们的公理。考虑一般 Grothendieck 拓扑斯（其中预层被限制为给定覆盖概念的层）中的模型可能是有趣的，特别是如 Johnstone 的拓扑拓扑斯 [36] 这样的大拓扑斯；这允许区间对象是通常的拓扑区间（对应的可表层），因此类型论的模型与空间的经典同伦类型有相当直接的联系。然而，尽管 Hofmann-Streicher [31] 的宇宙构造（[18] 第 8.2 节中满足完全泛等性公理的纤维化宇宙构造的基础）可以通过使用层化 [59, 第 3 节] 从预层拓扑斯扩展到层拓扑斯，但层化似乎与 CCHM 的纤维化概念交互不佳。在另一个方向上，Frumin 和 Van Den Berg [25] 的最新工作使用我们的初等、公理化方法，使用非 Grothendieck 拓扑斯，即有效拓扑斯 [35]。最后，Uemura 也应用这种方法证明了立方类型论中命题调整公理的独立性。

\chapter{分解泛等性公理}

在上一章中，我们使用初等拓扑斯的内部语言给出了 Cohen 等人 [18] 模型的公理化表述。虽然使用内部语言的方法是新的，但许多证明的结构，例如纤维化在类型构造子下封闭的事实，或使用粘合来发展泛等性，都遵循了 Cohen 等人原始工作的相同结构。在本章中，我们将使用内部语言来呈现新的工作，即：在任何满足我们公理的拓扑斯中给出泛等性公理的一个替代证明。

我们通过首先将泛等性分解为一系列更简单的性质来做到这一点。我们解释了为什么这些新性质中除了一个之外，不仅应该在满足我们公理的拓扑斯中成立，而且事实上在任何以通常方式构造 CwF 的预层拓扑斯中都应该成立（如第 3.2 节所述）。然后我们解释了剩下的那个性质如何在任何满足我们原始公理的拓扑斯中容易地得到验证。事实上，这个我们称之为\textbf{收缩}的构造是一般粘合构造的一个特例。然而，其定义和随后的证明比用于粘合的那些要简单得多。

\section{分解}

回忆定理 2.1.9 中给出的适当泛等性公理分解为一个朴素形式连同一条计算规则。在本节中，我们在具有函数外延性的 MLTT 中进一步将泛等性公理分解为更简单的性质，但不假设 UIP 或泛等性。注意本节中给出的证明应在源语言中解释，即我们最终希望建模的语言，而不是在拓扑斯的内部语言中。我们证明宇宙 $U_i$ 的泛等性公理等价于表 6.1 中给出的宇宙 $U_i$ 的公理 (1) 到 (5)。

我们首先将朴素泛等性 $\mathtt{UA}_i$ 分解为公理 (1)--(3)。这些公理也来自 $\mathtt{UA}_i$。回忆我们取函数外延性作为环境假设。首先，我们回顾单例类型的定义。

\begin{table}[h]
\centering
\begin{tabular}{|c|l|l|l|}
\hline
& \textbf{公理} & \textbf{前提} & \textbf{等式} \\
\hline
(1) & $\mathtt{unit}$ & $:$ & $A = \sum_{a:A} 1$ \\
(2) & $\mathtt{flip}$ & $:$ & $\sum_{a:A}\sum_{b:B} C\,a\,b = \sum_{b:B}\sum_{a:A} C\,a\,b$ \\
(3) & $\mathtt{contract}$ & $: \mathtt{isContr}\,A \to$ & $A = 1$ \\
(4) & $\mathtt{unit}\beta$ & $:$ & $\mathtt{coerce}\,\mathtt{unit}\,a = (a, *)$ \\

\hline
\end{tabular}
\caption{($A, B : U_i$, $C : A \to B \to U_i$, $a : A$, $b : B$ 且 $c : C\,a\,b$，对于某个宇宙 $U_i$)}
\end{table}

\begin{definition}[单例]
给定类型 $A$ 和元素 $a : A$，我们可以定义
\[
\mathtt{sing}(a) \triangleq \sum_{x:A}(a = x)
\]
为 $A$ 中等于 $a$ 的元素类型。通过路径归纳容易证明，类型 $\mathtt{sing}(a)$ 在定义 2.1.1 的意义上总是可收缩的。
\end{definition}

\begin{theorem}
宇宙 $U_i$ 的公理 (1)--(3) 合起来逻辑等价于 $\mathtt{UA}_i$。
\end{theorem}

\begin{proof}
我们首先证明正向方向。假设给定公理 (1) 到 (3)。我们现在旨在定义一个项 $\mathtt{ua} : \mathtt{UA}_i$。给定任意类型 $A, B : U_i$ 和等价 $(f, e) : A \simeq B$，那么对于所有 $a : A$ 我们知道 $\mathtt{sing}(a)$ 是可收缩的，因此由 $\mathtt{contract}$ 我们有：
\[
1 = \mathtt{sing}(f\,a) = \sum_{b:B} f\,a = b
\]
此外，对于任何 $b : B$，由 $e(b) : \mathtt{isContr}(\mathtt{fib}_f(b))$ 我们知道 $\mathtt{fib}_f(b)$ 是可收缩的。因此，由 $\mathtt{contract}$，我们有：
\[
1 = \mathtt{fib}_f(b) = \sum_{a:A} f\,a = b
\]
因此，我们如下定义 $\mathtt{ua}(f, e) : A = B$：
\begin{align*}
A &= \sum_{a:A} 1 && \text{由 unit} \\
&= \sum_{a:A} \sum_{b:B} f\,a = b && \text{由 funext 和 contract 在 } \mathtt{sing}(f\,a) \text{ 上} \\
&= \sum_{b:B} \sum_{a:A} f\,a = b && \text{由 flip} \\
&= \sum_{b:B} 1 && \text{由 funext 和 contract 在 } \mathtt{fib}_f(b) \text{ 上} \\
&= B && \text{由 unit}
\end{align*}
其中 $A = B$ 的证明由上述计算每一步的连接给出。

反向方向来自于明显的映射
\[
\begin{aligned}
A &\to \sum_{a:A} 1 & \sum_{a:A}\sum_{b:B} C\,a\,b &\to \sum_{b:B}\sum_{a:A} C\,a\,b \\
a &\mapsto (a, *) & (a, b, c) &\mapsto (b, a, c)
\end{aligned}
\]
都容易证明是双可逆的，从而是等价的，以及任何可收缩类型等价于 1 [62, 引理 3.11.3]{62} 这一事实。因此给定 $\mathtt{ua} : \mathtt{UA}_i$ 我们只需将这些等价应用于它们即可得到所需的等式 (1)--(3)。
\end{proof}

接下来，我们将朴素泛等性 $\mathtt{UA}\beta_i$ 的计算规则分解为公理 (4) 和 (5)。由于 $\mathtt{UA}\beta_i$ 依赖于 $\mathtt{UA}_i$，而公理 (4) 和 (5) 分别依赖于公理 (1) 和 (2)，我们事实上证明了 $\mathtt{UA}_i$ 和 $\mathtt{UA}\beta_i$ 这一对，与公理 (1)--(5) 之间的逻辑等价。

\begin{lemma}
函数 $\mathtt{coerce}$ 是复合的。即，给定类型 $A, B, C : U_i$，以及等式 $p : A = B$ 和 $q : B = C$，我们有 $\mathtt{coerce}(p \cdot q) = \mathtt{coerce}(q) \circ \mathtt{coerce}(p)$，其中 $p \cdot q$ 是 $A = C$ 的传递性证明。
\end{lemma}

\begin{proof}
通过对 $p$ 或 $q$ 之一，或两者进行路径归纳，是直接的。
\end{proof}

\begin{theorem}
宇宙 $U_i$ 的公理 (1)--(5) 合起来逻辑等价于 $\sum_{\mathtt{ua}:\mathtt{UA}_i} \mathtt{UA}\beta_i(\mathtt{ua})$。
\end{theorem}

\begin{proof}
对于正向方向，我们从定理 6.1.2 知道公理 (1) 到 (3) 允许我们构造一个项 $\mathtt{ua} : \mathtt{UA}_i$。此外，如果我们假设公理 (4) 和 (5)，那么我们可以证明对于所有 $(f, e) : A \simeq B$ 有 $\mathtt{coerce}(\mathtt{ua}(f, e)) = f$，如下。

由于 $\mathtt{ua}$ 被构造为五个等式的连接，根据引理 6.1.3，我们有沿 $\mathtt{ua}(f, e)$ 的强制等于沿复合等式 $\mathtt{ua}(f, e)$ 每个阶段强制结果。我们从 $\mathtt{unit}\beta$ 和 $\mathtt{flip}\beta$ 分别知道沿 $\mathtt{unit}$ 和 $\mathtt{flip}$ 的强制结果。在 $\mathtt{contract}$ 的情况下，我们在两个可收缩类型之间强制，因此可以将强制取为我们喜欢的任何函数，因为所有这样的函数都是命题相等的。因此，从任意 $a : A$ 开始，我们可以追踪此过程每个阶段发生的情况，如下：
\[
a \mapsto (a, *) \mapsto (a, f\,a, \mathtt{refl}) \mapsto (f\,a, a, \mathtt{refl}) \mapsto (f\,a, *) \mapsto f\,a
\]
因此我们看到对于所有 $a : A$ 有 $\mathtt{coerce}(\mathtt{ua}(f, e))(a) = f(a)$，因此由函数外延性我们有 $\mathtt{coerce}(\mathtt{ua}(f, e)) = f$，如所要求。

对于反向方向，我们假设给定 $\mathtt{ua} : \mathtt{UA}_i$ 和 $\mathtt{ua}\beta : \mathtt{UA}\beta_i(\mathtt{ua})$。我们现在可以应用定理 6.1.2 从 $\mathtt{ua}$ 构造满足公理 (1) 到 (3) 的项 $\mathtt{unit}$、$\mathtt{flip}$ 和 $\mathtt{contract}$。

由于 $\mathtt{unit}$ 和 $\mathtt{flip}$ 是通过将 $\mathtt{ua}$ 应用于明显的等价构造的，因此由 $\mathtt{ua}\beta$ 我们知道将这些等式应用 $\mathtt{coerce}$ 将返回我们开始时的等价。由此我们可以容易地构造满足公理 (4) 和 (5) 的项 $\mathtt{unit}\beta$ 和 $\mathtt{flip}\beta$。
\end{proof}

\begin{corollary}
宇宙 $U_i$ 的公理 (1)--(5) 合起来逻辑等价于 $U_i$ 的适当泛等性公理。
\end{corollary}

\begin{proof}
通过结合定理 2.1.9 和定理 6.1.4。
\end{proof}

\section{在类型论模型中的应用}

在本节中，我们讨论了推论 6.1.5 给出的结果在尝试构造泛等类型论模型时为何有用的一个原因。具体而言，我们相信这种分解对于证明具有区间对象的类型论模型支持泛等性公理特别有用。我们首先用一般术语解释为什么我们相信情况如此，然后精确说明在第 5.1 节描述的结构拓扑斯的具体情况下会发生什么。如那里所讨论的，这个结果因此将适用于 [18] 中呈现的立方体集模型，但它也应该适用于许多类似的类型论模型 [11, 6, 8, 54, 12]。

注意我们假设函数外延性。泛等性的每个模型都必须满足函数外延性 [62, 第 4.9 节]，但在类型论模型中验证函数外延性通常比验证适当的泛等性公理容易得多。特别地，函数外延性在任何包含适当区间对象的类型论中都成立，参见 [62, 引理 6.3.2]。

经验表明，公理 (1)、(2)、(4) 和 (5) 在许多潜在的泛等类型论模型中容易验证。为了理解为什么，考虑这种模型中 $A \simeq B$ 的解释是有用的。类型论中的命题等式通常不被解释为模型元理论中的等式，而是作为类型上的构造，例如 HoTT 模型中的路径空间。因此，使用带族范畴的记号，类型论中的等价将产生项
\[
f \in \mathtt{Ter}(\Gamma \vdash A \to B) \qquad \text{和} \qquad g \in \mathtt{Ter}(\Gamma \vdash B \to A)
\]
它们不是精确的逆，而是模命题等式解释的逆，即存在项
\[
\alpha \in \mathtt{Ter}(\Gamma \vdash \mathtt{Id}_{A \to B}(g \circ f,  id )) \qquad \text{和} \qquad \beta \in \mathtt{Ter}(\Gamma \vdash \mathtt{Id}_{B \to A}(f \circ g,  id )).
\]
这些的存在并不意味着 $g \circ f =  id $ 或 $f \circ g =  id $，例如在立方体集模型中 $\alpha$ 和 $\beta$ 将是复合与恒等之间的路径，如第 3.3.3 节所述。然而，在许多模型中，$A$ 和 $\sum_{a:A} 1$ 的解释，以及 $\sum_{a:A}\sum_{b:B} C\,a\,b$ 和 $\sum_{b:B}\sum_{a:A} C\,a\,b$ 的解释将是同构的，即存在来回的态射，它们在模型元理论的等式意义下是逆。这在第 3.2 节描述的那种类型论预层模型中是成立的，更一般地在任何验证 1 和 $\Sigma$ 的 eta-规则的模型中也应该成立，因为那时来回的映射在类型论的判断等式意义下是逆，因此它们在模型中的解释将是严格逆。

这意味着我们可以通过证明这种更强的同构概念产生类型之间的命题等式来满足 (1) 和 (2)。验证公理 (4) 和 (5) 然后应该简化为涉及此构造的两个实例的相当直接的计算。

这留下了公理 (3)，它捕捉了每个可收缩空间可以连续变形为一个点的同伦条件。希望是验证前面的公理应该相当直接，将此作为唯一需要检查的非平凡条件。

我们现在检查在第 5.1 节中公理化的结构拓扑斯的具体情况下会发生什么。为便于引用，我们将此拓扑斯称为 $\mathcal{E}$。对于本节的其余部分，我们回到在 $\mathcal{E}$ 的内部类型论中工作。

\subsection{严格化}

回忆第 5.1.3 节中我们假定了以下项的存在：
\begin{align*}
\ax{9} : &\{\varphi : \mathtt{Cof}\}(A : [\varphi] \to \mathcal{U})(B : \mathcal{U})(s : (u : [\varphi]) \to (A\,u \cong B)) \to \\
&(B' : \mathcal{U}) \times \{s' : B' \cong B \mid \forall(u : [\varphi]).\,A\,u = B' \land s\,u = s'\}
\end{align*}

换言之，这说给定任何对象 $B : \mathcal{U}$ 和任何余纤维部分对象 $A : [\varphi] \to \mathcal{U}$ 使得 $A$ 在其定义的每个地方都同构于 $B$，那么可以构造一个新对象 $B' : \mathcal{U}$ 扩展 $A$，同构于 $B$，并且这个同构扩展原始同构。

我们现在将这个严格化性质从对象提升到纤维化。

\begin{theorem}
给定 $\Gamma : \mathcal{U}$ 和 $\Phi : \Gamma \to \mathtt{Cof}$，余纤维-部分纤维化 $A : \mathtt{Fib}(\Gamma|\Phi)$ 和全纤维化 $B : \mathtt{Fib}\,\Gamma$ 且满足 $\mathtt{iso} : A \cong B[\iota]$，我们可以构造一个新类型和同构：
\[
A' : \mathtt{Fib}(\Gamma) \qquad \text{和} \qquad \mathtt{iso}' : A' \cong B
\]
使得
\[
A'[\iota] = A \qquad \text{和} \qquad \mathtt{iso}' \circ \iota = \mathtt{iso}
\]
其中 $\iota$ 是包含 $\Gamma|\Phi \hookrightarrow \Gamma$。
\end{theorem}

\begin{proof}
给定 $\Gamma : \mathcal{U}$，$\Phi : \Gamma \to \mathtt{Cof}$，$(A, \alpha) : \mathtt{Fib}(\Gamma|\Phi)$ 和 $(B, \beta) : \mathtt{Fib}\,\Gamma$ 且满足 $\mathtt{iso} : A \cong B \circ \iota$，我们定义 $A'$, $\mathtt{iso}'$ 为：
\begin{align*}
A'\,x &\triangleq \mathtt{fst}(\ax{9}(A(x, \_), B\,x, \mathtt{iso}(x, \_))) \\
\mathtt{iso}'\,x &\triangleq \mathtt{snd}(\ax{9}(A(x, \_), B\,x, \mathtt{iso}(x, \_)))
\end{align*}

现在考虑需要成立的等式。由 $\ax{9}$ 的性质我们已经有 $A' \circ \iota = A$ 和 $\mathtt{iso}' \circ \iota = \mathtt{iso}$。因此我们只需定义合成结构 $\alpha' : \mathtt{isFib}\,A'$ 使得 $\alpha'[\iota] = \alpha$。

由于 $A' \cong B$ 且 $\beta : \mathtt{isFib}\,B$，我们可以使用引理 5.3.8 推导出 $A'$ 具有合成结构，我们称之为 $\alpha'_{\mathrm{pre}}$。然后我们定义 $\alpha' \triangleq \mathtt{realign}(\Phi, \alpha, \alpha'_{\mathrm{pre}})$ 使用引理 5.3.10 以确保 $\alpha'[\iota] = \alpha$。

注意，由于 $\mathtt{realign}$ 在重索引下是稳定的，在这个意义上对于任何 $\gamma : \Delta \to \Gamma$ 有 $\mathtt{realign}(\Phi, \beta, \alpha)[\gamma] = \mathtt{realign}(\Phi \circ \gamma, \beta[\gamma \times  id ], \alpha[\gamma])$，并且上面使用的所有其他构造都是逐纤维执行的，我们可以推断定理 6.2.1 中给出的构造在重索引下也是稳定的。
\end{proof}

\subsection{纤维化之间的错位路径}

我们现在引入纤维化之间的一种新关系，我们称之为\textbf{错位路径}。这类似于定义 5.5.1 中引入的纤维化之间路径的概念，只不过在端点处，路径只需同构于 $A$ 和 $B$，而非等于 $A$ 和 $B$。

\begin{definition}[纤维化之间的错位路径相等性]
定义 CCHM 纤维化之间的错位路径类型 $\_\sim_{\cong}\_ : \{\Gamma : \mathcal{U}\} \to \mathtt{Fib}\,\Gamma \to \mathtt{Fib}\,\Gamma \to \mathcal{U}_1$ 为
\[
A \sim_{\cong} B \triangleq (P : \mathtt{Fib}(\Gamma \times \mathbb{I})) \times (A \cong P[\langle  id , 0\rangle]) \times (B \cong P[\langle  id , 1\rangle])
\]
\end{definition}

我们可以证明每个错位路径都可以改进为纤维化之间的常规路径。首先，我们引入纤维化上的一个构造。

\begin{definition}[纤维化的不交并]
给定纤维化 $A, B : \mathtt{Fib}\ \Gamma$，我们定义一个新的纤维化
$$A \veebar B : \mathtt{Fib}\big((\Gamma \times \mathtt{I})|\Phi\big) \qquad \text{其中} \qquad \Phi(x,i) \triangleq (i = \mathtt{0}) \veebar (i = \mathtt{1})$$
由 $(A,\alpha) \veebar (B,\beta) \triangleq (C,\gamma)$ 给出，其中
\begin{align*}
& C : (\Gamma \times \mathtt{I})|\Phi \to \mathcal{U} \\
& C\big((x,i),u\big) \triangleq \Big(\big(\lambda\_ : [i=0] \to A\ x\big) \cup \big(\lambda\_ : [i=1] \to B\ x\big)\Big)u
\end{align*}

这里 $C$ 是族 $A$ 和 $B$ 的一种不交并，注意到 $(\Gamma \times \mathtt{I})|\Phi \cong \Gamma + \Gamma$，因此我们可以将 $C$ 视为本质上是 $[A,B] : \Gamma + \Gamma \to \mathcal{U}$。

为了看出 $C$ 是纤维化的，我们观察到区间 $\mathtt{I}$ 在公理 $\ax{1}$（图 5.4）的意义上是内部连通的。这意味着任何路径 $p : \mathtt{I} \to (\Gamma \times \mathtt{I})|\Phi$ 必须要么分解为 $p = \langle p', \mathtt{0}, * \rangle$，要么分解为 $p = \langle p', \mathtt{1}, * \rangle$。

因此，$C$ 的任何复合问题必须要么完全在 $A$ 中，此时我们使用 $\alpha$ 来构造解；要么完全在 $B$ 中，此时我们使用 $\beta$。更多细节我们请读者参考定理 5.5.3，其中族 $C$ 作为中间构造出现，并且我们在那里更详细地定义了这个纤维化结构。注意这个构造在重索引下是稳定的：
$$A[\gamma] \veebar B[\gamma] = (A \veebar B)[\gamma \times  id  \times  id ]$$
对于任何 $\gamma : \Delta \to \Gamma$。最后，观察到对于所有 $A, B : \mathtt{Fib}\ \Gamma$ 我们有：
$$(A \veebar B)[\langle  id , \mathtt{0}, * \rangle] = A \qquad \text{且} \qquad (A \veebar B)[\langle  id , \mathtt{1}, * \rangle] = B$$
\end{definition}

\begin{definition}[不对齐路径的拼接]
给定 $\Gamma : \mathcal{U}$，纤维化 $A, B : \mathtt{Fib}\ \Gamma$ 和一个不对齐路径 $(D, f_0, f_1) : A \sim_{\cong} B$，我们定义如下：
$$f_0 \veebar f_1 : A \veebar B \cong D[\iota]$$
\begin{align*}
& (f_0 \veebar f_1)(x,i,u) : \big(\mathtt{fst}(A \veebar B)\big)(x,i,u) \to \big(\mathtt{fst}\ D\big)(x,i) \\
& (f_0 \veebar f_1)(x,i,u) \triangleq \begin{cases} f_0\ x & \text{当}\ u : [i=0] \\ f_1\ x & \text{当}\ u : [i=1] \end{cases}
\end{align*}
其中 $(x,i,u) : (\Gamma \times \mathtt{I})|\Phi$。观察到对于所有 $\gamma : \Delta \to \Gamma$ 我们有，
$$(f_0 \circ \gamma) \veebar (f_1 \circ \gamma) = (f_0 \veebar f_1) \circ (\gamma \times  id  \times  id )$$
并且也有：
$$(f_0 \veebar f_1) \circ \langle  id , \mathtt{0}, * \rangle = f_0 \qquad \text{且} \qquad (f_0 \veebar f_1) \circ \langle  id , \mathtt{1}, * \rangle = f_1$$
\end{definition}

\begin{lemma}[改进不对齐路径]
存在函数
$$\mathtt{improve} : \{\Gamma : \mathcal{U}\}\{A\ B : \mathtt{Fib}\ \Gamma\} \to A \sim_{\cong} B \to A \sim_{\mathcal{U}} B$$
\end{lemma}

\begin{proof}
取 $\Gamma : \mathcal{U}$，$A, B : \mathtt{Fib}\ \Gamma$ 和 $(P, f_0, f_1) : A \sim_{\cong} B$ 并观察到 $f_0 \veebar f_1 : A \veebar B \cong P[\iota]$。因此我们可以使用定理 6.2.1 来严格化 $P$ 以获得 $P' : \mathtt{Fib}(\Gamma \times \mathbb{I})$ 使得 $P'[\iota] = A \veebar B$，其中 $\iota$ 是限制 $(\Gamma \times \mathtt{I})|\Phi \to \Gamma \times \mathtt{I}$。现在考虑将 $P'$ 沿着 $\langle  id , \mathtt{0} \rangle : \Gamma \to \Gamma \times \mathtt{I}$ 重索引，我们得到：
$$P'[\langle  id , \mathtt{0} \rangle] = P'[\iota \circ \langle  id , \mathtt{0}, * \rangle] = P'[\iota][\langle  id , \mathtt{0}, * \rangle] = (A \veebar B)[\langle  id , \mathtt{0}, * \rangle] = A$$
且类似地 $P'[\langle  id , \mathtt{1} \rangle] = B$。因此我们有 $P' : A \sim_{\mathcal{U}} B$ 如所需。此外我们有
$$\mathtt{improve}(P[\gamma \times  id ], f_0 \circ \gamma, f_1 \circ \gamma) = \mathtt{improve}(P, f_0, f_1)[\gamma \times  id ]$$
对于任何 $\gamma : \Delta \to \Gamma$。
\end{proof}

\subsection{函数外延性}

如前所述，函数外延性在任何包含具有特定计算性质的区间对象/类型的类型理论中直接成立，参见 [62, 引理 6.3.2]。对于立方体类型论的情况，参见注记 5.3.16 或 [18, 第 3.2 节]。

\subsection{公理 (1)、(2)、(4) 和 (5)}

如前所述，我们可以通过证明存在一种在严格同构的（纤维化的）类型 $A, B : \mathtt{Fib}\ \Gamma$ 之间构造路径的方式来满足公理 (1) 和 (2)。

\begin{theorem}[同构等价性]
给定纤维化 $A, B : \mathtt{Fib}\ \Gamma$ 且 $f : A \cong B$，我们可以构造路径 $\mathtt{isopath}(f) : A \sim_{\mathcal{U}} B$。
\end{theorem}

\begin{proof}
给定上述 $A, B, f$，令 $B' \triangleq B[\mathtt{fst}] : \mathtt{Fib}(\Gamma \times \mathbb{I})$ 并注意到 $f : A \cong B'[\langle  id , \mathtt{0} \rangle]$ 且 $ id  : B \cong B'[\langle  id , \mathtt{1} \rangle]$，其中 $ id $ 是明显的同构 $B \cong B$。因此我们可以定义
$$\mathtt{isopath}(f) \triangleq \mathtt{improve}(B[\mathtt{fst}], f,  id ) : A \sim_{\mathcal{U}} B$$
如所需。注意，在这种情况下，$\mathtt{improve}$ 实际上只会在 $\mathtt{0}$ 处改进 $B[\mathtt{fst}]$，因为在 $\mathtt{1}$ 处我们沿着恒等映射改进，这不起作用。注意，对于任何 $\gamma : \Delta \to \Gamma$ 我们有：
\begin{align*}
\mathtt{isopath}(f \circ \gamma) &= \mathtt{improve}(B[\gamma][\mathtt{fst}], f \circ \gamma,  id _{B[\gamma]}) \\
&= \mathtt{improve}(B[\mathtt{fst}][\gamma \times  id ], f \circ \gamma,  id _B \circ \gamma) \\
&= \mathtt{improve}(B[\mathtt{fst}], f,  id _B)[\gamma \times  id ] \\
&= (\mathtt{isopath}\ f)[\gamma \times  id ]
\end{align*}
因此 $\mathtt{isopath}$ 在替换下是稳定的。
\end{proof}

\begin{corollary}
公理 (1) 和 (2) 在 $\mathcal{E}$ 中的纤维化类型的 CwF 中成立。
\end{corollary}

\begin{proof}
明显的同构 $A \cong A \times 1$ 和 $\sum_{a:A} \sum_{b:B} C\ a\ b \cong \sum_{b:B} \sum_{a:A} C\ a\ b$ 在定义 5.1.6 的意义上都是严格的同构。因此我们可以构造所需的路径 $A \sim_{\mathcal{U}} (A \times 1)$ 和 $\big(\sum_{a:A} \sum_{b:B} C\ a\ b\big) \sim_{\mathcal{U}} \big(\sum_{b:B} \sum_{a:A} C\ a\ b\big)$。因此公理 (1) 和 (2) 成立。
\end{proof}

我们已经看到我们可以使用我们的公理轻松满足性质 (1) 和 (2)。然而，我们还需要知道沿着这些相等性强制转换时会发生什么。这可以一般性地陈述为任何严格同构的类型。

\begin{theorem}
给定纤维化 $(A,\alpha), (B,\beta) : \mathtt{Fib}\ \Gamma$ 且 $f : A \cong B$，沿着 $\mathtt{isopath}\ f$ 强制转换（命题地）等于应用 $f$。
\end{theorem}

\begin{proof}
取上述 $(A,\alpha), (B,\beta), f$ 并令 $(P,\rho) = \mathtt{isopath}\ f$。通过展开所使用的构造，我们可以看到 $\rho$ 是通过重新对齐某个 $\rho_{pre}$ 获得的，而 $\rho_{pre}$ 又是通过将 $\beta[\mathtt{fst}]$ 沿着同构转移获得的：
$$\mathtt{iso}(x,i) = \mathtt{snd}\big(\ax{9}((A \veebar B)(x,i,\_), B\ x, (f \veebar  id )(x,i,\_))\big) : P\ x \cong B\ x$$
现在考虑任意的 $x : \Gamma, a_0 : A\ x$ 并注意到
$$\mathtt{iso}(x,\mathtt{0}) = (f \veebar  id )(x,\mathtt{0}) = f\ x \qquad \text{且} \qquad \mathtt{iso}(x,\mathtt{1}) = (f \veebar  id )(x,\mathtt{1}) =  id $$

同时，观察到
$$\forall i.(i = \mathtt{0} \veebar i = \mathtt{1}) \Rightarrow (\forall i.i = \mathtt{0}) \veebar (\forall i.i = \mathtt{1}) \Rightarrow \mathtt{1} = \mathtt{0} \veebar \mathtt{0} = \mathtt{1} \Rightarrow \bot$$
由公理 $\ax{1}$，通过实例化 $\forall$，由公理 $\ax{2}$，因此 $\forall i.(i = \mathtt{0} \veebar i = \mathtt{1}) = \bot$。现在计算：
\begin{align*}
&\mathtt{coerce}(\mathtt{isopath}\ f)\ x\ a_0 \\
&\quad = \rho\ \mathtt{0}\ \langle x,  id  \rangle\ \bot\ \mathtt{elim}_{\emptyset}\ a_0 && \text{由定义} \\
&\quad = \rho_{pre}\ \mathtt{0}\ \langle x,  id  \rangle\ (\bot \veebar (i = \mathtt{0} \veebar i = \mathtt{1}))\ \mathtt{elim}\ a_0 && \text{由引理 5.3.10（或类似）} \\
&\quad = \rho_{pre}\ \mathtt{0}\ \langle x,  id  \rangle\ \bot\ \mathtt{elim}_{\emptyset}\ a_0 \\
&\quad = \rho_{pre}\ \mathtt{0}\ \langle x,  id  \rangle\ \bot\ \mathtt{elim}_{\emptyset}\ (\mathtt{iso}(x,\mathtt{0})\ a_0) \\
&\quad = \beta\ \mathtt{0}\ \langle x,  id  \rangle\ \bot\ \mathtt{elim}_{\emptyset}\ (\mathtt{iso}(x,\mathtt{0})\ a_0) && \text{由引理 5.3.8} \\
&\quad = \beta\ \mathtt{0}\ \langle x,  id  \rangle\ \bot\ \mathtt{elim}_{\emptyset}\ (f\ x\ a_0) \\
&\quad = \beta\ \langle x,  id  \rangle\ \bot\ \mathtt{elim}_{\emptyset}\ (f\ x\ a_0) && \text{由切片} (\mathtt{iso}(x,\mathtt{1}) =  id ) \\
&\quad = \beta\ \langle x,  id  \rangle\ \bot\ \mathtt{elim}_{\emptyset}\ (f\ x\ a_0) && \text{由切片}
\end{align*}
由于这只是应用于 $f\ x\ a_0$ 的平凡/空复合，我们可以如下构造从 $f\ a_0$ 到 $\mathtt{coerce}(\mathtt{isopath}\ f)\ x\ a_0$ 的路径：
$$\mathtt{fill}\ \mathtt{0}\ \beta\ \langle x,  id  \rangle\ \bot\ \mathtt{elim}_{\emptyset}\ (f\ x\ a_0) : f\ x\ a_0 \sim \mathtt{coerce}(\mathtt{isopath}\ f)\ x\ a_0$$
因此，沿着 $\mathtt{isopath}\ f$ 强制转换总是命题地等于应用 $f$。
\end{proof}

\begin{corollary}
公理 (4) 和 (5) 在 $\mathcal{E}$ 中的纤维化类型的 CwF 中成立（对于在推论 6.2.7 中构造的项）。
\end{corollary}

\begin{proof}
由定理 6.2.8。
\end{proof}

\subsection{公理 (3)}

鉴于前一节，唯一剩下的公理是公理 (3)。我们此处的目标是，给定一个可缩纤维化 $A : \mathtt{Fib}\ \Gamma$，定义路径 $A \sim_{\mathcal{U}} 1$。注意，对于任何 $\Gamma : \mathcal{U}$，存在唯一的纤维化结构 $!_1$ 使得 $(\lambda\_ \to 1, !_1) : \mathtt{Fib}(\Gamma)$。因此我们将模糊地写 $1 : \mathtt{Fib}(\Gamma)$ 表示对 $(\lambda\_ \to 1, !_1)$。

\footnote{注意有不同的方式来解释模型中的 $\mathtt{coerce}$。这种解释通常与直接解释定义 2.1.6 获得的解释不同。然而，这两种解释在模型中总是路径相等的（另一种解释会有更多平凡/空复合），因此当使用另一种解释时结果仍然成立。}

\begin{definition}[族的收缩]
给定族 $A : \Gamma \to \mathcal{U}$，我们定义 $A$ 的收缩为
\begin{align*}
& C_A : \Gamma \times \mathbb{I} \to \mathcal{U} \\
& C_A(x,i) \triangleq [i = \mathtt{0}] \to A(x)
\end{align*}
注意，对于任何 $\gamma : \Delta \to \Gamma$，我们有 $C_{A \circ \gamma} = C_A \circ (\gamma \times  id )$。
\end{definition}

我们现在需要证明当 $A$ 既是纤维化的又是可缩的（定义 5.4.4）时，$C_A$ 是纤维化的。回顾引理 5.4.5，一个族既是纤维化的又是可缩的等价于对该族具有扩展结构（定义 5.4.3）。

因此我们如下构造 $C_A$ 的纤维化结构：

\begin{theorem}
如果 $(A,\alpha) : \mathtt{Fib}\ \Gamma$ 是可缩的，那么我们可以为 $C_A$ 构造复合结构。即，存在函数
$$\mathtt{isFib}_C : \{\Gamma : \mathcal{U}\}\{A : \Gamma \to \mathcal{U}\} \to \mathtt{isFib}\ A \to \mathtt{Contr}\ A \to \mathtt{isFib}\ C_A$$
\end{theorem}

\begin{proof}
取 $\Gamma : \mathcal{U}$，$A : \Gamma \to \mathcal{U}$，$\alpha : \mathtt{isFib}\ A$ 和 $\mathtt{ctr} : \mathtt{Contr}\ A$。由于 $A$ 既是纤维化的又是可缩的，那么由引理 5.4.5，我们可以构造扩展结构 $\varepsilon = \mathtt{toExt}\ \alpha\ \mathtt{ctr} : \mathtt{Ext}\ A$。我们然后可以如下定义复合结构 $\mathtt{isFib}_C\ \alpha\ \mathtt{ctr} : \mathtt{isFib}(C_A)$：
$$\mathtt{isFib}_C\ \alpha\ \mathtt{ctr}\ e\ p\ \varphi\ f\ c_0\ u \triangleq \varepsilon\ \big(\mathtt{fst}(p\ \overline{e})\big)\ \big(\varphi, \big(\lambda v \to f\ v\ \overline{e}\ u\big)\big)$$
对于 $u : [\mathtt{snd}(p\ \overline{e}) = \mathtt{0}]$。由于 $\varepsilon$ 是扩展结构，我们有对于任何 $v : [\varphi]$，
\begin{align*}
\mathtt{isFib}_C\ \alpha\ \mathtt{ctr}\ e\ p\ \varphi\ f\ c_0 &= \lambda u \to \varepsilon\ \big(\mathtt{fst}(p\ \overline{e})\big)\ \big(\varphi, (\lambda v \to f\ v\ \overline{e}\ u)\big) \\
&= \lambda u \to f\ v\ \overline{e}\ u \\
&= f\ v\ \overline{e}
\end{align*}
因此 $(\varphi, f) @ \overline{e} \nearrow \mathtt{isFib}_C\ \alpha\ \mathtt{ctr}\ e\ p\ \varphi\ f\ c_0$ 如所需。因此我们为 $C_A$ 定义了有效的复合操作。进一步，注意对于任何 $\gamma : \Delta \to \Gamma$ 我们有：
\begin{align*}
&\mathtt{isFib}_C\ (\alpha[\gamma])\ (\mathtt{ctr} \circ \gamma)\ e\ p\ \varphi\ f\ c_0\ u \\
&\quad = \mathtt{toExt}\ (\alpha[\gamma])\ (\mathtt{ctr} \circ \gamma)\ \big(\mathtt{fst}(p\ \overline{e})\big)\ \big(\varphi, \big(\lambda v \to f\ v\ \overline{e}\ u\big)\big) \\
&\quad = \big((\mathtt{toExt}\ \alpha\ \mathtt{ctr}) \circ \gamma\big)\ \big(\mathtt{fst}(p\ \overline{e})\big)\ \big(\varphi, \big(\lambda v \to f\ v\ \overline{e}\ u\big)\big) \\
&\quad = (\mathtt{toExt}\ \alpha\ \mathtt{ctr})\ \big(\gamma\ (\mathtt{fst}(p\ \overline{e}))\big)\ \big(\varphi, \big(\lambda v \to f\ v\ \overline{e}\ u\big)\big) \\
&\quad = (\mathtt{toExt}\ \alpha\ \mathtt{ctr})\ \big(\mathtt{fst}((\gamma \times  id )(p\ \overline{e}))\big)\ \big(\varphi, \big(\lambda v \to f\ v\ \overline{e}\ u\big)\big) \\
&\quad = (\mathtt{isFib}_C\ \alpha\ \mathtt{ctr})\ [\gamma \times  id ]\ e\ p\ \varphi\ f\ c_0\ u
\end{align*}
\end{proof}

因此 $\mathtt{isFib}_C\ (\alpha[\gamma])\ (\mathtt{ctr} \circ \gamma) = (\mathtt{isFib}_C\ \alpha\ \mathtt{ctr})[\gamma \times  id ]$，因此 $\mathtt{isFib}_C$ 在重索引下是稳定的。

\begin{theorem}
存在函数
$$\mathtt{contract} : \{\Gamma : \mathcal{U}\}(A : \mathtt{Fib}\ \Gamma) \to \mathtt{Contr}\ A \to A \sim_{\mathcal{U}} 1$$
\end{theorem}

\begin{proof}
给定 $\Gamma : \mathcal{U}$，$(A,\alpha) : \mathtt{Fib}\ \Gamma$ 和 $\mathtt{ctr} : \mathtt{Contr}\ A$，我们观察到
$$C_A[\langle  id , \mathtt{0} \rangle](x) = C_A(x,\mathtt{0}) = \big([\mathtt{0}=\mathtt{0}] \to A(x)\big) \cong \big(1 \to A(x)\big) \cong A(x)$$
$$C_A[\langle  id , \mathtt{1} \rangle](x) = C_A(x,\mathtt{1}) = \big([\mathtt{1}=\mathtt{0}] \to A(x)\big) \cong \big(\emptyset \to A(x)\big) \cong 1$$
因此我们有
$$\big((C_A, \mathtt{isFib}_C\ \alpha\ \mathtt{ctr}), f_A, g_A\big) : (A,\alpha) \sim_{\cong} 1$$
其中 $f_A : A \cong C_A[\langle  id , \mathtt{0} \rangle]$ 和 $g_A : 1 \cong C_A[\langle  id , \mathtt{1} \rangle]$ 是上面指出的明显同构。因此我们可以定义
$$\mathtt{contract}((A,\alpha), \mathtt{ctr}) \triangleq \mathtt{improve}((C_A, \mathtt{isFib}_C\ \alpha\ \mathtt{ctr}), f_A, g_A) : (A,\alpha) \sim_{\mathcal{U}} 1$$
如所需。进一步，对于任何 $\gamma : \Delta \to \Gamma$，我们有：
\begin{align*}
&\mathtt{contract}((A,\alpha)[\gamma])(\mathtt{ctr} \circ \gamma) \\
&\quad = \mathtt{improve}((C_{A \circ \gamma}, \mathtt{isFib}_C\ (\alpha[\gamma])\ (\mathtt{ctr} \circ \gamma)), f_{A \circ \gamma}, g_{A \circ \gamma}) \\
&\quad = \mathtt{improve}((C_A \circ (\gamma \times  id ), (\mathtt{isFib}_C\ \alpha\ \mathtt{ctr})[\gamma \times  id ]), f_A \circ \gamma, g_A \circ \gamma) \\
&\quad = \mathtt{improve}((C_A, \mathtt{isFib}_C\ \alpha\ \mathtt{ctr})[\gamma \times  id ], f_A \circ \gamma, g_A \circ \gamma) \\
&\quad = \mathtt{improve}((C_A, \mathtt{isFib}_C\ \alpha\ \mathtt{ctr}), f_A, g_A)[\gamma \times  id ] \\
&\quad = \mathtt{contract}((A,\alpha), \mathtt{ctr})[\gamma \times  id ]
\end{align*}
因此 $\mathtt{contract}$ 在重索引下是稳定的。
\end{proof}

\begin{corollary}
$\mathcal{E}$ 中的纤维化类型的 CwF 建模公理 (3)。
\end{corollary}

\section{结论}

在本章中，我们展示了单值性公理如何被分解为一组替代性质。这种分解发生在 MLTT 内部，不涉及任何特定模型。然后我们展示了这些新性质如何被任何满足我们第五章公理的拓扑斯满足。大多数新性质相当直接地从严格性公理 $\ax{9}$ 和纤维化的性质得出，例如重新对齐引理（引理 5.3.10）以及纤维化在同构下封闭的事实。最后的性质，编号 (3)，通过定义族的收缩（定义 6.2.10）被证明成立。这可以被视为第 5.4 节中呈现的粘合构造的显著简化情形。

在第五章中，我们展示了如何使用拓扑斯的内部语言重复 Cohen 等人 [18] 的许多构造。尽管我们无法使用内部语言构造单值宇宙，我们确实展示了一些与单值性相关的结果。其思想是这些结果应该意味着，如果存在一个包含所有小纤维化的宇宙，那么该宇宙将满足单值性公理。在本章中，我们用我们相信更简单的构造证明了相同的结果。因此，当本章的结果与第五章的结果一起考虑时，证明了以下定理：

\begin{theorem}
给定任何作为我们公理（图 5.4）模型的拓扑斯 $\mathcal{E}$。如果 $\mathcal{E}$ 还建模 $\mathcal{U}$-小纤维化的类型论宇宙，那么该宇宙满足单值性。
\end{theorem}

公平地说，应该注意到，虽然这个单值性公理的证明避免了使用第 5.4 节的粘合构造，但这并不意味着我们完全消除了对粘合的需要。特别是，证明 $\mathcal{U}$-小纤维化宇宙是纤维化对象（在 1 上的纤维化）的外部证明仍然需要使用粘合构造。宇宙是纤维化的事实是至关重要的，不仅为了确保它包含在纤维化类型的 CwF 中，而且为了允许我们在宇宙中复合路径，如我们在定理中所做的那样。

因此，为了给出立方体集合模型中单值类型论的完整内部描述，唯一缺失的部分是构造这样一个纤维化类型的宇宙。这个问题在第七章中得到解决。
